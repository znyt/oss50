　　但是出乎他们意料的是，刘哔刚刚走到车前，还不等他们反应过来，就看到其背后突然出现的一对黑色翅膀。
<br/>　　完了，是高阶丧尸！
<br/>　　孙强心中一惊，一股浓郁的恐惧感从心底升腾了起来，但还未来得及有任何动作，就感到脖颈处一疼，整个人被一只大手抓着，提在了半空中。
<br/>　　“哪来的鼠辈，敢在这里埋伏我。”手中提着孙强，刘哔飞到了半空中，一对漆黑的翅膀在背后忽闪着，随着最后一抹夕阳的余光消失在地平线，整个人仿佛都和黑暗融为了一体。
<br/>　　众人心中皆是一凉，下意识的，他们将刘哔认成了高阶丧尸，此时都是有些绝望。
<br/>　　额，事实上倒也没错，他们歪打正着的，恰好撞破了刘哔的身份，却是将等级给弄混了。
<br/>　　虽然借助着特殊的体质，刘哔现在的实力要比一般的筑基期强大很多，但还不足以和高阶丧尸相比拟。
<br/>　　那种级别的丧尸，可是随便动一动都能引起山河崩裂的存在。
<br/>　　“大人饶命，大人饶命，我们愿意追随您，请大人高抬贵手，放过我们！”刘哔的话刚落音，先前说话的那个火系异能者，‘扑通’一下跪在地上，不间断的磕着响头。
<br/>　　随着这个火系异能者的动作，其余众人随即也反应了过来，连忙有样学样，纷纷跪在地上，磕头求饶着。
<br/>　　他们对自己团队的实力十分了解，虽然并不能算弱，但是碰见这种级别的丧尸，根本没有半点胜算，甚至连逃跑都是不可能的事情。
<br/>　　外表和人类没有区别，可以口吐人言，甚至有飞行的能力。
<br/>　　这几种特征，可都是八九阶的领主级丧尸才能做到的事情，给他们一万个胆子，他们都不敢有半点的反抗之心。
<br/>　　被他们的态度搞得一愣，刘哔随即反应了过来，自然想到了他们的想法，嘴角不由得露出一抹邪笑，也没过多解释，任由他们心中继续误会下去。
<br/>　　“你们是哪个基地的人，聚集在这里，莫非是想要抓我不成？”挥手将孙强的脖子捏断，刘哔发动了吞噬技能，一根根血色的细小出手从其手中浮现，瞬间将孙强给吸成了一具皮包骷髅。
<br/>　　这是他将吞噬技能升级之后新出现的特质，用这样的手法，吸收率要提高了许多。
<br/>　　不知是不是孙强等级过低的缘故，刘哔并没有获得什么技能，只是感觉身体内的灵种似乎壮大了一些，除此之外还收获了六百点积分。
<br/>　　虽然同为筑基，但和之前在僵尸村里击杀的那几头毛僵相比，这家伙给他提供的积分可少了许多。
<br/>　　毕竟，那些毛僵都是经过精心的培育，虽然实力并算不上真正的筑基高阶，但等级却是到了，故此给他提供的积分也有一千之多。
<br/>　　和那个鬼修老头，不相上下，不过两者之间的实力，却不是在一个档次上的。
<br/>　　“我们，我们是一号基地的搜索队，来这里也不是为了埋伏您，只是刚从镇子里出来，想要在此地歇息一会！”看着孙强被杀死，火系异能者面色有些惶恐，他知道孙强的身份，如果回到基地，恐怕他的下场也不会好很多。
<br/>　　但现在最重要的，还是先能保住自己的性命，如果这次真的能逃出去，大不了他就一走了之，再也不回一号基地好了。
<br/>　　心中打定了注意，火系异能者将头颅伏在地上，对刘哔的问话不敢有丝毫隐瞒。
<br/>　　微微点了点头，刘哔眼神没有太大波澜，一号基地的搜索队他自然知道，甚至先前的时候他还去军营应征过，最终却因为异能问题被刷了下来。
<br/>　　倒不是等级不够，事实上他的异能等级虽然不高，但也算是合格，问题是他的异能属性，实在是让人有些蛋疼。
<br/>　　在这个大部分人都是注重攻击或者防御能力的环境下，他偏偏觉醒了一个没什么卵用的辅助技能。
<br/>　　如果只是一般的辅助技能也就算了，刻苦修炼一下，等级高了也会发挥不小的作用，可是他的异能特质，却是让自己的眼球可以转动180度。
<br/>　　当然，这个特质只能作用于自己身上，等级越高，可以运行的时间就越长。
<br/>　　一开始，他还对自己的异能抱有希望，至少也算是一个超能力，比那些没有超能力的幸存者，总要好一些吧。
<br/>　　修炼了一段时间，他成为了三阶异能者，这个等级，在当时全世界都找不出几个，可结果却依旧让他失望了，他的这个能力，真的屁的用都没有。
<br/>　　这段时间里，他也接触了诸多异能者，其中也不乏有一些异能特殊的，他们的异能效果同样稀奇古怪，自然也会有毫无用处的异能。
<br/>　　比如，可以在直肠中中制造出任何直径超过10厘米的东西。
<br/>　　又比如，可以让身体变得对任何东西都过敏。
<br/>　　或者，是一打响指会胃溃疡。
<br/>　　诸多等等，这些稀奇古怪的技能，仿佛是命运随意开下的一个玩笑，却让他们彻彻底底的绝望。
<br/>　　他抱怨，不甘，愤怒，如果他的异能不是这种样子，哪怕只是一个普普通通的异能都好，以他的天分，也足以成为一方王者。
<br/>　　可现实就是如此，他所做的这一切都没有什么用处。
<br/>　　想明白了之后，刘哔也彻底将异能丢在了一旁，不再修炼，与其浪费晶核修炼这种东西，还不如省下来换取一些生存资源。
<br/>　　和他所了解的修士不同，异能者想要修炼，只能转换晶核内的能量，然后融入自己体内，除此之外，没有任何其他的方法。
<br/>　　而他的弟弟只是一个没有觉醒异能的普通幸存者，晶核对其自然也没有半点用处。
<br/>　　前面的十年里，刘哔都认为自己不会再有什么希望了，他唯一能做的，就是拼尽全力，让自己和刘莽一起活下去。
<br/>　　虽然这日子过得确实是有些烂，但既然有机会不死，那总得活着试试。
<br/>　　幸好，似乎是命运对他开够了玩笑，在他濒死的时候，给他弄了个狗屁系统，才保住了他的性命，甚至还带给他强大起来的希望。
