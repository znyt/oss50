　　第670章
<br/>　　苏樱还是有做大妇的风范，或者说，她有想做大妇的风范。相夫教子，家园和睦，她还想拉着樊梨花共同与柳芷聊聊，但很明显，樊梨花在这方面没有什么天赋，同样也不想有天赋。
<br/>　　无论事情如何，柳芷的身份已经从伺候的丫鬟提升为妾，虽然谈不上太好，却已是柳芷期盼的理想状态。
<br/>　　程大雷不太关心这件事，他也不是很了解，直到晚上伺候的丫鬟从柳芷变为苏樱的丫头小蝶，程大雷才算是知道发生了什么。
<br/>　　他窝在书房中翻阅情报，最近军策府绞尽脑汁在想，尚未拿出切实可行的办法。程大雷也就是造造谣，想破局的办法。
<br/>　　只是身边端茶递水的丫头从柳芷变成小蝶，二人的性格截然不同，柳芷偶尔装傻，但小蝶是真傻。
<br/>　　她不是很关心程大雷在做什么所谓的大事，天真烂漫，毫无心机。如此，倒是正中程大雷下怀，他本身就不是个太规矩的人。
<br/>　　“这白玉美人一定生得十分好看吧。”小蝶在一旁伺候，忍不住道了一句。
<br/>　　“嗯，应该是吧。”程大雷道：“我也没见过，不过，崔相生得獐头鼠目，贼头贼脑，他的女儿能好看到那里，说不得是以讹传讹。”
<br/>　　有种女人能好看到令男人自惭形秽，令女人心生艳羡，小蝶心里幻想着：“如果能见见她就好了，你就不想见么？”
<br/>　　口中如此称呼，换做旁人，直接拉下去打杀也是寻常。不过，程大雷倒是不在意这点，他心中寻思着，继而道：“想看自然是想看的，只不过她是我们的敌人呢。”
<br/>　　“你一个山贼，还讲究这样，抢过来不就行了。”
<br/>　　程大雷摇头苦笑，这小蝶委实有些异想天开。忽地转念一想，也并非是不可能么。这白玉美人传说智谋无双，但没听说她有武艺，如果直接抢过来，岂非折损宋伯康一大助力。
<br/>　　闻君有白玉美人……
<br/>　　程大雷不禁想起哪位风流盗帅，可惜呐，自己虽然有踏月留香的美梦，却没有香帅那出神入化的本事呐。
<br/>　　如此，也只得灭掉这非分之想。
<br/>　　不过，对于这位玉美人，程大雷还真是想见一见的。
<br/>　　是怎样的美貌，值得了一座城池？
<br/>　　……
<br/>　　凉州城。
<br/>　　宋伯康面对这位名扬长安的小姑姑，也有些忐忑。她一身白袍，立在桌前，凝神盯着面前的作战图。在其脸上，挂着一面黑纱，看不清本人模样。
<br/>　　莫说程大雷，便是宋伯康也是想见见的，好美恶丑之心人皆有之，宋伯康也并不例外。但就连他，也没见过这位玉美人的相貌。
<br/>　　传说，曾有位大名狂士一睹芳容之后，自己挖掉双眼，只说见过天下如此美人，下半生再不忍心看任何丑陋。
<br/>　　外人只以为这件事是传说，但宋伯康知道这件事的确发生过。不过，事实和传说还是有些差距，那狂士的双眼并非其自己挖掉，而是玉美人刺瞎。
<br/>　　外人只说这位玉美人的身份、才智、美貌，但知道她心冷者怕是没几个。
<br/>　　“凉州城何时可以动兵？”
<br/>　　清冷的声音忽然想起，宋伯康收起心思，道：“正在夜以继日的准备，要发兵攻打蛤蟆城，需要在十日后。”
<br/>　　兵马粮草武器都要准备妥当，十日的时间已经不算长。
<br/>　　崔白玉点点头，手指落在地图上：“京州的支持会在五天后到，十日后便发兵攻打蛤蟆城，凉州方面有什么打算？”
<br/>　　这不像是商议，倒像是一场考试，关键是宋伯康并没有任何不适，他道：“现在程贼的势力已经被分开在蛤蟆城和朔方城两地，我们准备先打朔方城，程大雷必定派兵支援，我们也可围点打援，消耗他的力量。”
<br/>　　崔白玉露出赞许的眼神，道：“家父派你来这里，果然没有看错人。”
<br/>　　宋伯康心中有些不是滋味，什么叫做『派』，凉州不是自己的，凉州包括自己都是崔相的。崔白玉刚到就拿了兵权，自己这凉州王根本就是个摆设么。
<br/>　　“不过……”崔白玉话锋一转：“围点打援是消耗别人，也是消耗自己，我们的粮草运送路线太长，一旦被切断，后勤供应不上，局势岂不是对我们不利。”
<br/>　　宋伯康不像刚才一样恭顺，道：“战场上的事瞬息万变，怎能开始便预料到所有事，也还是需要随机应变。”
<br/>　　“战场上的确瞬息万变，但也该在未开战前，便尽可能想到更多。”崔白玉算是小小教导了宋伯康一下。
<br/>　　“那郡主有什么高见？”宋伯康有些不服气。
<br/>　　“我还需了解。”崔白玉道：“将关于程大雷的所有资料送到我的房中，我先要知道他是怎样的人。”
<br/>　　程大雷和宋伯康和平共处许久，但双方其实都明白，和平是暂时的，战斗是必须的。于是暗地里都在收集双方的资料。程大雷如此，宋伯康也是一样。
<br/>　　关于程大雷多大年纪，怎样经历，什么武艺，甚至每天吃什么饭，都是宋伯康想要收集的。当然，收集得到收集不到，是另外一件事。
<br/>　　过去这几年，已经收集到堆积如山的资料，今日都送进崔白玉居住的院落里。
<br/>　　院外是死忠的亲随，里面是伺候的丫鬟。程大雷有他的丫鬟，崔白玉自然也有。如柳芷和小蝶，无非就是端茶倒水，偶尔陪程大雷闲聊几句。但崔白玉身边的人，都是千挑万选出来。她们不仅是专业的丫鬟，也可以说是专业的谋士。
<br/>　　今日，屋中灯火通明，崔白玉坐在案后，三名贴身丫鬟来来去去，将案卷分门别类的整理后，再送到崔白玉面前。
<br/>　　她们与其说是丫鬟，倒不如说是崔白玉的助手。崔白玉有神算子的名声，她身边的丫头也不可能太笨，如小蝶这样的人肯定是不够格的。
<br/>　　这三个丫鬟就是崔白玉精心挑选出来，她们分别有个名字，唤作：小犬、小鹿、小刀。
<br/>　　旁人家的小姐都是给丫鬟起花啊蝶呀这类的名字，崔白玉如此，足见这女人的出奇不同之处。
