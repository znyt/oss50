　　第196章
<br/>　　大河之水天上来。
<br/>　　河水浩浩而下，灌进山谷，吞没篝火，淹没野草，然后是房屋，然后是马匹，人……人们在疯狂，人们在呼喊，他们在疯狂呼喊，但不再是为了庆祝，而是为了逃生。
<br/>　　人群疯狂向谷口涌去，可洪水猛兽之所以被称为洪水猛兽，是因为洪水比猛兽更猛。他们奔跑的速度抵不过洪流的速度。
<br/>　　此时谷口已挤满了人，谷口狭窄，人一旦都挤过来，路便被堵死了。
<br/>　　“让我出去，让我出去！”
<br/>　　“谁敢挡我的路，杀！”
<br/>　　人们争路，互相厮杀，然后路被堵得更死。
<br/>　　白日，他们疯狂的想要进来，夜里，他们疯狂的想要出去，所以说人，有时候真是一种奇怪的动物。
<br/>　　刘关张来到程大雷身边，看着山谷内的状况，表情不变，只是冲程大雷轻轻点点头，然后一起蹲在程大雷身边，和程大雷一起……欣赏。
<br/>　　“这些戎族烧杀抢掠，无恶不作，大当家将他们一网打尽，功德无量呐。”
<br/>　　“功德无量……”程大雷撇撇嘴，心道：你是不知道我的恐惧值现在是个如何可怕的数值。
<br/>　　半月前，程大雷已经派关鱼去截流断水，更早之前，徐神机就提醒过程大雷：山谷其实并不是适合建设山寨，雨季一到，谷内的水排不出去，山寨极容易被淹没。
<br/>　　当初程大雷没往心里去，只是让大家深掘河道，以便排水。却是没想到，徐神机当初的话，应到了今天。
<br/>　　程大雷在城头装神弄鬼，一来是唬住戎族，让山寨的弟兄撤出去，二来也是等待合适的时机。
<br/>　　他想和系统耍心眼，提前把抢夺压寨夫人的任务完成，结果没收到任务完成的提示，反而因为和苏樱同床共枕的缘故，积累了很多日的风雨。
<br/>　　大雨连绵，的确有雨季的原因，但也有程大雷借助系统的缘由。不得不说，一连能有这么多日风雨，证明程大雷夜里还真是勤奋呐。
<br/>　　cos丞相弹琴退仲达我是不行，但cos关二爷水淹七军还是绰绰有余的，此刻本命的关二爷就在身边蹲着抠脚呢。
<br/>　　如今天时（雨季）地利（山谷）人和（关鱼）三者齐全，一声令下，一挂大河天降，所以说，你不死谁死。
<br/>　　程大雷蹲在悬崖上观水，只见洪流过处，逃无可逃。戎族阵中，并不缺少顶级英雄，可在这天道之威下，莫说双臂有五百斤力气，便是有千斤力气，又能做得什么。
<br/>　　金问道死抱着一颗大树，不多时被水盖过了头顶。八王子站在屋顶上，冲着程大雷破口大骂一句，水便灌进了他的喉咙……
<br/>　　戎族大王子，那位指挥十万大军的大将被洪流冲击撞到山石上，头破血流而死。
<br/>　　他白日巡视山谷时，只看到此地易守难攻，是个军事要地，那时他怕是绝没有想到，此地不仅适合屯兵，还适合埋人。
<br/>　　本就是一座天造地设的坟墓。
<br/>　　“好啦，走吧，没什么好看的，咱们找个地方歇脚去。”程大雷拍拍身上的土站起来。
<br/>　　“大当家，要不要把山寨的弟兄接回来。”
<br/>　　“现在咱们没粮没人，这里又不知要乱到什么时候，等安定下来，再接他们回来吧。”程大雷道。
<br/>　　转过身，大家的身影渐渐消失。
<br/>　　嘟，正义的少年喔，恭喜你完成隐藏任务：歼灭入侵幽州的北蛮部。因你极其出色的完成任务，所以可以在三种绝世武学中任选其一：
<br/>　　甲：小李飞刀
<br/>　　乙：易筋经
<br/>　　丙：身无彩凤双飞翼，心有灵犀一点通
<br/>　　……
<br/>　　因为这一战，程大雷虽然将戎族大军一网打尽，但建好的三级山寨丢了。基本盘没了，一切需要重新再来。
<br/>　　他又带人回到蛤蟆岭上，从戎族的大营中拆除帐篷，收集余粮，重新打基本盘。
<br/>　　在这段时间内，帝国的疆域内，还发生了许多事情。
<br/>　　在西北边关，守将林问天溃败，再举剑自尽时，被亲兵救下，一路护送着回到京城。御史参他带兵无方，被明帝下旨斩了。
<br/>　　一个牧羊人冒充帝国六王子，募集军队抵抗守城，被引为笑柄。他被官兵抓了，在准备处死时，他贿赂守卫逃脱，从此下落不明。
<br/>　　北蛮部十位王子死在幽州，北蛮王痛哭一日，第二日又麻溜快的封了十个新的王子。
<br/>　　戎族数个部落组成的军队从凉州长驱直入，一直打到京城脚下。明帝派出官员与戎族议和，接受了极其苛刻的条件，戎族这才大摇大摆的扬长离去。
<br/>　　一时间举国震惊，多少书生才子慷慨激昂，多少军人扼腕叹息，有一王姓御史撞死在大殿上，临死前历数明帝十大罪，将和谈称为帝国之耻。
<br/>　　然后其死后鞭尸十日，满门抄斩。
<br/>　　不管以怎样的方式，战争总算是结束了。流离失所，侥幸在戎族铁蹄下活下来的人们回到家乡，埋葬亲人的尸体，擦干眼泪，继续耕耘祖辈留下来的土地。
<br/>　　将时间稍微往回拨一下，视线落到青牛山。
<br/>　　好多日后，洪水终于退去，曾近欣欣向荣的山寨，如今布满尸体。十万具尸体堆在一起是怎样的场景，那是地狱里的事情，在人间怕是难以想象，只知道很久很久以后，这片土地长出的野草分外茂盛。
<br/>　　一棵大树上，金问道死死的抱在上面，过来很久，他睁开眼睛，两手一松，从树上掉了下来。自然不会有什么事，因为脚下堆满尸体。
<br/>　　他晃晃脑袋，深一脚浅一脚往外走。
<br/>　　山壁上有一块凸起的石头，仔细分辨才发现，上面趴着一个人。他年纪还小，身上已伤痕累累，但睁开的眼睛证明他还活着。
<br/>　　这是福德勒。
<br/>　　福德勒是被洪水冲到上面的，然后一直没有下来，也不知困了多久，三天，七天……他双手一松，也跌在地上。咬咬牙，站起来，挣扎着往外走。
<br/>　　脚下是一具具浮尸，睁大眼睛，尸体被水泡得苍白，膨胀。
<br/>　　若说这是人间地狱，当没人反对，而从古至今，能走出地狱的只有两种存在。一种是鬼，一种是魔。
<br/>　　问一声，你是鬼还是魔。
