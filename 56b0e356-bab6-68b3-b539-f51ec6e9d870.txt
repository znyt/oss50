    “走原路。”陈兴没怎么犹豫就做出了决定。这里的地形有点儿像迷宫，贸然改变方向，容易迷失。<br/>    <br/>    他从背包里拿出一个小布袋，搜集了一些小石头，准备到分岔路就丢一颗，以便于认路。不过朝前走了很长一段时间，遭遇了五六只石像，依然没有出现分岔路。<br/>    <br/>    “这里很大啊……”<br/>    <br/>    二十多分钟后，陈兴停止了探索，留在原地思考。他心里有些纳闷，按理说，楼下是集会的场所，而且从壁画上看，来的人不少。<br/>    <br/>    要满足这么一大批人进进出出，设计上不可能太复杂，二十分钟都走不出去。其次，这里看起来像是防御设施，困住和阻挡敌人的地方。由此可见，高塔的出口很可能就在一层，只是他们不知道开启的方法。<br/>    <br/>    为了保险起见，陈兴决定往回走。先解决后路的问题，再考虑深入探索。如果被困在这里，找到什么宝物都没用。<br/>    <br/>    可这时，阿乔木却提出了反对意见。<br/>    <br/>    “我认为我们应该更有冒险精神。”<br/>    <br/>    “我同意你的观点。”陈兴说道。<br/>    <br/>    阿乔木愣了一下，大概是没想到他这么轻易就让步了，有些不敢相信地问道，“你的意思是，我们继续？”<br/>    <br/>    “不……”陈兴很快就打破了她的幻想，“我只想确定一个问题，假设我现在的判断是错误的，那么，我们应该听谁的？”<br/>    <br/>    阿乔木当即没了声音，表情有些无奈。<br/>    <br/>    “听谁的？”陈兴逼视着她，再次问道。<br/>    <br/>    “你。”阿乔木面无表情地说道。<br/>    <br/>    “你似乎有些不服气啊？”陈兴说道。<br/>    <br/>    “没有。”阿乔木摇头否认，表情依然木纳。<br/>    <br/>    “我必须提醒你……”陈兴盯着阿乔木的眼睛，态度十分认真，“如果你让我感觉到不爽，我现在就会让你后悔。”<br/>    <br/>    “我会注意的。”阿乔木本能地向后缩了缩。<br/>    <br/>    “走吧。”陈兴挥了下手。阿乔木小心翼翼地点了点头，朝来路走去。<br/>    <br/>    “啊！”经过陈兴身边时，陈兴忽然搭住她肩膀，吓了她一跳。<br/>    <br/>    “老实点儿！”陈兴恶狠狠地说道。<br/>    <br/>    “是，是……”阿乔木低着头朝前走，就像个犯错的孩子。<br/>    <br/>    接下来，两人一前一后，相隔着十余米。看着阿乔木的背影，陈兴忽然觉得，带一个这样女人在身边其实也不错。打架的时候可以在前面挡子弹，睡觉的时候可以用来当垫子，一举两得。<br/>    <br/>    鬼狐虽然长得不错，身材也过得去，但比起阿乔木还是不如。首先，阿乔木胸部和屁股够大，肉感十足。据他观察，胸围应该是34D，和叶阳清燕差不多。尽管比不上叶阳白柳的38F、苏娜的32E，但也够用了。其次，阿乔木肌肉扎实，身体很有弹性，作为床垫的功能十分优秀。其三，阿乔木对他表现出极大的畏惧，有一种说不出的征服感。<br/>    <br/>    思绪之间，前方的阿乔木忽然停住了脚步。<br/>    <br/>    “嗯？”陈兴发出质疑和催促的声音。<br/>    <br/>    “这条路有问题。”阿乔木皱着眉头，略有所思地说道。<br/>    <br/>    “什么问题？”陈兴左右环视，两侧依然是沉寂无声的石头守卫。<br/>    <br/>    “我们刚才在这里发生了战斗。”阿乔木说道。<br/>    <br/>    “你怎么判断的？”陈兴问道，这里的场景十分相似，很难判定具体位置。<br/>    <br/>    “上一场战斗结束后，我走了七千三百五十四步，然后往回走……”阿乔木解释道，“我的脚步很稳定，误差不会超过十米，这是作为沙民的基本生存技能。要知道，在无边无际的大沙洲，距离和方向就是生命。”<br/>    <br/>    两人再往前走了一段，陈兴也感觉到问题。虽然他没有这么精确的距离感，但是从时间和行进速度上看，他们已经超出上一场战斗的位置了。<br/>    <br/>    但是，这里只有一条路，中间没有任何岔口。<br/>    <br/>    鬼打墙？<br/>    <br/>    他想到地球上的灵异传说，有些不寒而栗。<br/>    <br/>    所谓的“鬼打墙”，在科学上被称作“潜意识圆周运动”，就是人在夜晚或郊外行走时，因为分不清方向，自我感知模糊，不知道要往何处走，所以老在原地打转。<br/>    <br/>    而另一种是非科学的解释，就是有神鬼在作怪，影响人类的判断。<br/>    <br/>    他加快脚步朝前走，脸色变得越来越凝重，刚才的邪念早就抛之脑后。<br/>    <br/>    二十分钟后，他们依然停留在通道中，没有看见上来的雾门。无论距离还是时间，都已经远远超出原来的位置。他清晰地记得，他们一共摧毁了九个石像，可是返回的路上，他们没有见到一具残骸。<br/>    <br/>    “晕沙，我们晕沙了……”<br/>    <br/>    阿乔木喃喃自语地说着，声音有些颤抖。虽然陈兴不知道“晕沙”的意思，但从字面上理解，应该和鬼打墙差不多。<br/>    <br/>    “天啊，是赛特，他，他要带走我们！”阿乔木目露惊恐，紧紧地抓着陈兴的手臂。<br/>    <br/>    “冷静点儿。”陈兴拍了拍女人的肩膀，然后原地坐下，仔细思考之前的每一个细节。<br/>    <br/>    分析问题的最佳方式，就是罗列出所有的可能性，然后逐个推敲。<br/>    <br/>    可以确定，他们遭遇了无法理解的事情。<br/>    <br/>    第一种可能性，他们遇到了超自然现象。但是，以他三世为人的经历，鬼打墙的可能性很低。这个世界对超自然现象的解释已经很完善，不存在所谓的“鬼”，只有“灵体”。<br/>    <br/>    灵体是带有物种生命印记的灵能，只是一团能量，没有作怪的能力。<br/>    <br/>    那么还有其它可能性，比如说……<br/>    <br/>    迷宫在移动！<br/>    <br/>    许久过后，他重新站了起来。要验证这一点，就要继续前进，搜集更多的信息。<br/>    <br/>    “走吧。”他起身说道。<br/>    <br/>    “不，赛特要带走我们，我们无力抵抗……”阿乔木似乎受到了惊吓，拼命地摇晃着脑袋，拒绝行动。<br/>    <br/>    “赛特是谁？”陈兴问道。<br/>    <br/>    “大沙洲的统治者，沙民的先祖，无所不能的沙漠之神。”阿乔木说道。<br/>    <br/>    “赛特到底有多可怕，我不清楚，但是我可以向你保证……”陈兴捏起她的下巴，“如果你再不走的话，我会比赛特更可怕。”<br/>    <br/>    阿乔木身体一颤，马上站起来，继续前行。或许在她的世界里，陈兴是比神灵更可怕的存在。<br/>    <br/>    对此，陈兴十分满意。<br/>    <br/>    <br/>
