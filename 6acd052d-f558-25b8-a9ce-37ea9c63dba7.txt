陆寒霆掀起英俊的眼皮透过后视镜看了叶翎一眼，然后将目光落在夏夕绾澄亮的翦瞳上，“我跟她不熟，你随意。”
<br/>
<br/>        这话更像是说给夏夕绾听的，夏夕绾避开了他的目光。
<br/>
<br/>        这时叶翎挽住了夏夕绾的纤臂，“绾绾，我们好久没有见面了，今天晚上你不要回去了，跟我回醉玉欢吧，陪陪我。”
<br/>
<br/>        夏夕绾还没有说话，前面的陆寒霆已经迅速抬眸看来了，“这样不好吧？”
<br/>
<br/>        “哪里不好了？”
<br/>
<br/>        “我记得你们刚在三亚见过，也没有很长时间没见面。”
<br/>
<br/>        “一日不见如隔三秋，陆总，请个假呗，就许你壕掷十二亿为美人，却不许我家绾绾陪闺蜜，什么道理？”
<br/>
<br/>        陆寒霆一阵头疼，他知道夏夕绾这个好闺蜜叶翎是故意的，“如果绾绾不回去，会有很多人想她的，比如奶奶，小圆圆…”
<br/>
<br/>        叶翎无情pass，“那是他们的事儿。”
<br/>
<br/>        陆寒霆不说话了，他深邃的狭眸落在夏夕绾的脸上，眼眸炙烈暗沉。
<br/>
<br/>        夏夕绾当然读懂了他的意思，他不许她去陪叶翎，他要她回家。
<br/>
<br/>        堂堂陆氏大总裁还将奶奶和小圆圆搬出来了，其实都是他的挡箭牌。
<br/>
<br/>        夏夕绾点头，“好，翎翎，我去陪你，陆先生，你跟奶奶说一声，我今晚就不回去了。”
<br/>
<br/>        陆寒霆收回了目光，他看了一下窗外璀璨的霓虹灯，然后掀了掀薄唇，“就住这一晚，明天我来接你。”
<br/>
<br/>        “那不行，绾绾在我那里住多久，那要看我家绾绾的心情。”叶翎说道。
<br/>
<br/>        前面的男人没有再说话，因为抗议也是无效的，总不能跟她的闺蜜抢人吧？
<br/>
<br/>        夏夕绾抬眸看了一眼前面的陆寒霆，他英俊精致的面上并没有什么情绪起伏，白色衬衫的衣袖往上卷了两道，戴着名贵腕表的大手按在方向盘上，从容有力的开着车，优雅而迷人。
<br/>
<br/>        不过夏夕绾知道他生气了，因为她看见他抬手解开了一颗衬衫纽扣，整个人无形之中笼罩上了一层不悦和阴郁。
<br/>
<br/>        夏夕绾移开了目光。
<br/>
<br/>        ……
<br/>
<br/>        醉玉欢。
<br/>
<br/>        夏夕绾和叶翎下了车，叶翎很有礼貌的跟陆寒霆说了一句“谢谢陆总，再见陆总”，然后拉着夏夕绾的小手头也没回就走了。
<br/>
<br/>        两个人上了电梯，叶翎得意道，“这位陆总好像生气了，他这是活该，作为你的娘家人我不帮你出气谁帮你出气？”
<br/>
<br/>        夏夕绾暂时确实不想理陆寒霆，分开冷静一下对彼此都好。
<br/>
<br/>        两个人进了公寓，叶翎笑着倒过来，“看来你从三亚回去的那一晚让陆总很满意啊，你没看见刚才我在车里说让你不回家了陆总那神情，要不是你撑腰，我早就怕了。”
<br/>
<br/>        夏夕绾雪白的小耳垂一红，“我们什么都没有做。”
<br/>
<br/>        “真的假的，绾绾快老实交代，你们发展到哪一步了？”
<br/>
<br/>        “我们…”夏夕绾吞吞吐吐的躲闪道，“我们就…亲亲了…”
