　　她们无法理解，这两个看似关系很好的家伙，为什么突然就动起了手来。
<br/>　　在此之前，这个突然出现的喰种，不还将什造从她们手中救了下来么。
<br/>　　对于黑奈白奈脸上细微的表情，什造没有留意，她缓缓拉开了自己的外套，露出里面满满的黑色匕首。
<br/>　　“你先等我，把她们给解决了，然后我在找机会把你给送出去，你原来住的地方肯定是回不去了，我还等帮你再租个……”
<br/>　　说未说完，什造的脸色僵硬了一下，她感受到脖颈处传来的一股轻微的刺痛感。
<br/>　　“你想吃了我？”侧了侧头，什造目光中带着一股不可置信的神色。
<br/>　　刘哔手中始终凝聚着能量，但很奇怪的，什造根本没有半点挣扎。
<br/>　　她只是看着他，眉头微蹙，目光中却没有丝毫怨恨。
<br/>　　有的只是不解，以及失望。
<br/>　　不知道为什么，看到什造的眼神，刘哔体内静止的心脏，猛然跳动了一下。
<br/>　　就好像，他即将失去一件很宝贵的东西一般，自己这样的动作，应该是很伤她的心吧。
<br/>　　这个家伙，一直在想怎么将他救出去，甚至还打算租个房子，藏下他。
<br/>　　或许，什造一直都将他当做朋友吧，只是自己一心想要吃她，并没有察觉到这一点。
<br/>　　像她那样的经历，有个朋友一定也很难吧，被背叛的时候肯定也很痛吧。
<br/>　　不知道为什么，这一刻的刘哔，突然有些后悔，他的意识，在两个抉择中不断的徘徊着。
<br/>　　他不知道，自己现在应该如何选择，对于眼前的这个人类，他不敢轻易去选择相信。
<br/>　　十年的时间，无数末世人用鲜血铭刻出了一个道理，信任，也就意味着会出现被背叛的可能。
<br/>　　场中的情况安静了下来，似乎是受到刘哔心境的影响，地上森然的白骨手爪逐渐变得模糊，黑奈白奈注意到这一点，目光都是微微一动。
<br/>　　束缚时间快到了，敌人却还没有察觉，她们觉得，趁这个时间或许会有机会冲出去。
<br/>　　反正领悟什造的出现本就是意外的事情，她们的目标，还是冲过眼前的这道出口。
<br/>　　只要能达成目的，这边无论发生什么样的事情，和她们都没有太大关系了。
<br/>　　没有注意到这一点，刘哔的心境逐渐平稳了下来，看着眼前雪白的脖颈，他嘴角露出一抹自嘲的笑容。
<br/>　　什么时候，镰刀也要考虑韭菜的感受了，过了几天相对安定的日子，他居然都会对食物产生仁慈了。
<br/>　　在这一刻，刘哔心中没有了丝毫犹豫，用口中已经初成的尖牙，刺破了什造薄弱的皮肤。
<br/>　　一股香甜的液体流入喉咙，尝到血肉的味道，刘哔双眼中的红色变的更加鲜艳，就好像，要从中滴出血来一般。
<br/>　　正欲将什造脖颈处的血肉撕下，一股危险的感觉从刘哔心底升起，让他皮肤上的汗毛都竖立了起来。
<br/>　　没有任何犹豫，一直在防备着的刘哔，将能量汇聚在手中，向前一伸，抓穿了什造的胸口。
<br/>　　与此同时，什造快速从怀中拔出十余柄黑色匕首，速度极快的向左前方扔去。
<br/>　　两声惨叫响起，不知何时，黑奈白奈竟然挣脱了鬼手技能的束缚，正往他后面的出口俯身急冲着。
<br/>　　而黑奈背后触手状的赫子，正悬在刘哔背后，几乎快要刺进了他的身体。
<br/>　　十余柄匕首出现在空中，阻住了红色的触手赫子，有几把角度刁钻的，越过赫子插入黑奈身体，让她发出一阵惨叫。
<br/>　　而什造这边的状况也不是很好，刘哔误以为她要对自己进行偷袭，故此并没有任何留手，整个手掌，将什造的胸口中央洞穿。
<br/>　　也幸亏是刚刚什造攻击的时候为了调整角度，侧了一下身体，否则的话，这一下击中的，就是她的心脏了。
<br/>　　看着背后半跪在地上黑奈，刘哔像是明白了什么，瞳孔微微一缩。
<br/>　　就在他刚刚分神的短暂时间里，这两双胞胎居然挣脱了他的束缚，反而对自己出手。
<br/>　　再加上他注意力全在什造掏取匕首的动作上，下意识的，他就认为是什造对他发动攻击，可这匕首，明显的是用来帮助自己。
<br/>　　心中一股戾气升腾了起来，刘哔松口，放开了什造，任其无力的跌倒在地面上。
<br/>　　他的心情有些烦躁，但他终归欠下了一些什么。
<br/>　　白奈看到妹妹受到攻击，心中一紧，着忙的赶了过来，想要查看一下自己妹妹的伤势。
<br/>　　一颗清秀的头颅飞起，滚落在了白奈的脚边，惨叫声，戛然而止。
<br/>　　看着自己妹妹的头颅，白奈呆愣了一下，脸色狰狞了起来，但还未等她有什么动作，刘哔的攻击便立刻来到了面前。
<br/>　　“噗嗤。”
<br/>　　没有任何反抗的余地，刘哔抓着白奈的头发，硬生生将她的头颅从其脖子上拔了下来。
<br/>　　顿时，像是喷泉一般，殷红的鲜血从无头尸体上喷涌了出来，虽然白奈并没有出手，但这并没有产生什么影响。
<br/>　　既然是姐妹，什么都要求一样，那么，死法也都一模一样好了。
<br/>　　将手中的头颅扔到一边，刘哔拍了拍手，来到什造身边，蹲下了身体。
<br/>　　“对不起。”眼中带着些许愧疚，刘哔手中出现了一管药剂，动作温柔的，喂什造喝下。
<br/>　　这是抑制丧尸病毒的解药，价值一千积分，只要感染时间不超过十分钟，都可以用这个东西将病毒袪除。
<br/>　　没有任何反抗，事实上什造现在也几乎没有办法反抗了，以她现在虚弱的状态，不昏迷过去就已经很出人意料了。
<br/>　　喂完解药，刘哔又花费一千积分，从商城里兑换了一枚丹药，喂什造吃了进去。
<br/>　　这是疗伤用的丹药，多少会起到一些治疗的效果，如果运气好的话，她还有机会活下来。
<br/>　　之所以没有兑换效果更好的疗伤丹药，是因为刘哔的积分已经不够了，就这两千积分，也都还是击杀黑奈白奈获得的。
<br/>　　做好这一切后，刘哔将什造抱起，找了一处并不显眼的房间放下，微微低下了头。
<br/>　　“在这等我，我去吃点东西，然后就回来，”伏在什造耳边，刘哔留下这句话语，推门走了出去。
<br/>　　门外，依旧是那条走廊，只是出口已经没人守护了。
