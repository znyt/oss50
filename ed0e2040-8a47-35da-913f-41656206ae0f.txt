
<br/>　　<br/>　　    王惊蛰的面前站着七个人，没算青爷，对面首先是齐峥坤，随后是两个年岁与他相仿的青年，都二十几岁左右，身形挺拔穿着身普通的中山装，星目剑眉鼻梁挺拔，气质显得很出众，这两人身后是三个中年，穿着是一身便衣大褂手里拿着折扇，就跟台上说相声的装扮差不多。<br/>　　<br/>　　    王惊蛰的视线穿过他们几人落在了最后一人的身上，他看过去的时候对方也在看着他，两人双目相接没有火花四溅，眼神里都没有透出什么敌意，反应都特别的平淡，就好像走路的时候两个陌生人擦肩而过一样。<br/>　　<br/>　　    但是，平淡并不代表不重视，王惊蛰和对方几乎在同时心中就警惕起来。<br/>　　<br/>　　    今天晚上的遭遇，要比前一天王惊蛰到长安后碰见茅小草被围攻，把她救出来的那幅局面要难得多。<br/>　　<br/>　　    前一夜，围攻茅小草的是一群连名号都不敢报出来的乌合之众，今天这一群，都不用介绍就知道肯定是玄门五脉中的人，只是具体不知道是哪一脉的罢了。<br/>　　<br/>　　    前一夜可能连开胃小菜都算不上，今天的才是正餐，而且还是大餐。<br/>　　<br/>　　    对面七人，王惊蛰身后只站着一位小草姑娘。<br/>　　<br/>　　    短暂的寂静过后，人群中最后那一位，悄然往外面退了几步，他无声无息的退了出去，却仿佛瞬间就吹响了战斗的号角，齐峥坤率先发难，因为他离王惊蛰最近，近到两人之间就只隔了一张桌子的距离。<br/>　　<br/>　　    齐峥坤突然抬起手，他的手不光虎口是老茧，十根手指上全都是粗糙的死皮，甚至离得近了你还能看见他的手指上居然已经被磨的已经没有了指纹，手掌上也没有了掌纹的脉络。从这一幕上就能看得出来，齐峥坤的八卦游龙掌平日里练的有多么用心了。<br/>　　<br/>　　    “咔嚓”齐峥坤抬起手之后，一巴掌就拍向了两人之间的那张桌子上，桌子是红木的大概有七八公分厚百斤左右，虽然看起来很久了但也能看得出来很结实，他一巴掌拍了过去，桌子四个脚顿时应声裂开散架子了。<br/>　　<br/>　　    王惊蛰右手轻轻一推茅小草，将她给推到了一边，然后神色平静的举起胳膊跟对方接了一记，但随即他的手就往下方一拉，将齐峥坤手上的力道全都给卸了下去。<br/>　　<br/>　　    两人之间初一交手，双方的路子就显现了出来，齐峥坤出手刚烈凶猛，王惊蛰则是四两拨千斤般的轻巧，似乎没有分出胜负来。<br/>　　<br/>　　    齐峥坤抿了抿嘴，鼻子里冷哼了一声，右腿朝前迈步，手臂回缩手肘猛的朝着王惊蛰脑袋一侧砸了过去，但对方似乎早就知道他会出此招，王惊蛰没等对方手肘伸过来，身子就朝左边轻轻一晃，等齐峥坤手肘砸过来的时候，他忽然伸出拳头，拇指凸起照着他肘部就用力的点了过去。<br/>　　<br/>　　    “嘶······”齐峥坤倒吸了一口冷气，感觉浑身就是一麻，身体有不到一秒钟是处于麻痹的状态了。<br/>　　<br/>　　    人的手肘关节上，有个点，平日里不小心碰一下都会感觉手臂异常的酸痛，如果被人用力狠狠的敲那么一下，那种酸爽语言都无法形容出来。<br/>　　<br/>　　    高手过招，失之毫厘差之千里，齐峥坤就只有不到一秒钟麻痹的时间，就被王惊蛰把握住了，他直接抬起脚奔着对方的胸口就踹了过去。<br/>　　<br/>　　    “砰”齐峥坤胸口挨了一脚，顿时就朝着后面跌跌撞撞的退了出去，他身后的两人连忙扶住了他，齐峥坤恼羞成怒的吼道：“放开我，再来”<br/>　　<br/>　　    再来，却不是齐峥坤独自上前了，那两个青年看出来单凭他自己似乎很难对付得了王惊蛰，就同时跟齐峥坤冲了过去。<br/>　　<br/>　　    王惊蛰看见对方冲过来之后，又伸手推了下后面的茅小草，将她推到了角落里。<br/>　　<br/>　　    小草姑娘默默的看着他，忽然想到一句话，男人若是粗心的时候，就跟熊瞎子掰玉米一样，掰开一个丢一个，当男人细心的时候就像是破案的福尔摩斯，任何细节都不会遗漏掉。<br/>　　<br/>　　    齐峥坤和另外两人，三方夹击，而且还是在一间面积不过几十平的房间里，看似好像轻而易举的就能围攻住王惊蛰，可以让他束手束脚的，但事实却恰好相反，而且还是令人错愕的。<br/>　　<br/>　　    三人每次出手，在即将要碰触到王惊蛰的时候，他总是能恰好，甚至看似是巧合的躲避过去，就比如那人一拳轰向王惊蛰面门，第二人给他来了个双峰贯耳的时候，王惊蛰极其精准的缩了下脖子然后往后躲了一下，他的面门无恙，两耳也没被拍到，他躲的时机和角度都是严丝合缝的完美，用力过小会被打个正着，用力过大就是浪费了力气，一切都是那么的恰到好处不偏不倚。<br/>　　<br/>　　    王惊蛰略微皱着眉头，抵挡或是躲避着三人的夹击，辗转腾挪之间，几下就被逼得靠近了墙角，齐峥坤撇了撇嘴，那意思是这回还能看到你往哪里跑。<br/>　　<br/>　　    但这时候没想到的是，王惊蛰脚后跟突然往后踢了过去，脚下“啪”的一声踢到了墙面后，人突然就朝着地上给滚了过去，就像那一招非常难看的懒驴打滚一样，王惊蛰从三人身下的缝隙中顿时就冲了出去，随即他转过身体，人仰头背对着地面，右脚连续三次快速的踢中了齐峥坤和另外两人的脚腕。<br/>　　<br/>　　    “噗通”一人跌倒，一人踉跄着被怼到了墙上，齐峥坤反应最快脚腕往后挪了一下，踉跄着躲了出去。<br/>　　<br/>　　    王惊蛰左脚蹬了下地面，身体“唰”的一下瞬间就站了一起，同时收起右脚，脚尖就踢到了旁边散架了的桌子腿上，伸手就把一根木腿给抄在了手中，然后猛地就抡了过去。<br/>　　<br/>　　    “噗”一声闷响，桌子腿正好抡在了举起胳膊硬挡的齐峥坤手上，他呲牙咧嘴的疼的直冒冷汗，揉着手臂连忙往旁边躲了过去。<br/>　　<br/>　　    王惊蛰眯着眼睛，再次拎起木腿照着另外两人劈头盖脸的就砸了过去。<br/>　　<br/>　　    “砰”<br/>　　<br/>　　    “噗”<br/>　　<br/>　　    “咣当······”<br/>　　<br/>　　    几下之后，那两个被逼近墙角的人就宛若被痛打落水狗一样，给揍得抱着脑袋就蹲了下去。<br/>　　<br/>　　    “啪，啪，啪”院子里有人鼓起了掌，然后叹了口气说道：“你们停手吧，别在那丢人现眼了，你们每一次出手都能在人家最完美的算计里，还拿什么跟人交手啊？难道你们没看出来，这是一种精准的卜算之道么，你们怎么出招他都能给你算出来了，还打个屁啊！”
