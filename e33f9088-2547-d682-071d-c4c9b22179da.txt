　　Derrick拎着箱子迅速从住处离开，直奔机场而去。
<br/>　　在Derrick从他住处离开不久，克罗恩的保安队也赶到了Derrick的住处。
<br/>　　克罗恩打开门，伸手拦住了跟在自己身后的其他人，看着留在地上的脚印和行李箱万向轮留在地板上的痕迹，说道：“人已经离开了，时间不超过半个小时，带着一个旅行箱，应该是去机场。走，去机场，了解一下目前最短时间航班，Derrick不会提前就有逃跑计划，他的目的地应该是随机的并且不会使用自己原来的护照，所以必须了解清楚三个小时内所有的航班，分散到各个航空公司取票柜台和自助取票柜台，他可能会进行简易化妆！”
<br/>　　别墅外，四辆黑色的防弹路虎迅速掉头，朝着机场的方向驶去，在赶往机场的路上，克罗恩通知了另外两组人立刻赶往机场。他们这些人就算是分散在各个航空公司的取票柜台和所有取票柜台，在这么大的机场，那么多的游客中要找出可能经过了化妆的Derrick，并不是一件容易的事情，一旦被Derrick通过了安检，登上了飞机，要想再把他抓回来几乎就是不可能的事情。
<br/>　　机场停车场，Derrick拎着行李箱走进停车场厕所，打开行李箱取出一套衣服和一个假头套，走进隔间中，把身上的衣服换掉，戴上假头套后，走出隔间站在洗手台前，跟护照上的照片对照了一下，对着护照亲了一下，说道：“还好早有准备，我他妈都快要爱上我自己了。”
<br/>　　Derrick把自己原来的衣服扔进厕所垃圾桶，随后拖着行李箱进入了航站楼，按照提示牌找到了达美航空公司的取票柜台；可是当Derrick走向取票柜台的时候，却发现在取票柜台前，站着两个警察，正在跟取票柜台中的工作人员了解并且记录着什么。
<br/>　　Derrick没有直接朝着取票柜台走去，而是拖着行李箱朝一台自助取票机走去，这两个出现在取票柜台前的警察给了Derrick一个很不好的感觉，他总觉得这两个警察是奔着自己来的。
<br/>　　自助取票机前，有不少人在等待，而就在Derrick排队等待的时候，克罗恩的保安队也抵达了机场。
<br/>　　排在队伍中的Derrick一直在注意着四周的情况，当看到那几个黑色西装的“维伦恩”生物制药公司保安走进候机楼大厅，Derrick立刻把自己的头埋低下来，并且拖着自己的行李箱挤出队伍，朝着通向停车场的出口走去。Derrick很清楚“维伦恩”生物制药公司保安队里的这些人是什么样的人，这些人都是杀人不眨眼的存在，就是五个自己加起来都打不过他们一个，被他们找到自己，自己根本连逃跑的机会都不会有。
<br/>　　克罗恩走进候机楼大厅，按住挂在耳朵上的耳麦问道：“有没有发现目标？”
<br/>　　“暂时还没发现。”
<br/>　　“这家伙一定在这里。”克罗恩抬头看了眼不断翻滚的航班提示板，立刻通过耳麦对着航站楼中的保安队员，说道：“中国！如果是我，我一定会选择逃去中国，因为那里是最安全的地方，queen的势力深入不到中国境内，我们这些人更可能去中国，那无疑就是去找死！三个人跟我去达美的取票柜台，其他人留意附近情况。”
<br/>　　“妈的，动作还真快。”
<br/>　　Derrick匆匆离开候机楼出发大厅，搭乘电梯回到停车场，从行李箱中取出现金，然后从垃圾桶中捡了两个易拉罐丢进行李箱，把行李箱上锁后丢在停车场一辆汽车边，自己则是朝着他从助手那里借来的汽车快步走去。
<br/>　　拉开车门，钻进汽车，Derrick拿出手机拨打了一个报警电话，说道：“我在候机楼停车场发现一个行李箱，行李箱被上了锁，并且附近没有看到人。”
<br/>　　自从“9.11”事件之后，“反恐”就成了世界各国重中之重，没有任何一个国家愿意重蹈美国“9.11”的覆辙。
<br/>　　接到Derrick的报警电话后，警察很快就抵达了停车场，并且在行李箱附近用警戒线拉圈出一个警戒范围，排爆小组穿着厚重的防爆服和排爆工具进入警戒线内，控制着排爆机器人朝着行李箱靠近过去。
<br/>　　Derrick看着忙成一团的警察，把手机里的手机卡取出，折断扔出汽车，换上一张新的手机卡，发动汽车离开停车场。
<br/>　　而在Derrick驾驶着汽车通过警察设立在停车场出口的警戒，离开停车场的时候，停车场中的保安队员还是发现了Derrick的汽车。
<br/>　　“我看到Derrick了。”
<br/>　　汽车中的保安队员按住耳麦，向克罗恩汇报道：“他驾驶着汽车从停车场离开！”
<br/>　　“立刻跟上他！”
<br/>　　耳麦中传来克罗恩的指示，“我们马上过来。”
<br/>　　Derrick离开机场停机场后不久，就发现了跟在自己后面的汽车，嘴里轻声咒骂了一声，加深脚下的油门，加速向前驶去，但是跟在Derrick后面的汽车却像是一块牛皮糖一样，死死地粘着他，甩都甩不开他。
<br/>　　“Fuck！”
<br/>　　Derrick手在方向盘上拍了一下，打了一把方向盘朝着购物中心的停车场驶去，进入购物中心停车场后，Derrick迅速离开了汽车，扔掉了自己头上的假头套，进入到购物中心内。
<br/>　　“Derrick进入了购物中心，我们正在跟上。”
<br/>　　跟在Derrick汽车背后的保安队员走下汽车，跟上Derrick。
<br/>　　此时的Derrick就像是热锅上的蚂蚁，他知道自己已经被盯上了，而且被彻底盯死了，在这样的情况下，要想逃跑几乎是不可能的。但是他绝对不能落入到玛丽科娃手中，一旦落入玛丽科娃手里，他几乎就是死路一条。
<br/>　　Derrick脑子快速转动着，思考着解决办法，终于Derrick眼中闪过决然之色。如果说还有什么地方是玛丽科娃的手深入不到的，那就只有监狱了！不是联邦监狱，而是“黑蛇”监狱！被称为世界上看守最严密的“黑蛇”监狱，曾经有专门进行越狱挑战的高手尝试过挑战“黑蛇”监狱，但是他失败了，失败的结果就是被高压电电成了焦炭。
<br/>　　那里，可以说是目前Derrick能想到的，最安全的地方！
