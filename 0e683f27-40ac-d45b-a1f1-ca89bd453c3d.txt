　　林峰暗道也是，这陈勾成绩不好，但脑袋不笨，只是没有把心思放在学习上罢了，整天看小说动漫打游戏。
<br/>　　陈勾拿到了百鸟朝凤枪立即就去学习参悟了，林峰就扫了眼他身边的寡妇，对方正在和李广有的没的两人在一起喝咖啡晒太阳非常的惬意。
<br/>　　时不时李广还在那里吹牛逼，说自己之前如何如何牛逼，反正男人和女人聊天就是喜欢炫耀自己的经历过往。
<br/>　　老龙口的生活节奏非常的慢，没有怪物袭击，基本上一天就这样过去，喝喝咖啡，晒晒太阳，无比的惬意。
<br/>　　至于咖啡这种东西，只要是林峰的手下，哪怕是那些挖矿的煤矿工人，都可以喝咖啡。
<br/>　　这绝对是寻常人想都想不到的事情。
<br/>　　这就是林峰给自己手下的人的福利。除了咖啡，还有娃哈哈和蒙牛酸酸乳，这些都是奢侈品，放在曙光基地里，哪怕是一瓶，都绝对可以带走一个女人睡一晚上的好东西。
<br/>　　“你们慢慢聊。”
<br/>　　林峰笑着拍了拍李广的肩膀然后就走了。
<br/>　　他准备回基地去一次。
<br/>　　毕竟好久没有看到苏云老师了。也十分想念苏云老师的美腿和他可口的饭菜了。
<br/>　　在老龙口这几天，虽然吃的不差，但每天都是泡面搭配老干妈外加牛肉罐头和火腿肠，那也很容易吃腻的。
<br/>　　林峰开着军用悍马回到曙光基地，基地内有停车区域，但那一般都是给大人物专用的，而林峰恰恰就是大人物。
<br/>　　大人物三个字，可不是林峰自己给自己脸面上贴上去的。
<br/>　　而是如今林峰的确对得起大人物三个。
<br/>　　排行榜前十的存在，许多人都知道林峰的名字。
<br/>　　当然，认不认得出林峰那就是另外的事情了。
<br/>　　“有没有好心人给点吃的啊，我们一家三口才从老家逃难来到这里。”
<br/>　　林峰虽然已经改变了许多，实力也变强了，但曙光基地大体上还是一如往常那般。
<br/>　　黑铁战士虽然多，但手无缚鸡之力的普通人也不少，这其中还是要以妇女为多了，尤其是带着娃的单身妇女。
<br/>　　林峰看到基地门口一个衣衫褴褛的女子跪在门口在那里求一口吃的，怀里抱着一个七岁的男娃娃，背后还有个大篓子，那篓子里还装着一个一岁的女娃娃。女娃娃正在那里眨巴着天真无邪的大眼睛啃着手指，很显然是饿肚子了想喝奶了。
<br/>　　过路之人要么无视要么还是无视。
<br/>　　“不错，胸部挺大的，不愧是生过娃娃的女人，这就是不一样，这样吧，今天晚上给爷爷我玩一晚上，这包旺旺小小酥就是你的了。要是把爷爷给伺候的舒服了，这一包奶粉可以送你。”
<br/>　　在这个妇人面前站着两个青年，这两人都是四阶的黑铁战士，实力很不错。
<br/>　　“你小子，居然好这口？这种妇人都玩？那边大街上还几个嫩模，姿色比这个强多了，老极品了，昨天小冲还玩到了一个三线明星。”另外一人道。
<br/>　　“你懂个屁，人妻有人妻的韵味，你不知道曹操就好这口，我也好这口。”
<br/>　　另外一人一脸鄙夷。
<br/>　　说着他看着这个跪在地上的妇人。
<br/>　　“怎么样考虑清楚了没有，现在食物可金贵的很，玩一晚上，一包旺旺小小酥外加一包奶粉，绝对的划算。”
<br/>　　妇人看着自己手中的孩子最后一咬牙还是要同意了。
<br/>　　“我家里缺个保姆，每个月三十晶石，不嫌弃的话可以来我家里。”
<br/>　　林峰走了过来，瞥了眼那几个青年然后对着妇女说道。苏云在外面历练，家里倒也的确没有了打扫的人。
<br/>　　“呦呵，小瘪三，怎么，抢老子看中的女人。”
<br/>　　“还装逼，还请保姆，你特么装大少爷啊。。”
<br/>　　两人都冷笑道。显然他们都认为林峰是故弄玄虚，这年头还有人会请保姆？
<br/>　　那个妇女似乎也不敢相信自己听到的耳朵。
<br/>　　林峰看着她道：“我家里缺个打扫清洁的保姆，你愿不愿意。”
<br/>　　“小子，你特么还来真的不成？”
<br/>　　两个人都瞪着眼道。
<br/>　　“故意来破坏爷爷们难得的性质是吗？”
<br/>　　林峰看着他们，“怎么就破坏你们的兴致了，人家还没答应你们的要求。”
<br/>　　“你……”
<br/>　　两人顿时阴沉着脸道。
<br/>　　“小子，给你三个数，最好给我滚远点，不然老子不介意让你看看这个世界的残酷和人性的残暴！”
<br/>　　“趁我心情不错滚远点。”
<br/>　　林峰淡然道。
<br/>　　“呦呵，还特么挺牛逼啊。你当你是周华还是当你是寒鸦了啊。嗯？”
<br/>　　“小子，作业写完了吗你来这里装逼。”
<br/>　　两人对视一眼乐呵了。
<br/>　　砰砰。
<br/>　　下一刻。
<br/>　　两人都倒在地上，嘴巴上一个大手印，这一巴掌给他们打的牙齿都断了。
<br/>　　“你，你特么敢动手？”
<br/>　　“你死定了，在曙光基地动手，你死顶了！”
<br/>　　两人都又气又怒，不过很快他们就冷笑起来。
<br/>　　因为他们看到安保大队的宪兵小队来了。
<br/>　　足足七八个人，全副武装的过来。
<br/>　　“发生什么了？”
<br/>　　其中一个小队长扫了眼他们然后目光落在林峰身上。
<br/>　　“是他！他敢在曙光基地内动手！这就是赤果果的挑衅和践踏基地的律法！这必须要严惩！以儆效尤！！”
<br/>　　“没错！这位军官，一定要把这家伙给抓起来！吊起来弹他的小几几。不然他不知道在基地内动手的严重性！”
<br/>　　两个人都满脸冷笑，这个白痴，死定了！在基地内动手，那绝对是性质极其恶劣的！！！
<br/>　　然而下一刻他们就傻了眼了。
<br/>　　因为这个军官居然给林峰敬礼！
<br/>　　“林峰少将！”
<br/>　　那个小队长此时此刻居然还露出激动的神色。
<br/>　　新开的店被誉为曙光基地第一店，那就是因为来送花篮的人太多了，其中有许多大人物，什么陆勇之类的。
<br/>　　这个小队长他就是巡逻的，所以这基地内的一些屁事他都知道的清清楚楚。
<br/>　　“少，少将？”
<br/>　　那两个青年顿时傻了眼了。
<br/>　　“我现在怀疑你们寻事滋事，甚至在基地内强迫女人卖淫，要对你们进行抓捕!”
<br/>　　小队长立即把矛头对准了那两个青年。
