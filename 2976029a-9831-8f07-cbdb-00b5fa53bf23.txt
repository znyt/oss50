　　洪荒世界，星光黯淡，日月失色，风云倒卷，乾坤倒转。天地之间，煞气弥漫，天空平静得让人害怕，一股强烈的压抑感，扑面而来，让人感到窒息。
<br/>　　洪荒之中，各处名山福地洞天，不少大能者，只感觉眼皮狂跳，心中有一股强烈的危机笼罩而来，让人感到尸山血海的气息。
<br/>　　这一日，洪荒注定不平静。
<br/>　　“轰！”
<br/>　　陡然，洪荒天地一震！
<br/>　　一股强烈的压抑和沉重从太古星空深处传来，弥漫洪荒大地。洪荒之内万物生灵，顿时纷纷感到冥冥之中似有大灾难降临，不由大惊。
<br/>　　“这是怎么回事？好似有大祸降临的感觉？”一位在洞府中闭关的大罗金仙散修惊叫道。
<br/>　　“不好，到底发生何事？怎会让贫道如此心惊肉跳？”一个道人面色狂变，胆战心惊道，“我还是离开此处为妙，过些时日再回来，真不知道发生什么事？”
<br/>　　此人也是一位得到散修高手，能在洪荒生存下来，自然不是什么简单之辈，行事果断，一旦感觉到不妙，立刻离开洞府。
<br/>　　诸如此类之事，洪荒各地都有发生。
<br/>　　而一些大神通者，更是关闭山门，避免招惹因果。
<br/>　　“帝俊、太一小二，尔等何德何能，安敢言执掌洪荒天地？”帝江祖巫看向天空，猛声大喝，“想要立天庭，还需问过我巫族同不同意。”
<br/>　　“呵呵，那我便让尔等看看，我妖族是如何执掌洪荒天地的。”帝俊那霸气的声音响彻而起，回荡八方，道韵流转，整个洪荒天地为之一颤。
<br/>　　“巫族儿郎们，让他们看看我巫族的厉害，战！”帝江祖巫猛然大吼一声，向着妖帝帝俊挥拳杀去。
<br/>　　之后，十二祖巫一起动身，带领巫族成员杀向妖族。同时，东皇太一率领妖族十大妖神，迎向十二祖巫。
<br/>　　大战爆发，不可避免。一时之间厮杀声震耳欲聋，苍天泣血，乌云密布，败鳞残甲漫天飞舞。
<br/>　　战，战，战！
<br/>　　星空坍塌，虚空破碎，杀气逼人，吼声震耳。
<br/>　　洪荒之中，烽烟正浓，剑气冲宵汉，血染的江山。
<br/>　　琴鼓响，铁戈鸣，寒光烁烁照星空。
<br/>　　白发染血，尸骨万千，血泪天涯，铺红天地。
<br/>　　血与骨的开篇，一世凄凉的画卷。
<br/>　　巫妖两族之间的死仇，第一场宿命中的对决，就此展开。
<br/>　　无数准圣大能看着天空之中的战斗，极为凄惨。双方早已杀红了眼，其中最弱的，也是堪比天仙、玄仙境的高手。两族厮杀不断，整个洪荒，都成了两族的战场。搏杀激烈，尸山血海，巫族的凶残让众妖震撼不已。
<br/>　　巫族是最好战的种族，他们以战养战，越是战斗，他们就越是兴奋。身上洒满敌人的鲜血，脚下踏着敌人的尸骨，手里提着敌人的残骸，战斗画面极为凄美。
<br/>　　帝江祖巫对上妖帝帝俊，两者都是一族之长，更是一族之皇，一族之帝，从两者都是帝姓，就可看出，两者之间的雄心。
<br/>　　妖帝帝俊手持金黄色王者之剑，剑光带着帝者威严，一剑又一剑，是那么的浩大辉煌。帝江则是显现出祖巫法相，状如黄囊，赤如丹火，六足四翼，浑敦无面目。他挥动四翼，杀向帝俊，一拳一拳，全是暴力，空间法则的加持，让他与帝俊暂时斗得不分胜负。
<br/>　　东皇太一对上时间祖巫烛九阴与火之祖巫祝融，两者联手，杀向东皇太一。东皇太一虽然失去了混沌钟，但也得到了凶剑混元剑。他手持极品先天灵宝混元剑，剑光森寒，煞气惊天，带着无尽凶气，斩向两大祖巫。
<br/>　　但两大祖巫也并没害怕，对祖巫来说，他们的身体，就是最好的武器。两人挥动拳头，手持法杖，时间之力与火系法则的交织，与太一相持不下。
<br/>　　水之祖巫与风之祖巫对上妖师鲲鹏，水与风的结合，对上同样精通风系法则，吞噬神通的鲲鹏，三人大战不休，谁也不让谁，生死之间的搏杀，水与风的交织，血雨纷飞。
<br/>　　不周山，羲皇伏羲从不周山冲出，对上雨之祖巫玄冥与土之祖巫后土。天地飞雨，雨境之中，蕴含着的是强烈的死亡之力，大雨倾盆，陨落之处，草木枯荣，万物不兴，方圆数十里，寸草不生。谁沾谁死，雨水无情。
<br/>　　后土虽说是最小的一个，但她精通土系法则，只要脚踏大地，就有源源不断地力量加持而来，借助大地之力，她的力量反而最大。同样，大地之力也是防御之力最强的力量。她显现出发祖巫法相，一拳又一拳，空间震动，联合雨之祖巫玄冥，两者压着伏羲打。
<br/>　　其余的祖巫，则是对上十大妖神与其他大罗金仙的妖族，其中自然是祖巫大发神威，妖族的大罗，在祖巫的手下，被大肆屠杀，眨眼之间，便已陨灭百万不止。
<br/>　　两族之战，巫妖之血，聚而成河，渗透到大地之下，融入洪荒大地，进而汇入幽冥血海之中。这让冥河道人心中暗喜不已。
<br/>　　妖族被杀得节节败退，但奇怪的是，娲皇女娲并没有出现在战场之中，而是选择围观，看着眼前被大肆戮杀的妖族高手，尽管女娲心中不好受，想要出手去拯救，但想到了之前自己哥哥伏羲的话，她还是忍住了。
<br/>　　两族力量势均力敌，除了准圣之间的搏杀外，其他境界之间的搏杀，也极为惨烈，各大妖圣化出庞然躯体，展开本命神通，带着强烈杀气，带着复仇之心，一时之间，洪荒之中交战声不断，术法之光滔天。虚空震荡，空间破灭。
<br/>　　这种激烈的战斗，一时之间是很难决出胜负的，这让洪荒之中的准圣强者看着，都觉得眼皮直跳。但他们一个个都纷纷选择围观，两族之战，谁胜谁负对于他们都是不好的，最好就是两族同时损失殆尽，那样才好。
<br/>　　太华山劫魔殿，无尽的杀劫之气，无尽的煞气、凶气、魔气，晦气等劫煞之气，如同龙卷风一般狂涌而来，注入到劫魔殿深处，被万劫魔尊吸收炼化。劫魔殿方圆数万里，全部都笼罩着一层灰色的劫气，让人不得进入。一旦靠近，就会被这凶残的劫煞之气影响，迷红双眼，只会杀戮。
<br/>　　“哈哈哈，好！果然，杀戮，大劫，战斗，是本尊的天堂。来吧，无尽的劫气，回归本尊的怀抱吧。本尊的实力只有在这大劫之中，才能更快地提升。哈哈！”万劫魔尊狂笑，满脸疯狂之色，看着远处两族交战，眼中闪过兴奋之色。
<br/>　　万劫魔尊虽是元始的恶尸，但他的性格一点也不像元始，万劫魔尊更像是元始邪恶的一面，做事猖狂，杀戮不断，毫不畏惧，许多元始只敢在心里想想的事，他都敢去做。
<br/>　　战斗还在持续，巫族的战斗意识太强，搏杀凶猛，若是对上帝俊、太一等准圣，单打独斗显然不是对手，但对上其他妖族，他们却是十分勇猛。
<br/>　　战斗范围在不断扩大，死伤的巫、妖两族成员随着时间的推移也在不断增多。谁也不知道这一战要打到何时？
<br/>　　血与泪的交织，描绘一世的凄凉。
<br/>　　尸与骨的堆积，诉说两族的宿命。
