        结束后，苏漫雪趴在傅昼景的身上，右手的五指轮流敲击着他的胸膛，旁敲侧击地问：“筱颖若是知道，你把我给强了两次，她一定会闹着要跟你离婚吧！”
<br/>
<br/>        “我跟她已经离婚了。”傅昼景的脸色冷了几分，随之一把将苏漫雪从自己身上推开，起身下了床，拿起落在枕头上的浴袍利落地披上。
<br/>
<br/>        苏漫雪撑起身子，跪坐在床上，一脸诧异：“离婚？为什么离婚？”
<br/>
<br/>        “为什么离婚？嗯？我现在正在做的，不就是‘为什么离婚’的原因吗？”傅昼景站在床边，居高临下地斜睨着光着身子的苏漫雪。
<br/>
<br/>        系上浴袍的腰带后，他头也不回地进了浴室。
<br/>
<br/>        这个澡，他洗了许久，但总感觉自己身上还是很脏。
<br/>
<br/>        刚刚，苏漫雪是用的嘴，而他把她幻想成了筱颍，才得以做完。
<br/>
<br/>        蓦然回首，他才发现，自己最幸福的日子，是和筱颍在澳洲一起生活的那三年。
<br/>
<br/>        虽然那个时候，他们有经济压力，但是没有感情负担。
<br/>
<br/>        他和筱颍爱得很纯粹，偶尔会有小摩擦，但和好后，又能过上一段平淡无趣的日子。
<br/>
<br/>        对，平淡、无趣……
<br/>
<br/>        每天就是柴米油盐酱醋茶，他和筱颍，工作的工作，念大学的念大学，凑在一起的时候，逛超市，窝在家里看电影，牵着小爱一起出去散步，生活风平浪静，其实这才是真正的幸福。
<br/>
<br/>        为了救尚轩，他让季思妍二度怀孕，和季思妍举行婚礼，名义上是入赘季家，却背地里唯独没有去跟季思妍领结婚证。
<br/>
<br/>        而季思妍那傻女人，竟然把季氏集团百分之二十五的股份转让给了他，作为他入赘她家的“聘礼”。
<br/>
<br/>        呵！
<br/>
<br/>        作为一个男人，他活到这种地步，真不知道到底是被谁给害的。
<br/>
<br/>        洗完澡后，傅昼景看着洗漱镜里的自己，就像是看着一个令他自己都觉得恶心的陌生人。
<br/>
<br/>        他到底有多可悲？
<br/>
<br/>        一生下来，明明有父亲，却得不到父爱，最后连母亲都弃他而去，宁愿跟着父亲一起去天国，也不愿意留下来抚养他长大。
<br/>
<br/>        结果他被父亲的原配从孤儿院里领回来，进了傅家的大门，却遭受傅家长辈的冷暴力。
<br/>
<br/>        十八岁那年，他和季思妍谈恋爱，最后又被季思妍给抛弃。
<br/>
<br/>        他好像一直都是被别人所抛弃，一直不被任何人所需要……
<br/>
<br/>        只有筱颍，给了他那种被人所需要，被人所爱慕，被人所依赖的感觉。
<br/>
<br/>        可是，他怎么就把筱颍给弄丢了呢？
<br/>
<br/>        “小卫，你怎么哭了？”
<br/>
<br/>        耳畔恍若响起了筱颍的关问声，镜子里的他，眼角竟然真就挂着一颗晶莹剔透的泪。
<br/>
<br/>        呵，他究竟是为谁在哭？
<br/>
<br/>        自己？还是筱颍？
<br/>
<br/>        苏漫雪未等傅昼景洗完澡出来，自己便穿好了衣服，先行离开了。
<br/>
<br/>        她现在虽然是大明星，但是早就不是什么雏菊了，所以也需要生理需求。
<br/>
<br/>        傅昼景这男人的长相不亚于那些当红男明星，所以她觉得自己也不算亏，两个人各有所需，又没有感情负担，对她来说是最好不过了。
<br/>
<br/>        在这圈里混久了，她什么样的男人都见过，也接触过。
<br/>
<br/>        要拿他们一一做对比的话，总体来说，傅夜沉那男人才算得上是极品，还是褒义词的那种极品。
<br/>
<br/>        只可惜，这样极品优质的男人，她得不到。
<br/>
<br/>        其实，苏漫雪的心里，感情那一块，是空虚寂寞的。
<br/>
<br/>        不要对任何一个男人付出真心，唯独只有到手的金钱才是实实在在的。
<br/>
<br/>        苏漫雪一直这样劝诫自己。
<br/>
<br/>        这才在集中办公室里坐那么两个小时，童筱颖就被身边的女同事们给“围攻”了。
<br/>
<br/>        “听说，你跟我们傅大总裁谈过恋爱，后来又被咱们傅大总裁给甩了，那你有没有跟傅大总裁上过床？你为什么会被傅大总裁给甩了啊？傅大总裁有没有给你分手费啊？”女同事甲，毫不避讳，超级直白地三连问。
<br/>
<br/>        童筱颖冷汗涔涔……
<br/>
<br/>        前两个问题，她拒绝回答，最后一个问题，她才想起，大哥给她的那“一个亿”的分手费，她全都捐红十字会了。
<br/>
<br/>        “那个，跟我们的傅大总裁谈恋爱是什么感觉？”女同事乙接着问，“傅总平日里超级高冷啊！你是怎么把傅大总裁追到手的？”
<br/>
<br/>        “你们一个个都很悠闲是吗？”孙莉莉从自己的部长办公室里出来，看到集中办公室里好几个女人围在童筱颖的办公桌旁，一脸严肃地呵斥道。
<br/>
<br/>        下一秒，童筱颖终于呼吸到新鲜空气了，几个喜欢八卦的女同胞，总算是回到了各自的岗位上了。
