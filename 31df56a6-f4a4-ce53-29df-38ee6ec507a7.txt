　　“你真的决定这么做吗？”苏莫目光郑重的望着龙腾，沉声问道，此事非同小可，对方无异于是拿命在保他。
<br/>　　“只能这么做了！”龙腾郑重的点了点头，面色坚定。
<br/>　　“那此秘术，一旦施展之后能否解除？”苏莫再次问道。
<br/>　　“可以，我施展而出，那就只有我能解除。”
<br/>　　龙腾道，言罢，他深深的吸了口气，道：“事不宜迟，我们现在就做吧！”
<br/>　　“嗯！”
<br/>　　苏莫点了点头，他并未拒绝，这的确是对他有百利而无一害，也是目前最好的办法。
<br/>　　随即，苏莫和龙腾两人，再次进入了密室之中。
<br/>　　相生相灭之术，是一种特殊的秘法，需要用到两人的精血以及神魂，经过特殊之法的祭炼，会将两人的性命死死的捆绑在一起。
<br/>　　不过，说起来简单，但是这个秘法却并不简单，整整三个时辰，龙腾才最终完成，将他和苏莫的性命绑在了一起。
<br/>　　“苏莫，祖龙大典还有两个多月就要开始了，已经有各方势力的强者，陆续来到了龙都城，你切勿外出。”密室中，龙腾告诫道，这一次，不仅终极神王殿会有人来，天王宫地王宫等势力都会有人来。
<br/>　　“明白！”苏莫点了点头，接下来一段时间，他会闭门不出，一直闭关。
<br/>　　“嗯，一切小心谨慎！”龙腾再次叮嘱，随即两人有闲聊了几句，龙腾便起身离开。
<br/>　　龙腾离开之后，苏莫关闭了密室的大门，意念一动，进入了自己的空间戒指之中，开始闭关修炼。
<br/>　　他上次在迷宫城得到的宝物，还有一些，这一段时间，他的修为进展很快，距离四品修为，也不远了。
<br/>　　……
<br/>　　吞虚龙府，一座偏殿之中，吞虚龙帝和二长老，分宾主落座。
<br/>　　“轮回之身，万化神剑元神，不败战气元神，修混沌源力……！”吞虚龙帝微眯着眼眸，低声自喃。
<br/>　　“龙帝，您可能已经猜到了，这个所谓的轮回之身，很可能就是屠霄。”二长老面色郑重的说道。
<br/>　　“八九不离十了！”吞虚龙帝低声说道，虽然屠霄展露的元神，没有不败战气元神，而是天梦元神和千幻元神，但是，仅凭万化神剑元神和混沌源力，便基本上能锁定是屠霄了。
<br/>　　毕竟，世界上不太可能有这么巧合的事情，修混沌源力还有万化神剑元神，而且，修为境界也相差不多，那秦陨据说也是战力无比逆天，和屠霄很是吻合。
<br/>　　“若秦陨就是屠霄的话，那此人的元神天赋，未免也太过骇人了吧？”二长老眸中露出深深的惊疑之色，虽然那个秦陨与屠霄有些吻合，但是，他的心底是不愿意相信的。
<br/>　　万化神剑元神、千幻元神、不败战气元神、天梦元神，拥有四种超级元神，这实在是太逆天了。
<br/>　　“虽然难以置信，但是，这可能就是他能击败祖龙塔第十层祖龙的原因吧！”龙帝叹了口气说道，他也感觉难以置信，但是，这也正解释了屠霄能够得到祖龙精血的原因。
<br/>　　“那……龙帝，我们要不要把屠霄擒拿，送给帝一魂或者是拓跋青云？”二长老眸中精光一闪。
<br/>　　若是这么做的话，还能和帝一魂或者是拓跋青云交好，这两人可都是有至强者之资。
<br/>　　“没有什么意义！”龙帝闻言，微微的叹了口气，对于他来说，成为至强者才是目的，只有成为了至强者，他将来才有超脱纪元的可能。
<br/>　　就算帝一魂或者是拓跋青云，成为了至强者，对他也不会有什么大的帮助，因为，至强者也不一定能超脱纪元。

<br/>　　但是，这些人无一例外，都去寻找超脱纪元的办法，或者是离开了荒界，但是，这些人有的已经陨落，有的一去无回，从此消身匿迹。
<br/>　　从来没有谁，能真正的超脱纪元。
<br/>　　当然，或许有至强者真的成功的离开了荒界，得以超脱，但是，没有人能得知答案。
<br/>　　对于龙帝而言，他唯一的目的，就是成为至强者，才有离开的机会。
<br/>　　所以，擒拿屠霄，赠与帝一魂或者是拓跋青云，对于他毫无意义，反而会激怒龙腾，有害无利。
<br/>　　“那就这样不管他吗？”二长老问道，此人可是和龙腾关系莫逆，将来是龙腾得利的臂膀。
<br/>　　“我可没说不管！”龙帝微微笑了笑，眸中闪过一丝诡异的笑意。

<br/>　　“屠霄……屠霄……屠——霄！”紫霄低声自喃，随即脸色微变，眸中闪过犀利的杀机，道：“我敢肯定，此人必然是秦陨！”
