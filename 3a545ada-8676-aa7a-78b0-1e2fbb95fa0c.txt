　　第二天早上叶帝他们依旧是如同往常一样正常训练，吃过早饭后，众人便是再次走入到了训练室内。
<br/>　　“小叶子今天下午就是OG战队的比赛了，你难不成真要晚上发布歌曲啊？”进入比赛室的时候秦霄开口问道。
<br/>　　“是啊，你这要是晚上发布歌曲那就什么用都没有了，人家还等着你给他们打气呢，不过我感觉用歌曲打气怎么看都有些难！”蒋念跟着开口说道。
<br/>　　不过对于叶帝说是用歌手给他们打气这个，他是一点都不相信的，因为在他的印象里叶帝创作出来的歌曲基本上没有这个类型的。
<br/>　　“过一会的吧，等11点左右开直播。”叶帝想了想开口说道。
<br/>　　“不用提前发布一个通告么？”叶颖问道。
<br/>　　“微博上说一声吧，顺带着告诉北北他们，希望这首歌曲能够对他们有些帮助吧。”叶帝笑着说道。
<br/>　　片刻后荣耀战队的微博上便是由叶颖发布了一条微博：
<br/>　　“中午11点大魔王直播发布新单曲！”
<br/>　　这一条微博刚刚发布，便是引发了无数人的关注。
<br/>　　“我去，大魔王又要发新歌了，这个速度真的是有些无敌了。”
<br/>　　“别的歌手一年也就发布是十多首歌曲，感觉大魔王他一个月的工作量就能把别人一年的工作量完成了。”
<br/>　　“大魔王你务点正业吧，咱们马上就要世界赛了，不要总是将心思放到创作上啊，虽然我也很想听你的歌曲。”
<br/>　　“66666666666666，大魔王要出新歌了，期待啊。”
<br/>　　“作为一名荣耀战队的粉丝，我现在希望大魔王还是将目光放到英雄联盟上吧，毕竟咱们的对手还是挺多的，夏季赛可是一定要拿到冠军的啊。”
<br/>　　微博下方无数的网友第一时间开口留言，如今荣耀战队的微博早就已经成为了千万粉丝级别的大V了。
<br/>　　看着微博下方的留言，叶颖也是苦笑的摇了摇头，他还真没怎么看叶帝将心思放到创作歌曲上，每天他可是都在训练呢，这首歌曲也不知道他什么时候捣鼓出来的。
<br/>　　到了现在叶颖已经是越来越看不懂自己的这个弟弟了，之前虽然弟弟很聪明，但是哪里会有现在这个样子，而且高中的时候他也没听说过自己的这个弟弟创作过歌曲啊。
<br/>　　不过总的来说一切都是向着好的方向发展的，所以叶颖倒是也没有太过在意这些东西。
<br/>　　OG战队的队员这个时候也是第一时间知晓了这个消息，看到直播时间是在11点，在他们比赛开始之前，几个小伙子都兴奋的不得了。
<br/>　　“叶哥太够意思了，他这是知道咱们下午有比赛，这才提前开直播的吧。”看着这条微博，OG战队的上单别提有多兴奋了。
<br/>　　“是啊，叶哥太讲究了，感觉瞬间充满了斗志，下午的比赛贼有信心。”辅助跟着开口说道。
<br/>　　“不知道是谁在昨天说的，叶哥晚上直播什么用都没有了，叶哥既然答应了，那肯定是回遵守承诺的。”北北傲娇的说着。
<br/>　　在无数观众的等待中，时间不知不觉的接近了中午11点钟，这个时候虽然叶帝的直播间还没有开启，不过此刻他的直播间里已经是有着好几千万人在那里等待着呢，而且人数还在持续增加着。
<br/>　　而同一刻，叶帝他们这边也是准备的差不多了，在时间到达11点钟的时候，叶帝这边准时的开启了直播。
<br/>　　“今天这个直播算是惊喜吧，感不感动！”开启直播后，叶帝便是开口调侃道。
<br/>　　在叶帝这句话落下的一刹那，直播间里便是刷起了满屏的感动，这个样子倒是让一旁的蒋念艳羡不已。
<br/>　　他开直播的时候，直播间里的那些粉丝除了嘲讽就是嘲讽，哪里会像叶帝直播间的粉丝这么心齐，叶帝一句话他们就都刷感动。
<br/>　　“之前已经在微博上说了，今天要发布一首新的单曲，其实这首歌曲也是准备的非常突然，大家也都知道OG战队被我们收购的这一件事，不过最近一直在忙碌着我们荣耀战队的比赛，所以对他们倒是没怎么管理。”
<br/>　　“昨天北北给我打电话说OG战队的几名弟弟有些紧张，让我给他们打打气，其实一开始我是比较诧异的，毕竟他们已经在联赛上打了这么多场了，心里素质应该是有的，怎么可能面对一个线下赛就这么紧张呢。”
<br/>　　“后来我想了想终于搞明白了，他们是荣耀战队旗下战队，珠玉在前，所以我的这几名弟弟不想拖我们的后腿，没有闯进季后赛，他们想在这个线下赛中发光发热。”
<br/>　　“我现在想说的是大家放宽心态打就好，不要给自己太大的压力，而今天的这首歌曲也是我为OG战队的弟弟，还有所有的选手创作的一首歌曲，希望大家听到这首歌曲后，能够更加的相信自己，能够更加的自信。”
<br/>　　“要不然我们在等等吧，现在应该很多兄弟都还没下班，在等半个小时怎么样，咱们就先这么聊聊天。”叶帝开口说道。
<br/>　　听着叶帝的话，直播间里的众多观众纷纷应好。
<br/>　　而OG战队的成员这一刻心下也都是稍稍松了一口气，他们的内心活动还真被叶帝给猜中了，荣耀战队的成绩那么好，他们却连季后赛都没闯进去，几人都怕自己在下个赛季被卖出战队，所以这个时候都很忐忑。
<br/>　　在这种情况下他们自然是想在线下赛中取得一个好的成绩，这样的话之后还有留在战队的资本。
<br/>　　不过在听到叶帝的这句话后，他们知道自己是多想了，既然叶帝都已经这么说了，那么就代表着对方并不在意这个。
<br/>　　这一刻他们已经想到明年叶帝过来指导他们，他们实力突飞猛进的样子了，而对于叶帝所说的那首歌曲，他们也是越发的好奇了起来。
<br/>　　时间慢慢的走着，不知不觉间便是到达了中午11点30分，这个时间正好是下班下课的时间，几乎在到达这个时间的一刹那，直播间里的观众人数便是再次疯长了起来。
