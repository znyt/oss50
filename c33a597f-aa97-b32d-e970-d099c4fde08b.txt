　　“这和尚到底是什么来路？真佛镇杀印，金刚琉璃身，这可都是古老佛门之中不同分支的秘术，唯有地位极高的佛尊，才有资格修炼其一，这和尚居然都能够修炼？”
<br/>　　楚风眠看到这转轮和尚身上的显现而出的琉璃颜色，不由的眉头一皱道。
<br/>　　不管是真佛镇杀印，还是金刚琉璃身，都是佛门之中级别极高的秘术，而且还属于不同的分支。
<br/>　　哪怕是地位极高的佛尊，都难以同时修行，眼前这转轮和尚，居然是将这两道秘术，全部修行。
<br/>　　而且不管是这真佛镇杀印，还是金刚琉璃身，此人都已经是将这两道秘术修炼到了极高的地步。
<br/>　　这转轮和尚，在净土星域之中，绝对是有着极高的地位，不然是没有资格，同时修炼如此之多的高深佛门秘术的。
<br/>　　杀了此人，可能会有些麻烦。
<br/>　　不过事到如今，楚风眠也不怕什么麻烦了，这转轮和尚，已经是自己站出来跟楚风眠为敌了。
<br/>　　还差一点就阴了楚风眠，让楚风眠陷入万劫不复的境地，哪怕是这转轮和尚有着惊天的来历，今日楚风眠也要杀了他。
<br/>　　楚风眠眼神之中的杀机，越渐浓郁。
<br/>　　另一边。
<br/>　　那被楚风眠一剑逼退出去的转轮和尚，如今脸色也是无比的震惊。
<br/>　　“仙术！这是仙术之力！”
<br/>　　这真佛镇杀印的威力，几乎只在仙术之下，仙术之下，近乎无敌。
<br/>　　现在这真佛镇杀印居然是被楚风眠剑锋硬生生的打破，刚刚的那一剑，几乎是媲美仙术的威力。
<br/>　　以八劫古帝的境界，足以打出仙术，这已经是无比惊世骇俗了。
<br/>　　仙术的威力强大，但是对武道的领悟也极为苛刻，对寻常的九劫古帝来说，哪怕是他们真正得到的仙术，也施展不出来。
<br/>　　毕竟唯有武道领悟，达到天人境的层次，在足以施展出仙术来。
<br/>　　眼前楚风眠能够施展出仙术来，便是证明楚风眠的武道，已经是足以媲美天人境的武道。
<br/>　　“天人境的武道领悟！”
<br/>　　小耀王寒夜死死的盯着楚风眠，他也看的出来，楚风眠施展出剑术的玄妙。
<br/>　　小耀王寒夜在圣龙秘境的圣子之中，几乎是武道领悟最高的一尊人物，虽然没有得到力量传承，一身武道领悟已经是不亚于曾经的耀王。
<br/>　　因此便是有着小耀王这个称呼，可哪怕是曾经的耀王，也没有将武道领悟，达到天人境的层次。
<br/>　　眼前的楚风眠，居然做得到。
<br/>　　“转轮和尚，我来助你一臂之力！”
<br/>　　突然之间小耀王寒夜长啸一声，身躯之中灵力暴起，凝聚在了他的拳头之上，当即一拳向着楚风眠打出。
<br/>　　“荣耀神拳！”
<br/>　　在这一拳打出的一刻，从小耀王寒夜的背后，都是显现出了一尊虚影，这一尊虚影高大威猛，一举一动之间，都蕴含着极致的武意。
<br/>　　这一尊虚影身上蕴含的力量，几乎是跟小耀王寒夜身上蕴含的力量，一模一样，一拳打出，如同是这小耀王寒夜跟这背后虚影，一齐出手。
<br/>　　两股力量，同时从这一拳之中爆发而出，当即向着楚风眠轰击而来。
<br/>　　这一道虚影，不是其他，正是曾经的耀王，这小耀王寒夜，继承了耀王的传承，单单是武道已经是不在曾经的耀王之下。
<br/>　　一拳轰击而出，整个通道之中，都充斥着澎湃的武道之力，要以这武意，来硬生生将楚风眠所击毙轰杀。
<br/>　　楚风眠看到这蕴含极致武意的一拳轰击而来，脸色丝毫不动，他手持灵剑，便是天下间，没有任何事，足以是让楚风眠惧怕。
<br/>　　拳风接近轰击而来的一刻，楚风眠手中剑锋动了，一道剑光，冲天而起，剑光闪烁瞬间，宁烈的血腥味，当即是爆发了出来。
<br/>　　杀戮剑术！
<br/>　　楚风眠打出这一剑的一刻，他便是如同化身为了一尊古老的杀神，一剑之下，尽数灭亡。
<br/>　　楚风眠一路而来，不知道是斩杀过多少强者，敌手，所有的杀意，在这一刻爆发而出，凝聚在剑锋之上，打出这一招杀戮剑术！
<br/>　　血红色的剑光，足有千丈之长，笼罩了整个通道之中。
<br/>　　那小耀王寒夜打出的荣耀之拳，跟这杀戮剑术碰撞的瞬间，被当即打破，反震的力量，将小耀王寒夜当即是镇退出去。
<br/>　　一招击退了这小耀王寒夜，楚风眠也没有放了他的打算。
<br/>　　这三个人，对楚风眠来说都是祸端，能够杀其中一个，楚风眠的压力也会少的多。
<br/>　　承受追击，楚风眠的身躯连连闪烁，猛然之间又是一道剑光打出，直指这小耀王寒夜。
<br/>　　“戒律真言！”
<br/>　　那转轮和尚，突然之间来到了楚风眠的面前，抵挡住了楚风眠，在他的口中，自言自语一般念出了无数古老的梵文。
<br/>　　这些古来的梵文，随着转轮和尚开口念出，从空中当即凝聚，组成了一道古老的大阵，当即压迫在了楚风眠的身上。
<br/>　　楚风眠感觉到，他身上的灵力都在这梵文凝聚出的阵法压迫之下，有些不听使唤。
<br/>　　就连楚风眠的心思，如今都一下子混乱了起来，有着一丝不寻常的慌张。
<br/>　　这古老梵文，居然是足以压迫住楚风眠的力量，甚至是搅乱楚风眠的洗刷你。
<br/>　　“好厉害的真言！怪不得都说佛门武者，擅长蛊惑人心，在古老的时代之中，无数强者都被这真言所镇压，阪依佛门，以我的道心，都受到了几分影响。”
<br/>　　楚风眠心思一动，也察觉出这真言的厉害。
<br/>　　楚风眠二世为人，道心稳固到了一种不可思议的地步，当初在那心痕峰上，那一尊心魔之王，都无法扰乱楚风眠的道心，反而是都被楚风眠收复。
<br/>　　单单是论起道心，哪怕是天人境成仙的人物，楚风眠都毫不逊色。
<br/>　　可纵然如此，在刚刚听到那佛门真言的一刻，楚风眠都感觉到了几分慌张，灵力受到压制。
<br/>　　这转轮和尚的佛门真言，如果出其不意，没有防备的情况下。
<br/>　　哪怕是九劫古帝，只怕都有可能会受到这佛门真言的影响之下，失去心智，阪依佛门，成为佛门的奴隶。
