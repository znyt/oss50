　　看着蒋念的出装，叶帝也没有去消耗他的心思了，布甲四红这样的装备自己如果真过去顶着小兵消耗他，就算是自己有护盾，那也是非常亏的。

<br/>　　2.5秒的时间已经足够对方击杀自己了，不会有丝毫悬念的。
<br/>　　“夜雪做好视野，要保证寒冰在朝中上释放大招的时候咱们能看到。”叶帝对着夜雪狂歌说道。
<br/>　　“放心！”对于寒冰他们有着自己的一套应对办法。

<br/>　　选择寒冰另外的一个优势就是，在前期寒冰升级自己的e技能的时候，他是可以探查到自家打野的视野的，从而让自家打野暴露在对方的行迹之下。
<br/>　　而上路的香蕉利用艾克这个英雄和泰坦对拼，倒是也没有赚到太大的便宜，两人依旧是在平稳发育。
<br/>　　就在时间到达4分39秒的时候，场上终于是爆发出了人头，双方打野齐齐来到下路，最终螳螂这边拿到布隆一血的人头，而萝卜的圣枪游侠则是收下了对方秦霄的娜美的人头。

<br/>　　“不过这一边让螳螂这个英雄拿到一血的人头，接下来飞刀可就真的要起飞了，而且你注意到了没有，飞刀的螳螂并不是主w，而是主q的打法，这么看来他在六级肯定是要进化自己的q技能的，到时候他的伤害可是非常的高的，在小团战中绝对能够收割人头。”林雅楠跟着开口说道。

<br/>　　当时间到达23分钟的时候，双方的人头比已经是来到了16:11，可以说双方真的是一直在找机会。

<br/>　　不过最终还是白队这边略微吃亏了一些，实在是螳螂这个英雄太肥了，一个q技能的伤害直接能打掉圣枪游侠一大截血量，让萝卜这边根本不敢上前输出。
<br/>　　最终双方的第二局比赛在时间到达39分钟的时候便是以蓝队的获胜结束，飞刀毫无悬念的拿下了本场比赛的mvp。

<br/>　　“蓝白两队的几个新人进步都很大，原本萝卜和茄子这两个一直被针对的点，现在也可以独当一面了，而且这一局的比赛中，他们的下路其实并没有被霄神他们给压制住，萝卜也是打出自信来了。”林雅楠跟着开口说道。
<br/>　　“这样的话双方再次战成平局，接下来双方将会进行bo3的最后一场比赛，决胜局。”
