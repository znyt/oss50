　　云儿叹道：“我何尝不是这样想，可是，你们真的不是五通神的对手……”
<br/>　　“这个，办法总是有的。”叶少阳摸着下巴，沉吟半晌，说道：“办法慢慢想。对了，之前我偷听到你跟五通神的对话，说什么镇长又给他选了一批姑娘，让他从中挑选一个，三天之后送给他？”
<br/>　　云儿点点头，叹道：“只是不知道是哪家的女子，又要遭受我的经历……”
<br/>　　“不会了，”叶少阳道，“我不会给他这个机会。”
<br/>　　叶少阳跟毛小方商量了一下，觉得先要找出这被选出的几十个姑娘在什么地方，然后暗中监督，找机会从五通神手上把这姑娘救下来，毕竟现在一时半会也不能把五通神怎么样，只能从从长计议。
<br/>　　“对了，关于那个蛤蟆谷，里面是不是有个古墓？”叶少阳问云儿，“你对这件事了解吗？”
<br/>　　云儿想了一会，说道：“我只是听五通神提起过，那古墓里，似乎镇压着一尊邪神……”
<br/>　　“邪神？”叶少阳震惊。
<br/>　　“五通神是这么说的，说他的职责，就是镇压蛤蟆谷，让这里的……好像说的是龙脉，让龙脉无法成形，这邪神就永远都无法出来，否则邪神降临，将生灵涂炭。”
<br/>　　这么严重……
<br/>　　“我明白了。”毛小方拍着脑门说道，“少阳子，你看这五通神的法力怎么样？”
<br/>　　“并没有特别强。”叶少阳想了一下说道，之前跟五通神打了一架，虽然自己跟毛小方吃亏了，不过自己以七星龙泉剑加上阴阳镜，将五通神锁死了一段时间，虽然这两件法器都是九段光的神器，不过自己目前毕竟只有真人的法力，并没有办法将两件法器的灵力完全发挥出来，依然可以跟五通神抗衡一段时间。这也反面验证了五通神的实力，并非自己最初以为的那么强。
<br/>　　“我以前对付过一个邪神，是一只十二年蝉成精，虽然没有正神的位置，但实力在五通神之上。”
<br/>　　对付七奶奶的时候，自己当时是准地仙牌位，以一己之力仍然不是七奶奶的对手，最后还是借助樱花大阵，和小伙伴们一起的努力，才击杀七奶奶。这五通神的实力，比七奶奶要差得远。
<br/>　　“地仙实力，可与五通神一战。”叶少阳得出这个结论。
<br/>　　毛小方一点也不吃惊，点着头说道：“我也是这么想，之前我们都觉得蛤蟆谷里有特别强大的邪物，五通神负责镇守它，实力至少比这邪物不差，现在看来，五通神之所以能镇压蛤蟆谷，靠的其实不是自己的实力，而是利用了某种东西……这东西我们不知道，但肯定存在，五通神掌控着这种东西，能封印古墓里那位，让它不能出来，你觉得有没有这种可能？”
<br/>　　叶少阳点点头，在听了云儿的描述之后，他内心也产生了这种猜测。
<br/>　　三人聊了一会，叶少阳和毛小方要出门去打探消息，让云儿一个人在房间呆着。虽然叶少阳一再保证五通神不会来找她，云儿还是害怕，要跟他们一起出去，叶少阳也只好带着她。
<br/>　　“对了，你家就在这附近，会不会被人认出来？”叶少阳想起这茬，问道。
<br/>　　“应该不会，我家离这里还有二十里，镇上的人我都不认识，我那时候只有十六岁，如今三年过去了，我变了很多，就算有熟人见到，也不会认出来的。”云儿很放心。
<br/>　　云儿在山神庙里呆了三年，没有下过山，如今来到镇上，有一种回到人间的感觉，不禁触景生情，想到这三年来的经历，心头很不是滋味，叶少阳看出她情绪不对，安慰了一番。
<br/>　　下午的镇上，还是有一些人的，看打扮多是附近的山民，在镇上采购和出售各种货物。
<br/>　　一个身穿旗袍的女子，在人群中走过，很是惹眼，加上身材很棒，在叶少阳三人前面走过，叶少阳忍不住多看了几眼，突然产生了一种很想看看她长什么样子的冲动，于是趁着姑娘在一家绸缎庄里驻足停留的时候，走到了姑娘的前面，等她出来的时候，特意回头看了一眼，然后，当场就石化了。
<br/>　　美女走了过来，也发现了叶少阳奇怪的表情，瞥了他一眼，眉头微微一皱，然后对他嫣然一笑，径直走了过去。
<br/>　　叶少阳石化了有半分钟，猛然回过神来，冲了过去，对着美女背影大喊一声：“美华！”
<br/>　　这美女不别人，正是他第一个收下的鬼仆美华！！
<br/>　　美华转过头来，诧异地看着叶少阳。
<br/>　　叶少阳冲过去，一把按住她的肩膀，几乎要哭出来了。“原来你也来了，你是来找我的吗，能在这里遇到你，真好真好……”
<br/>　　见到美华，叶少阳激动得有点语无伦次了。
<br/>　　美华皱着眉头打量了半天，道：“你认识我？”
<br/>　　“废话！”叶少阳双手捧起美华的脸颊，用力揉起来，随即又把她抱在怀里，刚要说点什么，突然觉得下身一痛，是被美华用膝盖顶了一下，疼的叶少阳当场就哭了，弯腰蹲在地上，那滋味……
<br/>　　“为什么打人！”毛小方上前说道。
<br/>　　“为什么？你看不见他轻薄与我吗，这样的人不该打。”
<br/>　　毛小方也是语塞，上前拍着叶少阳的肩膀说道：“虽然人家长得漂亮，你可别乱来啊。”
<br/>　　叶少阳在地上蹲了半天，总算缓过来，走上前去看美华，看了好半天，确定的确是美华！自己没认错人！
<br/>　　“你不认识我了？”
<br/>　　“登徒子！走开！若不是在这闹市，我当场让你身死！”美华冷冷说道，转身要走。叶少阳哪里肯放过，冲上去挡住她的去路，说道：“你就说，你是不是叫美华？”
<br/>　　美华一下愣住，想起之前这个人叫过自己名字，当时还很纳闷，只是后来叶少阳的表现太像是占便宜的，让她忘了这件事，此刻再一次听到自己的名字从他口中叫出来，忍不住又打量了一番，皱眉说道：“我好像不认识你。”
