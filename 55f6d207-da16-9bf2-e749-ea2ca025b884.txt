　　万仙阵前，面对那浩瀚神威，元始面不改色，淡然处之。老子在牛背上，将这些东西置若罔闻，手持拐杖，昏昏欲睡。
<br/>　　只见对方万仙阵中，通天教主骑着奎牛，居中而立，多宝道人、龟灵圣母、无当圣母、金光仙、虬首仙、灵牙仙、乌云仙、毗卢仙、金箍仙、长耳定光仙等仙人站立两旁，通天教主环顾四周，扬声道：
<br/>　　“两位道兄既然如今来到此地，便是入了劫中，却也不必多讲，就请入我阵中，各凭所学，印证一二！”
<br/>　　元始看了一眼通天，淡淡道：“放心，本座既然已经到此，便会如你所愿，做过一场才是。”说着他转头看向一旁昏昏欲睡的老子，道：
<br/>　　“二弟此次就不用出手了。”
<br/>　　通天教主道：“甚好，自小我便一直被大兄压在身下，大兄什么都比我强，每次与大兄切磋，我都是大败，此次我集截教万仙之力，定要与大兄一决高低，这也许是我最后的机会！”
<br/>　　元始目光复杂地看着他，良久，才缓缓道：“你这又是何必呢，你也知道，就算是借此万仙大阵，你也很难与吾相抗。”
<br/>　　“不必多说了！”
<br/>　　通天教主大声道，“你我之间定要决出一个胜负，此次并不止你我二人之争，更是你我两教道统之争，封神一劫中，你阐教弟子屠我截教弟子，这番因果，便在此番大阵之中一同结算吧。”
<br/>　　“如此……也好。”
<br/>　　元始见劝他不过，便点了点头，算是同意。做一场也好，让他知道这其中的差距，也许未来他会更加刻苦一些。
<br/>　　“三弟，你且将你万仙大阵布出，我阐教弟子自会前来一会。”
<br/>　　通天教主在奎牛上，点了点头，龟灵圣母将手中龙虎玉如意高高一举，数名道人摇旗向着东、南、西、北四方指画，钟鼓齐鸣，瞬间出现数十名道人，乃是七政二十八星宿：角木蛟、亢金龙、氐土絡、房日兔、心月狐、尾火虎、箕水豹、牛金牛、女士蝠、虚日鼠、危月燕、奎木狼等。
<br/>　　二十八名道人分别身穿青黄赤白黑五色道袍，整整齐齐，飞霞流转，电舞雷鸣，分为四维方位在阵前排开站立。
<br/>　　其余众多仙人也俱皆各自按照通天教主所安排的，按照不同方位，摆放出不同的大阵，大阵一出，遮天蔽日，天地间无数煞气向着界牌关前充斥而来，气势简直是要将整个洪荒都覆盖进去，随处都是灰蒙蒙一片，死寂的气氛，充斥着洪荒天地，连太阳光都畏惧地躲了起来，不敢出现。
<br/>　　三清之中，元始擅长炼器、太上擅长炼丹、通天却是擅长这布阵之术，截教的大阵，几乎都是通天钻研出来的，更是洪荒天下一绝。
<br/>　　万仙大阵之中，通天教主立于八卦台上，掌管大阵核心。
<br/>　　万仙阵并不只是一座大阵那般简单，其实是大阵之中又套有小阵，阵中套阵，无限循环，每一座阵均有各自的妙处，最终何为一座万仙阵！
<br/>　　如之前三霄娘娘布置的九曲黄河大阵，在汜水关前大展神威，将阐教众仙顶上三花尽皆削去，掉落修为，如今在这万仙阵中，也被其他修士布置了出来。
<br/>　　又有十绝天君的十绝阵，此刻亦被布置出来，置于万仙阵中，充当一环。当初令阐教众仙头疼的十绝阵，此刻竟然如沧海一粟，毫不起眼。
<br/>　　又有两仪阵、太极阵、四象阵、三才阵、五行阵、七星阵、八卦阵、九宫阵、火龙阵、子母天魔玄阴阵、都天烈火仙阵、大须弥正反九宫仙阵、飞雨阵、九鼎阵、银雨阵等等，几乎截教门下，每个仙人，或者几个仙人一起，俱皆主持着一座大阵。
<br/>　　成千上万座大阵，几乎每一座都是通天教主亲传，威力无边。
<br/>　　如此成千上万座大阵合在一切，经通天教主改变，玄妙地合为一整体，由教主亲自主持！其中，更有教主的诛仙剑阵，那可是洪荒第一杀伐绝阵，堪称“非四圣不可破”的绝世凶阵，恐怖无比！
<br/>　　比之当年巫妖大战时的三百六十周天星辰大阵、十二都天神煞大阵，威力也有过之而无不及，毕竟当年的两座大阵，主持之人也只是准圣的修为。
<br/>　　哪里比得上如今圣人三重天修为的通天教主！
<br/>　　万仙阵的威势，惊天动地，洪荒之中瞬间掀起一股滔天威势，大阵一出，瞬间惊动整个洪荒！
<br/>　　洪荒之中诸多大能，或闭关的，或争斗的，此刻都停下手中之事，神念横空，关注到界牌关前，感受着大阵的凶威。
<br/>　　诸神在打量着这万仙大阵，心中却在暗暗吃惊。
<br/>　　好厉害的大阵！
<br/>　　万仙阵中，那一位位截教门人，一座座阵法相辅相成，人人都是凶神，人人都是杀神，煞气透过三十三天，直冲天外，甚至就连幽冥界中，都受到此阵影响。
<br/>　　这等阵法，就算是当年见过巫妖大战的大能们，都不得不佩服通天的阵法天赋，此阵凶威，早已超越周天星辰大阵与都天神煞大阵了。
<br/>　　真是亿万年难得一见！
<br/>　　大阵之外，元始依旧面色淡然，无人能够看清他心中所想。虽是如此，但他心中却也不得不佩服自己这位三弟，通天在阵法上的造诣，已经达到大道的领域了，所谓一阵生世界，也不过如此。
<br/>　　是以，元始当下便毫不吝啬地开口赞叹道：“这万仙阵果然不愧万仙之名，所谓超越仙之巅峰，已为大道，此阵已可称之为道阵了。大道之阵，一阵生世界，此阵乃万千大阵组成，更是万千世界，正应了三千世界组成一大千世界。此万仙阵便是一大千世界，其内众多小阵，堪比小千世界。万千大道，果然殊途而同归，一致而百虑！”
<br/>　　“凭此一阵，便已超越了巫族的都天神煞大阵与妖族的周天星辰大阵！”最终，元始下了如此定义。
<br/>　　听到元始的称赞，阵中的通天教主也露出了开心的笑容，紧接着便谦虚道：“大兄谬赞了！”
<br/>　　“本座可不会随意夸人，三弟你有这份本事！”元始淡淡道。
<br/>　　虽说如此，但元始也毫不畏惧，不说他那混元圆满的修为，就说这万千阵道世界又如何，他体内又何止万千世界？一个细胞便是一方世界，他的身体便是一个宇宙，早已包含了三千大千世界，百万中千世界，亿万小千世界，还有更多繁如星辰的洞天小世界等。
<br/>　　再说，阵道能够生世界，他的炼器一道依然能够炼出世界来，他随手一炼，便能够星辰珠，其内包含万千星辰，宛如万千世界。
<br/>　　就连太上，元始心中亦知晓，他的炼丹一样能够在丹中炼出世界来，那是丹道世界，炼丹，炼的便是自己的大道。一如元始的炼器，通天的布阵，到了圣人的境界，哪还会如仙人那般，学习这些副职业，他们所做的，都是为了悟道。
<br/>　　所以，元始刚刚会说万道殊途而同归，不论是炼丹、炼器、布阵，还是其他，最后都是为了大道本源。
<br/>　　本源为一，万道归一，一便是万，万便是一！
<br/>　　话虽如此，但能够领悟到这一步的，却是少之又少！
<br/>　　万仙已出，大阵已成，大阵之中，走出一人，为角木蛟柏林，他脚踏青云，手持一六棱长枪，遥指阐教众仙，口中大喝道：
<br/>　　“谁敢会我四维七政之阵？”
<br/>(本章完)
