　　第731章
<br/>　　魏牧冲一生过得并不愉悦。他是寒门出身，父母早亡，由兄嫂拉扯大。寄人篱下的滋味自然不好受，嫂子嘴硬心软，平常的白眼和冷嘲热讽也算不得什么。关键是魏牧冲务农不行，经商不行，平日里便只是读书读书读书。
<br/>　　读来读去，眼见豪强跋扈，民不聊生，读出个为国为民，读出个江山社稷……最后，读出个进士出身。
<br/>　　怀抱赤子之心进入庙堂，想着是为国为民做件事的。抱着这心思上书参了崔相一本，历数崔相十桩大罪。结果根本没有搅起什么风浪，他只是被发落到一清水衙门，做些抄抄写写的小事。
<br/>　　一肚子锦绣文章，为国为民的大愿，最后无处施展，整日介以酒浇愁，于是愁上加愁。
<br/>　　也就是这次相府被扳倒，他有了重新被启用的机会。其实他不算李乐天的亲信，李乐天几乎没有亲信。这次朝堂大清洗，李乐天突然发现了还有这么个人，在崔相掌权时，就敢上书参奏。这是个忠臣呐……
<br/>　　于是他被从清水衙门中拎出来，丢到汝南城做事，算是掌了实权的官职。一城之主，军政税一把抓，是有甩开膀子做一番事业的机会。但他哪里知道，一人得道，鸡犬升天。他起了势，他的穷亲戚也跟着发迹。这些都是过惯了苦日子的人，大半生受人白眼，受人欺凌，如今好不容易抖起来，还不得连本带利，将这些年受的苦都还回来。
<br/>　　于是汝南，变成了现在的汝南。
<br/>　　面对程大雷的质问，他只感觉一阵阵头晕眼花，耳边嗡嗡作响。随之，他大笑一声，笑得有几分凄厉。
<br/>　　“程当家的话未必没有道理，是魏某先入为主，小看了天下英雄。若有机会，倒想好好向程当家请教，听君一席话当胜读十年书。”
<br/>　　程大雷点点头：“放心，我不会给你这个机会的。”
<br/>　　魏牧冲不以为意，道：“从前程当家做的事功过自有世人评说，世人心底自然有杆秤，魏某也不敢妄下断言。但魏某确信一件事，现在杀了你，为国为民都是有利的。”
<br/>　　“原来还是要动手呐。”程大雷冷笑一声，又从怀里掏出一样物事。
<br/>　　金光灿烂，耀人眼晕，魏牧冲下意识就想跪倒。
<br/>　　“瞧见没有，这是陛下颁下来的圣旨，你敢向我动手，你眼里还有王法么，你眼里还有陛下么。”
<br/>　　魏牧冲并不怀疑这圣旨是真实的，刚才想要屈膝跪倒，不过是下意识的动作而已。而现在，他脸上并无多少反应。
<br/>　　“只要于国于民有利，王法又如何，陛下又如何。只要能杀了你，便是事后陛下将我千刀万剐，我也算死得其所。正如程当家刚才所说，魏某的前半生或许的确没做什么事。那今日就用这条命做一些事。”
<br/>　　程大雷忍不住为他拍手叫好：“当真是忠心耿耿照汗青，我倒是想问一句，假若我死在此地，西北凉州会如何，京州长安会如何。不敢夸下海口，十年大乱是跑不了的。魏城主，这也算为国为民么？”
<br/>　　魏牧冲一楞，一时不知该说什么才好。
<br/>　　“再则说，你确认你真的能杀掉我？”程大雷轻轻敲着剑柄。
<br/>　　魏牧冲道：“这座茶楼周围有一千精兵，整个汝南城有我五千城防兵，程当家便有插翅之能，也无法活着离开汝南城。”
<br/>　　“喔，你确定么？”程大雷问：“不如试一试，看我手中剑能砍几个脑袋，如汝南城又留不留得下我程大雷。”
<br/>　　魏牧冲攥着拳头，青筋暴起，杀子之痛自然要记在程大雷身上，他很想一声令下，在此地将程大雷铲除。但正如程大雷所说，自己真有把握杀掉他么？更况且，杀掉他之后，凉州必乱。到时候精锐的蛤蟆军杀到京州，陛下刚刚收拾妥当的长安，怕是会再次被攻破。
<br/>　　心中天人交战，难做决定。
<br/>　　程大雷挥挥手，道：“好了，赶快准备饭食，今天本当家要在汝南城歇息一夜，你要好生招待，若是招待不周，小心匹夫剑专斩不臣。”
<br/>　　魏牧冲深吸一口气，微微将头垂下：“请。”
<br/>　　程大雷看不见他此时脸上表情，不过据他想来，当是牙都快咬碎了。让一个朝廷命官对一个山贼卑躬屈膝，他心里自然不会好受。
<br/>　　但程大雷也不关心他的心路历程，只要他不冲自己下手就好。
<br/>　　程大雷大摇大摆的走出茶楼，那李槐倒在原地，魏牧冲一挥手：“拿了。”
<br/>　　李槐没有反抗，望着崔白玉的背影，纵然崔白玉连头都没有回，但他脸上却露出欣慰的笑容。
<br/>　　程大雷的住处被安排在汝南城的城主府，这府邸也算很有气派，但却没几个仆从下人，如此看过去，竟然有几分荒凉。
<br/>　　如此说来，魏牧冲的确是个难得的清官。但这确实也算不什么，魏牧冲自己守得住清贫，他的亲戚子弟未必受得了。汝南城如今被搞得乌烟瘴气，虽然手刃亲子有些残忍，但那魏明魏公子却未必不是死得其所。
<br/>　　人都退了，刘发财也收起一路上的漫不经心，认真道：“大当家，那姓魏的真会放我们平安离开么？”
<br/>　　程大雷摇了摇头，手指轻轻叩着桌面道：“是该想个脱身之策呐。”
<br/>　　魏牧冲没有动手，只是因为担心杀不了而已。不仅仅他一人这么想的，程大雷仇人是多的，许多人都巴不得程大雷赶紧去死。
<br/>　　既然已经暴露身份，可以相信，接下来的路程定然是杀机重重。
<br/>　　程大雷一死，京州必定是十年大乱。这种事，魏牧冲不是没有想到，但未必有多在乎。能杀了程大雷，定然可以青史留名，说来说去，他们心里未必想的是为国为民，还是想得自己多些。
<br/>　　事实与程大雷所料不差，当他进入城主府后，程大雷走出凉州的消息便已经传开。
<br/>　　大家都知道当今天子邀请程大雷前往藩王大会，但他能不能来，大家心里还是不知道。却是没想到，他当真是吞天的胆子，竟然敢走出凉州。
