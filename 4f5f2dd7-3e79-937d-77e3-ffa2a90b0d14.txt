　　洛冥将密报递给牧司令，眼神呆滞：“狼牙研究部研发出了解毒剂。”
<br/>　　牧司令一开始还没听明白，可当他看了那密报一眼之后，就整个人如遭雷击。
<br/>　　这是鲁初雪第一时间派人送来的简报，其内容并不全面，只有寥寥数语，最重要的内容就是提出各大基地供应特殊晶体的要求。
<br/>　　长久的沉寂在屋内延续，也不知过了多久，牧司令叹气闭上眼睛：“晚了三年。”
<br/>　　如果解毒剂能在末世爆发之初便出现，也就不会有这么多人的牺牲，也不会发生一系列被尸潮屠城的惨剧。
<br/>　　“不对。”洛冥却敏锐的察觉到一个要点：“解毒剂的原材料是特殊晶体，不然狼牙研究部让我们提供这些做什么？”
<br/>　　鲁初雪呈上的消息和要求列在一起，也显而易见。
<br/>　　牧司令豁然睁眼，道：“特殊晶体来自超级丧尸，而超级丧尸却是近期才大量出现的，并且五十万丧尸中才有一头，那么这解毒剂的数量……”
<br/>　　洛冥差点狂喜的心情一下子冷静下来，道：“只有少数人能服用。”
<br/>　　牧司令点头，紧接着皱眉道：“是狼牙研究部，不是楚涵？”
<br/>　　洛冥立即明白了过来，道：“楚涵估计还在南署基地的大战中，并不知道这一情况。”
<br/>　　“那就先压一压。”牧司令冷静道：“等南署那边的战事结束，也并不影响这解毒剂的调配吧？”
<br/>　　——————
<br/>　　在两名老者拿到这一密报的同时，在军营中的段江伟，也同样拿到了一份。
<br/>　　与两名老者第一时间的复杂心态不同，段江伟首先的情绪是震惊，要知道解毒剂早在末世二年末的时候就被各大基地的研究所放弃，因为根本毫无头绪。
<br/>　　包括楚涵本人也曾说过，丧尸病毒的感染是不可逆转的。
<br/>　　前后的矛盾让段江伟有些想不通，于是他做出了与两名老者一样的决定，那就是将这则密报压了下去。
<br/>　　一切按部就班，南署联盟军继续前行，这一则惊天的消息并没有因此改变他们的节奏。
<br/>　　只是当段江伟和两名老者刚做出这一决定之后没多久，又一则来自段氏基地的紧急密报就再次传来！
<br/>　　这一回，是鲁初雪在询问江左诸多具体细节之后，整理的一份有关解毒剂的详报，甚至包括提出了循环论和解毒剂的发明者，袁熙叶本人身份。
<br/>　　密报很长，也惊世骇俗。
<br/>　　于是在这则密报刚刚送到段江伟和两名老者手中后的没多久，便有两架直升机腾空，一架从上京基地出发，飞往段氏基地，一架从南署联盟军的军营出发，同样飞往段氏基地。
<br/>　　无论是两名老者还是段江伟，在看到解毒剂的详情之后，都统统坐不住了！
<br/>　　上京需要有人留守管理，于是只有洛冥一人独身前往，段江伟此时此刻也顾不得其他，一股脑的将他手中的工作推给其他将领，而后火速往段氏基地赶。
<br/>　　与第一份简报的内容不同，这则随后送来的详细密报，有太多他们曾经思考过，但却没能想通的东西，更有足以颠覆很多人认知和三观的可怕理论。
<br/>　　最为令人震撼的事，循环论与太多的情况吻合，尤其是丧尸进化的过程……
<br/>　　而当他们知道狼牙研究部的院长，那个解毒剂和循环论的主人名叫袁熙叶，为文明时代门萨内排名第十一位的可怕存在，还是楚涵的母亲之时。
<br/>　　将信将疑便一下子变成了强烈的好奇，于是这两人便第一时间来到了段氏基地，也将基地内毫无准备的鲁初雪吓了一跳。
<br/>　　“洛老！上将！”鲁初雪敬着礼，站在停机坪上望着两人几乎同时到达。
<br/>　　洛冥快步走到鲁初雪面前，张口就问：“袁熙叶呢？”
<br/>　　段江伟在见到洛冥之后也不意外，紧跟其后，眼神中不经意流露着急切。
<br/>　　他们很想见见那个叫袁熙叶的人，更想亲自与之交流有关循环论的事，至于解毒剂的真假，当循环论一出，便已经没有怀疑的必要。
<br/>　　鲁初雪显然没料到洛冥一上来就问袁熙叶，呐呐回答：“她不在这里。”
<br/>　　“走了？什么时候走的？回狼牙了？”洛冥一连三个问题出口，甚至已经准备好了当即再出发去狼牙。
<br/>　　鲁初雪连忙摇头：“她没有和研究部的其他人一起过来，两个月前狼牙研究部搬过来的时候，她就不在。”
<br/>　　听到这番话，洛冥和段江伟都一愣，激动的心情渐渐冷静下来。
<br/>　　洛冥眯起了眼睛：“解毒剂是刚刚研究出来的？”
<br/>　　“是。”鲁初雪着实回答：“今天上午才送到研究部成员江左的手中。”
<br/>　　“那些有关循环论的内容，也是江左说的？”段江伟的问题也问出口。
<br/>　　鲁初雪点头：“我并没有见过袁熙叶送来的资源，一切都是江左口述。”
<br/>　　段江伟和洛冥双双对视了一眼，都看到了对方眼中的谨慎，循环论骇人听闻是真，令人信服也是真，但这其中是否有太多的不巧？
<br/>　　两人一个为常年最高司令的参谋官，一个是被楚涵定论过多疑的年轻上将，经由多次转手的消息，反倒让两人一下子从根本上怀疑了起来。
<br/>　　“楚涵知道这件事么？”段江伟问出了最关键的问题。
<br/>　　鲁初雪摇头：“不知道，狼牙的同意人是何峰参谋。”
<br/>　　洛冥眉头一挑，再次与段江伟对视一眼，道：“又不是本人。”
<br/>　　段江伟托腮沉思：“这事，要不先问了楚涵再说？”
<br/>　　“同意。”洛冥点头道：“狼牙基地离的太远，过去亲自找到袁熙叶本人问清楚的时间，不如先去南署基地找楚涵，顺便看下这会儿的战况如何。”
<br/>　　鲁初雪连忙道：“南署基地的大战一切顺利，上午我派人过去询问解毒剂的时候，回来的人说一切顺利，已经快结束了。”
<br/>　　“那好，我们走。”洛冥是个说一不二的性子，立即转身走向直升机。
<br/>　　段江伟寸步不离的紧跟其后，两人就这么火急缭绕的上了直升机，直奔南署！
