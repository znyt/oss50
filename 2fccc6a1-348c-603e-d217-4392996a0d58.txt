　　时光长河中，一个在争夺命运之人渐渐模糊了意识。最后时刻，他仿佛听到了有人在呼唤自己。于是，他那沉睡了不知多久的眼睛，一下子睁了开来。
<br/>　　他的目光如电，像是要洞穿万古长河，在他的眼中，有混沌初开，阴阳两分的恐怖景象。
<br/>　　“不对，我不是已经死了吗，我这是在哪？怎么可能还活着……”
<br/>　　“不对，我似乎是忘了什么？我到底遗忘了什么？”
<br/>　　“……”
<br/>　　“我回到了过去，我又回来了，我得见过去，我经历了未来，我，我就是元始！其他种种，皆是虚妄！也尽皆是真实！不过，那是原始，却也不是元始！我在岁月中沉沦，我在轮回中争渡，现在，本座是该醒来了！”
<br/>　　“过去的，就让他过去吧，未来的，充满不确定，一切都是虚妄！过去是永恒不变的，未来是不可预测的，我要珍惜的，不是过去，也不是未来，而是现在。”
<br/>　　“过去种种，尽皆烟消云散，未来种种，吾又何须多管！把握现在，方见真我！”
<br/>　　“过去的我，可称之为‘逝我’，未来的我，可称之为‘超我’，现如今的我，可称之为‘道我’！斩去‘逝我’、‘道我’、‘超我’，得见‘真我’！”
<br/>　　“斩我明道诀，逝我、道我、超我，斩！斩！斩！”
<br/>　　时光长河之中，元始眼中猛然间迸发出璀璨的光芒，他仰天长啸，下一刻，他以无上神通，凝聚时光之刃。他挥动手中的时光之刃，直接斩向自己！
<br/>　　……
<br/>　　轰！
<br/>　　昆仑山，玉虚宫，赤芒冲天，有恐怖的气势冲天而起，与天地齐高，那是元始闭关之所。这个动静很大，整个昆仑山，乃至东方大地都不能平静，所有人都被惊动了，纷纷看向玉虚宫方向。
<br/>　　老子、通天更是第一时间来到玉虚宫前，想要看看到底发生了何事？
<br/>　　洪荒之中，两族正在大战，此刻谁也顾不上谁，他们早已杀红了眼。但其他关注此战的准圣强者，却是纷纷将目光投入到昆仑山方向，比起交战了一千多年的两族大战，他们更关心昆仑山发生的变故。
<br/>　　“嘶！那是……昆仑山方向？在那里，只有三清了。是谁成圣了吗？”有人纷纷被滔天异象惊住了，猜测是否是有人成圣？
<br/>　　“那是，玉清元始的气息，他又想要干什么？”
<br/>　　“混蛋，之前才弄出那么大动静，如今又弄出个更大的动静，他到底想要干什么？想要成圣就赶快，非得搞出些大动静来？”有人看不惯眼，嫉妒道。
<br/>　　“元始，要成圣了么？”有人眼皮直跳，瞳孔微缩，看向昆仑方向，猜测道。
<br/>　　“……”
<br/>　　各路大神，神念交织，都在纷纷议论，昆仑到底发生了什么事情。同样的，两族大战的强者，他们同样发现了元始弄出的动静，一时间，准圣高手间出现了短暂的静止，他们也想要看看，这是怎么回事？
<br/>　　轰！
<br/>　　玉虚宫中，又是一股恐怖的威压突然扫了过来，扫遍洪荒大地，这时，大战之中的众多巫妖两族的成员，纷纷被这股威压压得扑倒在地，事实上，除了太乙金仙以上者能够勉强抵挡，其他人，就连金仙强者，都在这股恐怖的威压下直接匍匐再来地上。
<br/>　　不过，只是一瞬间，这股威压便又消失不见了。就好像从来没有出现过一样，无数人愕然的从地上爬了起来，感觉之前的一瞬像是做了一个梦一样。
<br/>　　轰！
<br/>　　又是一股比之前更加强烈的威压亦昆仑山为中心，向着洪荒大地扑面而来。这一次，好多人都已经放松了，没有准备好，却正好被这股恐怖的威压压倒在地，就连许多太乙金仙，都被压倒。
<br/>　　与之前一样，这股威压也是来得快，去得也快，仿佛从来没有出现一样。
<br/>　　看着洪荒大地上跪伏在地的无数妖族、巫族成员，此刻，不管是妖帝帝俊、东皇太一、还是祖巫帝江等脸色全部变得阴沉难看。我族儿郎，竟然向你元始跪拜，这算是什么事？你这是在打脸吗？于是，他们看向昆仑山，眼光闪烁，不知在想些什么。
<br/>　　“靠！这元始到底在干什么，一而再再而三地放出强大威压，很好玩吗？”
<br/>　　这一刻，很多人都快骂娘了，你元始到底是怎么回事？想要成圣就赶紧，搞这些事来干什么。好玩吗？
<br/>　　“这元始在搞些什么名堂？”
<br/>　　就连一些强大的准圣也搞不清元始的目的。要说成圣了吗，这显然不是，就连成圣的异象也没有。要说不是成圣吗，他那股恐怖的威压中，又蕴含着一股强烈的圣道气息。这算怎么回事？
<br/>　　当然，也有一些人脸色变得更加凝重了，因为他们从元始的这股气息之中，看到的圣道的威压，显然，元始就算还没有成圣，但也不远了。
<br/>　　不管众人如何，却说玉虚宫中，元始收起身上的气势，渐渐恢复平常。他缓缓睁开双眼，眼中闪过一丝苍茫，混沌演化，天地寂灭，各种恐怖异象，尽在其中。良久，他才收回这些异象，回到最初。
<br/>　　在他的身旁，此时有另外两道身影，两道身影都身穿白衣，与元始有七八分相似。若是细看，就可以看出，这两道身影其中一道好似不属于现在，而是属于过去。还有一道眼神淡漠，无情若冰，就好像是一块冷冰，不见丝毫的感情波动。在他的身上，有一股强大的道韵流转，他仿佛就是道的化身。
<br/>　　“哎，还有有些不完善啊，只能斩出‘逝我’与‘道我’，‘超我’还是差了那么一点。”元始看着眼前的两道身影，眼中流出一丝无奈的神情，低声呢喃道。
<br/>　　“不过，也不远了，只要我再多花费一些时间，就一定可以斩出。那时，我便可以得见真我，证道混元。”说着，元始眼中闪现出强烈的光彩，如群星般璀璨，比太阳还炙热夺目。那是一种强烈的自信的表现。
<br/>　　“这两个‘我’要怎么办呢？”元始看着眼前的两道身影，默默思索道，“‘逝我’是过去的我，过去的，就让他过去了，眼光要往前看，所以，这尊‘逝我’只需等待一个机会，便让他重生吧。”
<br/>　　“‘道我’倒是好办，道我就像是一方世界的天道，他只会悟道，修道，掌道，不会干别的。而且，悟道效率还不是一般的快啊。那么……就让他到鸿蒙元始印之中，一直参悟其中的大道法则吧。”
<br/>　　说着，元始将两个“我”全部收入元始印之中，让他们为自己参悟大道法则。
<br/>　　之后，他感受了一下自己的实力，发现自己的实力已经突破了准圣，但是，还没有成为圣人或者混元。这就奇怪了，不是准圣，也不是圣人，那又是什么？
<br/>　　“唔！看来，准圣之后，还有路可走。吾此刻的境界，已经不算准圣了，但也还没有到圣人境界，那么，吾便就称它为‘亚圣’，或者‘伪圣’吧。”说着，他眼前一亮。
<br/>　　“吾如今已斩去‘逝我’与‘道我’，这算是亚圣中期巅峰了吧。”
