　　第409章
<br/>　　“大当家，人应该还没有走远，现在撒出人手，当还能寻回来。”刘悲询问程大雷的意思。
<br/>　　程大雷想了想，道：“罢了，既然是打晕守卫走的，看来去意已绝，那就随他去吧。”
<br/>　　福德勒是个沉默寡言的人，这是山寨很多人对他共同的评价。但此时程大雷不仅寻思，沉默何尝不是一种自我保护。他将自己封闭起来，没人知道他午夜梦回时脑海里想些什么。同时也没人知道他从何而来，要到何处去……这一点，就算李行哉也一无所知。
<br/>　　如今他要走，那就让他走吧。临走前肯留下四个字，显然是还记得蛤蟆寨这份情的。当然，也只是四个字的情分。
<br/>　　说到福德勒，便不由得想起李行哉。如今距离定下拯救李行哉的计划已经有些日子了。
<br/>　　“阿喜那边进行的如何，能不能把人救出来？”程大雷问。
<br/>　　“正要向大当家汇报，事情进行的很顺利，一切照计划进行。地道已掘到凉州城，今夜应该就能把人救出来。”
<br/>　　听到这消息，程大雷松了口气。人能救出来就好，这件事也算顺利解决了，之后李行哉是要留在蛤蟆城，或者是继续前往长安，程大雷都没意见。
<br/>　　却也不知道李行哉现在的情况如何。
<br/>　　程大雷心里琢磨了一句，踱步和刘悲一起走入城主府，仔细询问今夜的行动，看阿喜是如何安排的。
<br/>　　……
<br/>　　凉州城。
<br/>　　某种情况上来讲，李行哉现在的生活和刚开始相比，要好上许多。他如今的活动区域，已扩大到整座城主府，甚至偶尔也可以在重兵的保护下，在凉州城内转上一转。
<br/>　　宋伯康需要李行哉出现在城内军民面前，这某种程度上可以提升凉州城的凝聚力。李行哉穿着华服，坐着大撵在城内转一转，让城里人看见，帝国六皇子也出现在凉州城了，可见朝廷对此地的重视。
<br/>　　的确，李家失去民心良久，但帝国皇族对普通人来说还是不容易见到的生物。就像在程大雷前生，哪怕是在网上被骂出翔的明星，生活里真正见到了，也是忍不住会拍张照片的。
<br/>　　对于宋伯康的需求，李行哉很是配合，这换来一定程度上的自由。宋伯康说得不错，曾经的李行哉还是太过幼稚，这对于帝国皇子，尤其是对于一名拥有帝王之志的帝国皇子来说，任何幼稚行为都是致命的。
<br/>　　帝王无情，这说得不仅仅是对周围人无情，更是从内心深处扼杀自己的感情。
<br/>　　李行哉正强迫自己，学着习惯与自己讨厌的事物接触。这是一方面原因，还有更重要的原因：
<br/>　　李行哉一直觉得，福德勒仍被关在凉州城。
<br/>　　蛤蟆城与凉州城交换人质的事情，是瞒着李行哉进行，这件事没有任何人与李行哉说起。在李行哉的认知中，为了保护福德勒，自己必须配合宋伯康。
<br/>　　今日大撵走过长街，街道两旁挤满围观的人群。李行哉坐在高高的大撵上，身着黑袍，面带温柔微笑。不得不承认，李行哉生得很英俊。明帝有好基因，不管是李行哉，还是李乐天，李婉儿都有姣好模样，出类拔萃的气质。
<br/>　　那深邃的眼神，看得道路两侧一些小姑娘头晕眼花。一个王子，一个英俊的王子，单单这个字眼，已经可以衍生太多故事。就像无数说书人口中常说起的，一个英俊的王子爱上了一个平民姑娘，姑娘又不好看，脾气又坏，可王子偏偏爱得要死。连王位都不要了，整个天下的反对也顾不得，然后谱写成一曲可歌可颂的爱情长诗。
<br/>　　那个姑娘不怀春，少女心情总是诗，这些姑娘垂着头，想看也不敢看，不敢看却又悄悄抬起头张望。望来望去，望得自己满脸通红，眸子含水。
<br/>　　正在此时，咚地一声，一块砖头从街道两旁丢了出来，正砸在李行哉头顶。
<br/>　　宋伯康吓了一跳，忙道：“什么人，保护皇子！”
<br/>　　肇事者很快被抓了起来，送到大撵前，宋伯康勃然大怒，道：“拉下去，丢进大牢，问出他们的同伙。”
<br/>　　“慢！”李行哉肿起来的额头，伸出一只手拦住宋伯康的命令，看着向自己丢砖头的人。
<br/>　　对方是个二十出头的后生，虽然被卫兵押着，但却横眉立目，以愤怒的眼神盯着自己。
<br/>　　“你叫什么名字？”李行哉。
<br/>　　年轻后生恶狠狠的盯着他，突然破口大骂道：“不怕告诉你，老子姓王名德。”
<br/>　　“今天为什么这么做？”
<br/>　　“哈！”王德仰天大笑：“帝国大好河山，都是你们这些人葬送的，我们连肚子都吃不饱，你们却高高在上，作威作福，我恨不得把你们这些人全部杀死。”
<br/>　　李行哉怔了怔，表情有些失神。宋伯康道：“殿下，这等刁民已经丧心病狂，还是赶紧把他拉下去吧。”
<br/>　　“等一等。”李行哉突然从大撵上跳下来，看着王德道：“冲撞车撵，便是死罪，你……不怕死么？”
<br/>　　“死，你以为我们还活的下去么？”
<br/>　　李行哉目光环视身周，突然发现周围平民眼中都燃烧着什么，似乎叫做怒火。而那些本来含羞带怯的小姑娘，此刻眼中原先的水也凝成了冰。
<br/>　　原来天下百姓就是这么想我们李家的么。
<br/>　　李行哉心里叹了口气。
<br/>　　“殿下……”
<br/>　　“宋大人。”李行哉开口打断他，道：“能不能将这人放了？”
<br/>　　“殿下。”宋伯康近前一步：“威不可失，法不容情，这人放不得啊。”
<br/>　　李行哉点点头，却也没有再坚持什么。他重新上了大撵，将帘子放下，今天本也没有继续逛下去的心情，同样，李行哉也没有在百姓面前露面的勇气。
<br/>　　围观人激动的目光，并不是来源于对李家的尊重，完全是出于对新鲜生物的好奇，不用怀疑，就算拉头五条腿的狗，他们也会旁观的。激动是假的，心里的怒火是真的。
<br/>　　现在李行哉才总算明白，为何林少羽可以短暂时间拉起一支队伍。
<br/>　　天下看李家不顺眼的人，实在太多了。
<br/>(本章完)
