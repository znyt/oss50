        “大哥，我想跟你请假，这段时间，我不想去公司上班。”一想到傅昼景，童筱颖走到了傅夜沉的身旁，忽然间不知道哪儿来的勇气，就这么无所顾忌的脱口而出。
<br/>
<br/>        “啪”地一声，一根木桩被傅夜沉一掌给劈断了。
<br/>
<br/>        童筱颖被轻微地吓了一跳。
<br/>
<br/>        傅夜沉自然知道她请假是为了做什么，心里莫名其妙地隐隐作痛。
<br/>
<br/>        过了半晌，他才漫不经心地回答：“请假这事，你应该去跟你部长说。”
<br/>
<br/>        童筱颖上前一步，急忙解释道：“我说过了，但是部长不批！”
<br/>
<br/>        “公司有公司的规章制度，就算你是我的亲属，你也得遵守。既然，你部长不批，那你就别请假了。”傅夜沉淡定自若地说道。
<br/>
<br/>        童筱颖却郁郁寡欢地应了声：“噢！”
<br/>
<br/>        “还有事吗？”傅夜沉声音冷淡地问。
<br/>
<br/>        童筱颖摇了摇头，然后低着头，准备悻悻地离开。
<br/>
<br/>        “你的脸是怎么回事？”傅夜沉又问道。
<br/>
<br/>        他记得，他昨天带她回来时，她的脸蛋还白白净净的。
<br/>
<br/>        刚刚看她时，发现她的左脸颊上有淤青的痕迹。
<br/>
<br/>        提起脸，童筱颖下意识地摸了摸自己脸颊，才想起昨晚苏漫雪打她的那一巴掌，她还未上药。
<br/>
<br/>        如果跟大哥说，是苏漫雪误会了他两，才打的她这一巴掌，大哥一定不会信她。
<br/>
<br/>        童筱颖纠结了一会儿，才木讷地开口回答：“昨晚做噩梦了，自己打了自己一巴掌，把自己打醒了！”
<br/>
<br/>        希望她撒的这个谎，能瞒过大哥的眼睛。
<br/>
<br/>        傅夜沉不再多话，继续搏击着面前的木桩。
<br/>
<br/>        童筱颖转过身去，渐行渐远。
<br/>
<br/>        苏漫雪还在睡梦中的时候，就被刘婶给叫了起来。
<br/>
<br/>        自从那晚被傅夜沉关进了小隔间里睡了一晚，苏漫雪这回学乖了，再也不去招惹傅夜沉，每天晚上老老实实地待客房里睡。
<br/>
<br/>        这不，她还未睡醒，就被刘婶给吵醒，火气一上来，就冲着刘婶大发雷霆：“你他妈的有病啊！”
<br/>
<br/>        “啪”地一声，刘婶扬起手来，重重地一巴掌，掴在苏漫雪脸上，把苏漫雪打得一脸懵。
<br/>
<br/>        苏漫雪瞪大了眼睛，回过神来后，忽然面目狰狞地朝刘婶扑了过去。
<br/>
<br/>        “啊啊啊！你这个疯婆子，敢打我！吃了雄心豹子胆，是不是？”苏漫雪一屁股坐在刘婶肚子上，抬起手来，挥起了拳头。
<br/>
<br/>        就在她拳头即将落下之际，刘婶紧张地大喊了起来：“是大少爷让我来打你的！还让我来警告你，如果你再敢动二少奶奶一根汗毛，大少爷会让你吃不了兜着走！”
<br/>
<br/>        “什么？”苏漫雪怔愕了，举起的拳头也停在了半空中。
<br/>
<br/>        那个该死的童筱颖！
<br/>
<br/>        贱人！
<br/>
<br/>        居然敢算计她！
<br/>
<br/>        刘婶趁着苏漫雪走神之际，连忙推开了苏漫雪，从地上爬了起来。
<br/>
<br/>        她站起身后，居高临下地瞪着苏漫雪敦敦教诲道：“大少奶奶，我好心奉劝你几句。你最好懂得自知之明！你这大少奶奶的位置是怎么得来了，你心里应该比谁都清楚！你要想把自己这个假的大少奶奶的身份，改变成真的大少奶奶的身份。你最好把你的品行也跟着改一改！”
<br/>
<br/>        “呵？”苏漫雪坐在地上冷笑，抬眸瞥了刘婶一眼，反问，“我品行就这样，犯得着改吗？要你这做下人的来多嘴！”
<br/>
<br/>        “大少奶奶不听劝就算了，当我什么话都没说过便是！”刘婶不想再理会苏漫雪，撂下话后就转身离去。
<br/>
<br/>        苏漫雪一脸仇视地看着刘婶的背影，十指的指甲深深地掐入了地毯上。
<br/>
<br/>        她再气愤再气恼，有人好心提醒她了，她自然不是傻子，完全听不进去。
<br/>
<br/>        重新理了理自己的情绪后，苏漫雪走进了卫生间里，站在镜子面前看着镜子里这张“童筱颖”的脸，嘴角意味不明地扬了扬。
<br/>
<br/>        童筱颖在花园里散完步回来，准备帮刘婶一起做早餐的时候，苏漫雪正巧从中央卧梯上优雅地走下来。
<br/>
<br/>        她就像一个女主人一样，披散着海藻般的长发，打着哈欠，慵懒地走到童筱颖面前询问：“早餐准备好了吗？”
<br/>
<br/>        童筱颖瞥了苏漫雪一眼，默不吭声，转身准备往厨房里去的时候，苏漫雪突然伸手抓住了她的手腕。
<br/>
<br/>        “筱颍，昨晚的事，对不起。我真的是爱夜沉爱过头了，才会心急如焚，害怕失去他。你能理解我的心情吗？对不起！我不该误会你的！我错了！你原谅我，好不好？”苏漫雪一改常态，眨巴着眼睛，可怜兮兮地看着童筱颖。
