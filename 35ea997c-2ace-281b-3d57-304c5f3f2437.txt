　　跟随着车载导航的提示，罗昊驾驶着这辆七人座MPV驶出机场停车场，前往“代号小镇”。
<br/>　　“代号小镇”在当地话中，被称为“美丽的明珠”，在当地也算是一个旅游景点，旅行团可以带着来这个小镇观光，但是不能在小镇上过夜，下午五点，就会有人对还滞留在小镇上的旅行团进行驱逐。
<br/>　　小镇外有一大片空地，空地上用油漆画了对应的格子和数字，给外来车辆当做停车场来使用。
<br/>　　罗昊驾驶着汽车抵达停车场的时候，是当地时间下午四点，一些前来小镇参观游玩的旅行团正在从小镇中出来，走上停在停车场上的客车，准备离开。
<br/>　　“夜幕降临了，这个小镇将迎来它最真实的一面。”子午看着外面开始逐渐昏暗的天空，说道：“真是一个令人怀念的地方，而明天晚上，你们会见识到这个美丽的小镇最疯狂的一夜。记住，所有武器全都留在车上，不要带入小镇。”
<br/>　　随着旅行团的客车从停车场离开，宽敞的停车场变得空旷起来。罗昊、子午等人没有急着下车，进入小镇，而是把特种战术背囊打开，拿出里面的武器进行调试。
<br/>　　六个人的武器装备都一样，除了一把安装了激光指示器、红点瞄准镜，消音器的MPX冲锋枪外，还有一把MK23手枪，两枚闪光弹和两颗美制M57手雷，弹匣袋中塞满了压满子弹的弹匣，MPX和MK23的都有，武器都是校准好的，装入弹匣，把子弹上膛就能投入使用。
<br/>　　罗昊拿起一个两联装弹匣，塞进MPX冲锋枪中，把MPX冲锋枪塞入背囊中；拔出MK23手枪，退出弹匣，看了眼弹匣中金灿灿的子弹，把弹匣装回手枪，把手枪塞进快拔枪套中，再把快拔枪套塞进背囊中，拉好背囊拉链，把背囊放在汽车方向盘下面。
<br/>　　对着车厢中其他几个人点头示意了一下，推开车门走了下来。六个人在车厢中检查武器装备的时候，停车场中又有着好几辆车驶了进来，其中甚至还有宾利、保时捷、玛莎拉蒂这样的豪车。不过因为MPV四周车窗上贴着深色窗膜，除非是特别靠近，不然根本不可能看到车里的人在干什么，更何况，也不会有人注意停车上这辆黑色的克莱斯勒，这辆黑色的克莱斯勒根本无法吸引他们的注意力。
<br/>　　子午拎着那个装满了欧元的旅行袋，走在队伍最前面。小镇门口，六个人被拦了下来，一个金发碧眼的年轻男子拿着金属探测器对着六个人进行检查，并且要求子午和李然把他们手中装着钞票和笔记本电脑的旅行袋和背包打开进行检查，并且放入安检机进行扫描和检查。
<br/>　　确认六个人身上没有携带任何武器，旅行袋和背包中也没有藏有危险物品后，年轻男子把旅行袋和背包还给子午和李然，扫了眼六个人，问道：“自由者？”
<br/>　　子午点头。
<br/>　　年轻男子看向六个人的眼色立刻就变得轻蔑起来，拿了六张芯片卡在一台机器上进行扫描后，分别交给六个人，吩咐道：“你们每个人可以在这里滞留七十二个小时，必须在滞留时间超过之前，离开这里，不然你们会遭到驱逐，并且永远失去进入这里的权利。使用这张卡片，你们可以入住小镇上任意一家旅馆，在任意一家餐厅用餐，离开的时候你们需要把卡片归还，我们会根据你们在小镇上的消费结算。”
<br/>　　子午点了点头，表示对小镇规矩的了解，然后收起了卡片，拎着装有现金的旅行袋往小镇内走去。
<br/>　　六个人还没有走远，就又有几个人朝着小镇走来。走在最前面的，是一个穿着做工考究米色西装，年纪大概在五十多岁，留着浓密络腮胡的白人，在他左眼位置，挂着一片金丝边框的单片镜，连在单片镜上的细金链子垂在他左前胸塞着一块白色手帕的口袋中。
<br/>　　在这个白人左手边，站着一个体型壮硕，身高在一百八十五公分左右，棕色瞳孔棕色头发，留着美式“高且硬”，上身穿着一件狼棕色紧身短袖，下身穿着一条MC迷彩长裤搭配着一双丛林作战靴的白人男子。在这个白人男子左耳朵上，打着一圈银色的小耳环。
<br/>　　看到这个白人男子第一眼的时候，罗昊脑海中就冒出了两个字，军人！罗昊可以肯定，这个站在西装男人身边的白人男子，还有他们身后那几个留着美式“高且硬”，胡子刮得很干净，包括了黑白黄三种肤色的男人全都是男人。
<br/>　　罗昊打量着左耳上打着一圈银色小耳环的白人男子的时候，原本眼睛左右飘忽不定，警惕的观察着周围情况的白人男子也是把视线落在了罗昊身上，两个人的视线在这被夕阳镀成金黄色的空气中，进行了一次无形的碰撞。
<br/>　　络腮胡西装男人把一张名片，还有一封类似邀请函一样的东西交给之前接待罗昊、子午他们进入村庄的那个年轻男人，并且跟他进行着交谈，通过他们的交谈，罗昊知道这个络腮胡西装男人叫洛菲克，是代表洛克希德公司来参加明天晚上举行的“源代码拍卖会”。
<br/>　　年轻男人把几张黑色的卡片递给络腮胡西装男洛菲克，还有他身边那几个留着美式“高且硬”的职业军人，态度上比起刚才对自己这些人的态度要好的多。
<br/>　　洛菲克左手插在裤兜中朝小镇中走去，在从罗昊、子午等人身边经过的时候，他刻意停下了脚步，右手手指夹着那张黑色卡片在罗昊、子午这些人面前晃了一下，眼睛中闪过明显的不屑之色，然后继续往小镇上的一家旅馆走去。
<br/>　　等到洛菲克这些人走远后，罗昊才向子午请教自己的不解，“子午教官，为什么他们的卡片颜色跟我们不一样。”
<br/>　　罗昊拿出自己口袋中的卡片，他的那张卡片红色的，跟洛菲克那张卡片有着明显的区别。
