　　第673章
<br/>　　引一支奇兵夜袭，突破城防，近而占据凉州城。
<br/>　　程大雷手指轻轻叩着桌面，脑海中在不停思索。这件事的确需要冒一些风险，但也有很大的胜算。
<br/>　　再次睁开眼睛，程大雷问：“凉州城有我们多少人？”
<br/>　　吴用立刻明白程大雷在问什么，有蛤蟆城的探子潜在凉州城，收集各方面情报。不过目前宋伯康查得紧，许多人都撤了出来。
<br/>　　“大概有三十几人，目前想要再派人潜进去怕已经很难做到。”
<br/>　　程大雷点点头，如果城中有自己的人马，做事会顺利许多。
<br/>　　“大当家，要不要等一等，我们有充足准备后，再做打算。”
<br/>　　“机会稍纵即逝，等我们准备好了，也许对方也同样准备好了。”程大雷长舒一口气：“今夜便是今夜，我们就与老天赌一赌命。”
<br/>　　自己所献的计策能被程大雷认可，这也是吴用的荣誉。他挺起胸膛，自己在程大雷手下终于有了些价值。
<br/>　　然而，这只是一个初步的想法，具体施行还需要确定一些细节。
<br/>　　这次商议，只局限在二人中间，程大雷敲定有龙字军、蛮字军、虎豹军中的精锐执行，从中选出三千人，组成蛤蟆城最精锐的队伍。
<br/>　　先由白元飞携带火药潜入凉州城，以烟花为号，炸塌城墙后，里应外合，打宋伯康一个措手不及。
<br/>　　二人商议一番，忽然想起一件事，忙打开系统界面，领取了今日份的诸神祝福。
<br/>　　诸神的祝福是程大雷少数保留下来的几个技能之一，每三天激活一次，可以获得各种各样的属性加成，持续时间三天。
<br/>　　嘟，获得战神的祝福，有效时间三天。
<br/>　　随着一道系统提示，程大雷心头大振。忍不住开口道了一声：“当真是老天助我。”
<br/>　　吴用有些困惑，不明白程大雷怎么忽然亢奋起来。
<br/>　　“大当家，是发生什么事了么？”吴用道。
<br/>　　“没事，没事。”程大雷摆摆手，咬牙道：“今夜行动，本当家要亲自领军。”
<br/>　　程大雷当真很久没有亲自上过战场，蛤蟆城也没有发生过太大规模的战斗。而这一战，关乎蛤蟆城的生死存亡，一向平静的蛤蟆城进入战争状态。
<br/>　　怪兽苏醒，展现了极强的组织能力，只用了些许时间，便集结完毕。有龙字军的一千人，蛮字军的一千人，虎豹军的一千人。赵子龙、秦蛮、高飞虎、高飞豹、小白狼、史文恭等大将都将参与这次战斗。
<br/>　　当真是兵雄将猛，士气如虹，以少击多，没人有半分怯意，都期待着与凉州城放手一搏。
<br/>　　程大雷也穿好了盔甲，座下骑着照月乌角兽，身披血披风，背后扛着鬼面斧，腰间是匹夫剑。在他身后，是扛着一根大棒的银眸。
<br/>　　面前这三千人是蛤蟆城最精锐的力量，此战胜了，自己就可以制衡整个凉州。此战如果败了，自己怕就只有撒丫子逃命一个原则。
<br/>　　所以，这一战，自己当真输不起。
<br/>　　在天黑之后，程大雷挥挥手，下达了言简意赅的命令。
<br/>　　“出发。”
<br/>　　三千骑出城，奔向凉州城。
<br/>　　……
<br/>　　凉州城。
<br/>　　崔白玉打了个哈欠，继续翻阅着桌上的材料。到现在，她大概已对程大雷有了判断。此人不按常理出牌，似乎还蛮有情义，不过对世事却又有极强的洞察力。
<br/>　　倒是一个不可轻视的敌人，事实上，真正令崔白玉忌惮的，还是他背后的力量。
<br/>　　究竟是怎样的存在呢？
<br/>　　崔白玉放开心思去想，仍旧没什么线索，这令这件事情变得复杂起来。
<br/>　　“郡主。”丫鬟小鹿轻轻靠近：“宋伯康求见？”
<br/>　　崔白玉皱起眉头，宋伯康此刻跑过来做什么。事实上，她不是很喜欢宋伯康，她也并不喜欢任何人。虽然是天下一流谋士，但人情世故并不是她擅长的，原因是她不愿意去了解这样，因为太过浪费时间。
<br/>　　或许，对方是有重要的情报汇报也不是没有可能。
<br/>　　崔白玉戴上黑纱，道：“让他来进来吧。”
<br/>　　在自己房中，她自然没有戴黑纱的习惯。其实，崔白玉也知道自己生得美貌，但她并不以此为荣。因为每个见到她模样的人，都会忽略掉她的本事，而只关注她的相貌。
<br/>　　当宋伯康脚步匆匆走进院落里时，神情有些失望。大半夜过来，就是为了打对方一个措手不及，想要看看这白玉美人模样如何。他虽然不敢有非分的想法，但一窥芳容的想法还是有的。
<br/>　　“宋大人是有什么事么？”
<br/>　　“喔，也没有什么太要紧的事，只是过来看看郡主在这里住的如何，有什么进展没有？”
<br/>　　崔白玉立刻明白了，这宋伯康真的没什么重要的事，只是过来和自己闲聊。可她怎么有这个时间，表情立刻露出不满。
<br/>　　“如果宋大人没有要紧的事，就请先回吧，我还有事要做，就不奉陪了。”
<br/>　　崔白玉直接下了逐客令，一点面子没给崔白玉留。宋伯康心里有些不舒服，这小蹄子，当真以为天下是你崔家的么。自己好歹是一路诸侯王，可你却没给自己任何情面，还口口声声唤自己『宋大人』。若有一日我登了九五之尊，必让你这女人侍寝，看你到时候还如何冷若冰霜。女人么，在床上都是一般模样。
<br/>　　心中如此想，表面自然不敢流露出半分。他面带微笑，道：“那我就先告退了，郡主有什么事，尽管吩咐，宋某一定尽力让郡主满意。”
<br/>　　人在矮檐下，不得不低头，宋伯康虽然恼得厉害，但也不敢得罪这位白玉美人。
<br/>　　他离去后，崔白玉轻轻叹口气，神色有几分怜悯。
<br/>　　一身着戎装的男人靠近，毕恭毕敬道：“郡主可有吩咐属下做的事？”
<br/>　　崔白玉摇了摇头，道：“暂且留他几日吧。”
<br/>　　“是。”
<br/>　　这男人是崔白玉的亲卫队长，名叫哥舒离，算是崔白玉身边的死士。
<br/>　　崔白玉这次入凉州，目的有二，一是为了帮助宋伯康消灭程大雷。第二，就是看宋伯康是否规矩。
<br/>　　揣测宋伯康的心思，要比揣测程大雷的心思容易太多。
<br/>(本章完)
