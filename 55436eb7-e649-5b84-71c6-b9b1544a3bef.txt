　　毛小方道：“不用画像，只把名字放出去，试图传到他朋友的耳朵里，其余人又不知道他是谁，也不知道他长什么样，就算见面也认不出的。主要是少阳子只要不嫌弃名节受损就行。”
<br/>　　叶少阳耸肩说道：“我一个未来世界的人，有什么名节不名节的，只要能找到我那班朋友就行。”
<br/>　　道渊真人于是把这件事揽在自己身上，他虽然名义上闭关，并且被人嫌弃，不过怎么说也是张天师后人，在法术界是有身份派发法帖的。
<br/>　　“正好，我要是下山的话，就这么下山不行，我得去跟我爷爷说一声，顺便让他同意我派法帖，你们告诉我个地名，我到时候去找你们。”
<br/>　　两人答应下来，然后先告辞，去处理百鬼夜行的事。刚要动身下山，道渊真人又把叶少阳叫住，问他法帖上的有关他的罪名该怎么写。
<br/>　　叶少阳第一个想到的词就是“***掳掠”，不过转念一想，这个其实不归法术界管，直接报官就可以了，想半天也没什么好的罪名，让他随便安一个。
<br/>　　“尽量往大了编，这样才会引起震动，传播的也范围也广一点。”毛小方提醒道。
<br/>　　“我尽量吧。”道渊真人挠着头，一副若有所思的样子。
<br/>　　叶少阳二人下山之后，把之前拴在山脚下吃草的马牵上，然后直奔毛小方说的那个地方。
<br/>　　“有一百里左右，虽然路不太好，不过最多也就一两天准到。”
<br/>　　说到这里，毛小方想到了什么，一边骑马一边问道：“对了，少阳子，你们那个时代，人去外地主要靠什么，骑马还是坐汽车？”
<br/>　　“你也知道汽车？”叶少阳有点惊讶，自己这一路上来，可是一辆汽车都没见到过。
<br/>　　“在上海和北平见过的。不过跑的还没马快，火车我也见过，从天津到北京的。”毛小方自言自语，“就是太麻烦了，要修火车道，路线也是固定的，还是骑马方便。”
<br/>　　叶少阳笑道：“这个也不算泄露天机了，我可以告诉你，在我那个时代，全国各地都有火车道，想到哪个城市都方便，而且那个时候的火车，比现在这种要快得多，时速……一个时辰能跑几百里。”
<br/>　　毛小方听了，吓的差点从马上掉下来，“不可能有这么快吧，一个时辰几百里，那从北京到上海，难不成一台年就到了？”
<br/>　　“要不了一天，坐火车也就是大半天的工夫，如果坐飞机，也就两个小时吧。”
<br/>　　“飞机？”毛小方对这两个字感到很新鲜。
<br/>　　叶少阳简单讲了一下飞机的知识，毛小方听完之后，更加震惊。当叶少阳说到那个时代可以坐飞机出国的时候，毛小方问起了那个时代中国跟国外的关系，随后自己也觉得有些泄露天机，干脆不问了。
<br/>　　一路上，毛小方又缠着叶少阳说了很多百年之后的有关科技和生活方面的知识，听的心生向往，很想亲自过去体验一番。
<br/>　　两人策马奔腾，先回了一次平南县，叶少阳在客栈里找到翠云，告诉他自己要去办点事，让她继续在这里等着，翠云看他安全返回，很是开心，也放心让他去。
<br/>　　离开平安县，两人又骑马继续想西南方面走，一天之后，进了山，遇到无法骑马通过的地方，两人把马放了，步行过去。
<br/>　　进山之后，到处就是一片荒芜了，偶尔翻过一座山峰，会看到一些小村子，这些在大山里的村子，村民靠种地和打猎为生，很少外出，也不像十几年后那样农民工大批进城。至少在眼下这个阶段，这些生活在深山里的百姓反而是一种幸福，除了需要上贡附近的土匪之外，也免于遭受战火摧残，自给自足。
<br/>　　两人一路走到毛小方之前说的闹鬼的那条山谷，但没有立刻下去，而是爬上附近的一座山峰，朝山谷里看去。
<br/>　　山谷有一两千米长，十来米宽，东西走向，站在山顶上看上去，山谷中有灰白色的好像云雾一样的氤氲气体，在山谷中弥漫着。
<br/>　　“这是阴气。”叶少阳说道，阴气本来是黑的，但如果不够浓密的话，在白天阳光下就会显示出这种颜色。
<br/>　　引起弥漫，必有妖邪。
<br/>　　这是法术界的一句谚语，不过说的也只是一些封闭的空间，例如某些闹鬼的老宅之类的，本来空气也不流通，如果有鬼做了巢穴，便会聚集阴气，长久不散。普通人肉眼看不见阴气，但是进入这样的地方，会立刻感到浑身发冷，各种不舒服。
<br/>　　但是……山谷四通八达，气息流畅，居然整个山谷都充斥着浓郁的阴气，这让叶少阳感到十分地不可思议。
<br/>　　“只有两种可能。”叶少阳说道，“第一种，这山谷里有一处阴气之源，不断往外散发阴气，保持着山谷里阴气的浓度，还有一种可能，在山谷里有一道看不见的结界，将引起封固在山谷里，挥散不去。”
<br/>　　毛小方点点头，说道：“我当天走的匆忙，又是夜晚，根本没有仔细看，我们现在可以好好查验一番了。”
<br/>　　在下山之前，两人绕着山谷走了一通，惊人地发现，这里居然是一处风水宝地：山谷的尽头，矗立着三座山分，将山谷一分为四，一路下坡，采东来之气，从山峰上看过去，这三座山峰宛如虎踞之势，下方有溪流缠绕，如灵蛇腾舞。
<br/>　　这是一处虎踞龙盘的风水，不过溪流太小，不是大河，算不上是一个完全的风水局，所以只是蛇而不是龙，不然就成龙脉了。
<br/>　　尽管如此，这处风水也是好得不能再好，而且这是一处阴宅风水，如果在这座山谷埋葬先人，只要风水不变，后人一定飞黄腾达。只是……叶少阳搞不懂，这么好的一处风水宝地，为什么会被阴气覆盖？或者说，山谷里浓郁的阴气，跟风水有没有关系？
