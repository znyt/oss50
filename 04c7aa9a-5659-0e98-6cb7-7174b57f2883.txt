　　他乃是鬼王，能够开启空间门，将自己的死忠送进鬼界中自然不难，只是部署需要些时间而已。
<br/>　　数十年过去，他都没有找到机会。
<br/>　　曾经部署在鬼界的手下，则已经渗透到了阴寒域内部，甚至身居高位，只要他一声令下，曾经的死忠就会揭竿而起。
<br/>　　慕清霄出现在酆都，让他知道，自己的机会来了。
<br/>　　这是一场豪赌，输了便是带着一族永堕轮回，赢了在鬼界就拥有立足之地。
<br/>　　结果他赢了，赢的很轻松。
<br/>　　成王败寇，历史上的英雄，哪一个不是在赌？
<br/>　　此时，冥舟已经徐徐登岸，站在冥舟小甲板上的是一名裹着黑袍的身影，身材消瘦，看不清容貌。
<br/>　　此人，正是冥舟的主人。
<br/>　　瞥了眼黑袍身影，慕清霄目光一凝，后者给他的感觉就像是一团迷雾，完全看不透。
<br/>　　收回目光，视线落在天鬼皇身上，慕清霄眉宇间皱成了川子，忍不住问道：“此人也是你部署的？”
<br/>　　“不是，不过你放心吧，此人可信，先登船在说。”
<br/>　　天鬼皇坦率的摇了摇头，取出五颗鬼丹递给了黑袍身影。
<br/>　　只见黑袍中，伸出一只森白的手臂，接过鬼丹后便迅速缩回，随后站在小甲板上，如同木偶一样。
<br/>　　慕清霄眼底闪过一抹古怪，天鬼皇倒是觉得很正常，率先钻进冥舟中。
<br/>　　将目光收回，慕清霄也踏上冥舟，只是从黑袍身影身边走过的瞬间，星眸一眯，脚步也顿了一下。
<br/>　　在刚才一瞬间，他明显感觉有一道视线从自己身一闪即逝，后者在窥视他。
<br/>　　但后者依旧怔怔不动，慕清霄想起，冥河上严禁出手，也不知道是谁立下的规矩，紧接着就钻进冥舟中。
<br/>　　冥舟与农家小船一样，只是上面有个船篷。
<br/>　　船篷下的空间不大，能够容纳四人坐下，两边都有扶手，只是在无尽的冥河中，根本感觉不到一丝的安全感。
<br/>　　没有能量波动，也没有风痕，但冥舟却诡异的行驶了起来，慢慢离岸踏上冥河。
<br/>　　坐在凳子上，慕清霄有些无语，看了眼天鬼皇，找一个不认识的船家，这货的心脏真是大的可以。
<br/>　　若后者是别的鬼王，甚至鬼界高层安插的眼线，估计在过不久，天鬼一族就该灭亡了。
<br/>　　随手一挥，将空间隔绝，慕清霄看向天鬼皇，道：“现在可以说一说了吧？若是天鬼一族灭亡，可是你自己的错。”
<br/>　　“不可能的。”
<br/>　　摆了摆手，天鬼皇面色淡定自若：“鬼界最高层统领鬼族，无论是鬼王，或者是轮回路都在他们执掌范围，唯独冥河例外。”
<br/>　　慕清霄剑眉一挑，不明白他言下之意。
<br/>　　似乎看出了慕清霄的困惑，天鬼皇慢慢开口道来。
<br/>　　“我上船的时候就说过，冥河在鬼界的时间很模糊，似乎比神魔大战更久远，曾经鬼界高层也对冥河展开过调查，鬼仙陨落后，此事就不了了之。”
<br/>　　说到此处，天鬼皇有些感叹，强如鬼仙在冥河面前也是如此不堪，说陨落就陨落。
<br/>　　慕清霄洗耳恭听，冥河的事情，他知道的比天鬼皇还多，他现在唯一想知道的，是黑袍人的身份。
<br/>　　“他们不知何时出现的，非常神秘，修为深不可测，驰骋于冥河之上，鬼界称他们为‘引路人’。”
<br/>　　说罢，天鬼皇下意识瞥了眼船头的黑袍身影。
<br/>　　“他们？”
<br/>　　“不错，旁人哪怕得到黑冥木也无法渡过冥河，唯有引路人可以，他们不止一位，似乎一直存在于冥河之上。”
<br/>　　天鬼皇知道的信息也有限。
<br/>　　慕清霄脑海中不断思索，但引路人却着实没有听说过，最终将复杂的情绪抛之脑后，闭上双眼，开始修炼起来。
<br/>　　就在慕清霄运转心法的一瞬间，船头上的黑袍人身躯一颤，最终归为平静。
<br/>　　慕清霄与天鬼皇，谁都没有察觉到这一幕。
<br/>　　冥舟行驶在一望无际的冥河中，速度不快也不慢，抵达最底层所在的火鬼界，总共需要一周的时间。
<br/>　　冥河中没有任何生命迹象，也未曾听说过有任何危险，慕清霄也不需要为此担忧。
<br/>　　在这一周的时间里，平日里除了打坐修炼，便是温水煮茶，与天鬼皇有一搭没一搭的聊两句。
<br/>　　也不知道行驶了多少路程，总之慕清霄进入冥河后，就没见过岸边，眼中唯有一成不变的暗红色河水。
<br/>　　说冥河是一条河，但却不是江河所能比，一眼望不到尽头。
<br/>　　天鬼皇貌似不是第一次进冥河，除了与慕清霄闲聊，其余的时间都在沉思，或许在策划阴寒域的事情。
<br/>　　至于船头的黑袍人，从一始终都站在那里，未曾动弹过一下，似乎就是个木桩。
<br/>　　几天下来，慕清霄对此也见怪不怪。
<br/>　　就在这时，冥舟开始向南边方向偏移，天鬼皇则睁开双眼，道：“马上就要抵达最下层火鬼界的入口了。”
<br/>　　慕清霄微微点头，星眸中光泽流转，终于要抵达火鬼界了。
<br/>　　约莫两个时辰后，视线中终于出现了事物，映入眼帘的是一座座拔高的峰峦，峰顶都冒着白色烟雾。
<br/>　　似乎是火山群，刚进入这一代，便能清晰的感觉到，周围的温度明显提升了不少。
<br/>　　不多时，慕清霄的视线中就出现一片陆地，岸边长着一颗颗黑色的树木，似乎就是脚下的黑冥木。
<br/>　　冥舟徐徐靠岸，黑袍人终于转过身来，伸出一直苍白的手臂，做出一个请的动作。
<br/>　　见状，天鬼皇拱了拱手，率先登上陆地，可当慕清霄准备上岸的时候，却被苍白的手臂给拦了下来。
<br/>　　慕清霄没有察觉到恶意，瞬间紧绷的神经也松懈下来，看向黑袍人的目光充满困惑。
<br/>　　只见黑袍人手一挥，周围空间就瞬间凝固，虽然天鬼皇就在两米开外，却听不到其中的声音。
<br/>　　“鸿蒙未判，混沌未开……阴阳化日月，日月转鸿蒙。”
<br/>　　笼罩在黑袍下，沙哑的声音响起，听不出是男是女，只是声音中却透露着无尽的沧桑。
<br/>　　慕清霄目光一凝，道：“在下有些听不懂。”
<br/>　　话语刚落，黑袍人再次伸出手臂，只是掌中却多了一块令牌，令牌黑黝黝一块，上面印着神秘的纹路。
