　　“破！”
<br/>　　万劫魔尊眉心发光，黑色的元神世界飞了出来，纵天而上，在吞噬各种雷光，洗礼自身，淬炼元神，而后又导入肉身，双方一同吞噬天地雷劫。
<br/>　　这场大劫是一种检验，万劫魔尊在最可怕的雷光之中，淬炼魔体，淬炼魔刀、魔池、魔幡，淬炼元神，让自己更强，成为一尊伟大的魔神。
<br/>　　按理来说，他的这些魔兵正好与天地间至刚至阳的雷劫相互克制，但雷劫也是一种劫，只不过它是至刚至强的劫，只要是劫，他的魔兵便能吞噬。只不过，魔幡与魔池之中的魂魄，却是不能存活了。
<br/>　　毕竟有得必有失，但是，若是这些魔兵能够坚持下来，那么，它们以后的威力便能更上一层，而且还能够孕育出带有雷电属性的劫魔或者魂魄，威力更胜，魔刀更是带有雷电属性，攻伐至强。
<br/>　　毫无疑问，这是一个激烈的过程，充满了危险，随着可能在这万古难见的雷海之中化为灰烬，但熬过去的话，将天下难有大敌，可与圣人争锋。
<br/>　　雷光如瀑布，不断落了下来，劈在万劫魔尊身上，魔刀，魔池，魔幡之上，在上面刻下各种烙印，成为魔兵大道纹的一部分，烙印在魔体之上，便成了万劫魔体的一部分，这是自然构建而成，而非有意为之。
<br/>　　魔兵，浮浮沉沉，与黑色元神小世界一同被狂暴的神雷劈炼，魔气如云海般翻滚，代替了天地之间的云层，让这个地方显得恐怖而有可怕。
<br/>　　万劫魔尊抬头，身体像是无底洞，在与无尽雷海对抗的同时，也化身无数黑洞，在不断的吞噬着电芒，每一寸血肉都想混洞一般，早已经不知道吞纳了多少雷霆。
<br/>　　他的魔体，还有元神在发生一种可怕的蜕变，而今每一滴血液都蕴含了浓浓的杀机，内部有雷霆在炸响，任何一滴血珠只要飞出去，都可以毁灭星辰，炸碎大罗。
<br/>　　“当！”
<br/>　　天罚雷霆之中，有混沌钟响起，扰乱了时间，仿佛一瞬间经历了万古那么久远，让万劫魔尊的肌体都僵住了。在这一刻，他觉得像是过了一世，觉得身体苍老，化为了尘埃，而后又随着尘沙扬起，去轮回化为草木走兽等。
<br/>　　“叱！”
<br/>　　万劫魔尊一声大喝，让黑色元神一阵清明，运转之高神通，定住了己身，无视时间的紊乱，瞬间安静了下来。
<br/>　　接着，他全身气势大涨，魔道规则形成一方领域，定住了周围的虚空，并且以无上霸道的魔道之力，压向天穹，针锋相对，对抗起上苍劫伐。
<br/>　　此处顿时成了一片可怕的地方，各种法则交织，力量相撞，雷光飞舞，万劫魔尊全身上下的衣物早已不见了，在这狂暴的雷劫之下，都化为了飞灰，雷电劈来，根本保不住。
<br/>　　如今，唯有他的魔体不朽，还有他的魔兵可以与之对抗，别的东西，早已经在雷击之下，化为了飞灰。
<br/>　　这种浩瀚大劫，古今少有！
<br/>　　看到这一幕的大神通者们无不头皮发麻，浑身颤栗，只觉得巨大的生死危机笼罩其身，避无可避，逃无可逃。
<br/>　　大道演化成各种真实的形体，扑杀而来，那是各种生物，包括了狂暴的朱雀，主杀伐的白虎，强悍的真龙，傲世的麒麟……
<br/>　　当然，也有一些人形生物化形而出，许多堪比准圣圆满、混元初期的神明，扑杀过来，在与万劫魔尊激烈厮杀，大战在一起。
<br/>　　可怕的景象，无与伦比，无不让人震颤，每一种天劫都足以灭杀世间诸多准圣强者，而如今在这个地方却是如此层出不穷，无穷无尽。
<br/>　　从九九八十一重紫霄神雷开始，雷劫的数量就一直在递减，但是却更为恐怖，更为狂暴的镇压下来，源源不断。
<br/>　　“轰隆隆！”
<br/>　　轰的一声，黑白赤青黄五色仙光冲宵而起，在宇宙中化出一种莫名的气息，大五行雷劫出现了！
<br/>　　金、木、水、火、土演化在一起，交融合一，最后铸成一个巨大的大五行轮盘，碾压下来，对万劫魔尊的魔体造成了极大的威胁。
<br/>　　每个人身上都离不开五行，五行混乱，那是足以致命的伤害，唯有五行平衡，才能更好的存活。
<br/>　　大五行轮盘之上，有着五道五色齿轮，巨大的齿轮如同五颗恐怖的獠牙，在勾动上苍，引下更为可怕的雷劫。
<br/>　　这片浩大的星空都被毁灭了，一颗接着一颗的星辰炸开，成了粉碎。
<br/>　　万劫魔尊硬撼轮盘，肉身尽是血迹，都快要出现白骨了，这是十分恐怖的事情，要知道，之前与准提圣人大战之时，他都没有如此狼狈。
<br/>　　如今这般恐怖的雷劫，就算是圣人都不敢轻易进来，之前在一旁观看的准提圣人如今早已飞到很远，不敢踏进雷域范围之内，怕被万劫魔尊牵连到。就连圣人都如此，更不要说其他人了。
<br/>　　万劫魔尊劈碎磨盘，大五行之力全部溃散，魔血四溅，他露出凝重之色，仰天大吼，傲立虚空，睥睨天下，唯我独尊，魔威凌天。
<br/>　　他的元神世界飞出，吞噬这些五行仙光，还有雷劫，当然，魔兵亦是如此，吞噬了那破碎的五行轮盘，魔兵越发的恐怖了。他的身体不用他去可以控制，浑身毛孔都张开了，每一个细胞都像无底洞，吞噬五行雷劫。
<br/>　　他没有放过一丝力量，全部炼化，容纳己身，进行淬炼，补充海量所需，这是前所未有之事，一般人只想着去抵抗雷劫，却没有想过吞噬雷劫的力量。
<br/>　　“轰隆！”
<br/>　　在原先的天劫之上，再次出现一道惊世天罚降临，大道让人无法想象，覆盖了此方天地，击穿了原先所有的雷光。
<br/>　　这种景骇人心神！
<br/>　　这如九天降下的毁灭之光，毁灭万物，什么都将被灭掉，化为尘埃。
<br/>　　大劫降临，毁了之前所有的雷光，击穿了雷海大世界，更高层的力量出现，缭绕着混沌雾气。
<br/>　　那是一种苍茫波涛，很是诡异，也很玄妙，入眼无穷无尽，一会儿虚淡，一会儿缥缈，一会儿又如汪洋席卷。
<br/>　　最古老的本源法则在扩散，欲重新开天辟地，毁灭此处。
<br/>　　如此强大的天罚，实在是骇人听闻，令人无法想象，这已经不是天罚了，这是在灭世。宇宙中无人不再颤栗，心中对于天道的敬畏，更多了。
<br/>　　万劫魔尊双眼之中爆发出强烈的魔光，浑身上下充斥着霸气的威严，看着天上凝聚的灭世之劫，提着魔刀，毫不犹豫地便冲了进去。
<br/>　　“我为魔尊，掌万劫，执魔道，区区天罚又能奈我何？”
