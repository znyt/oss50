<br/>
<br/>        夏夕绾站在台阶上，有几分居高临下的看着宋明，少女眸光澄亮，淡漠从容，“抱歉，我登机时间已经到了，告诉你家先生，我已经跟陆寒霆离婚了，所以没有见面的必要，还有，以后不要动我朋友，这是第一次也是最后一次！”说完，夏夕绾转身。
<br/>
<br/>        宋明对夏夕绾有些刮目相看，她既然已经猜到他家先生是陆司爵了，她竟然不去，还敢出言警告，不愧是大少爷和老夫人看上的女孩，果真与众不同。
<br/>
<br/>        宋明看着夏夕绾纤柔的身影，不急不慢的勾唇，“夏小姐，大少爷现在在医院里，你确定不去看看他？”

<br/>        什么？

<br/>        夏夕绾脚步一滞，她迅速转身看着宋明，“陆寒霆怎么了？”
<br/>
<br/>        “大少爷吃了一瓶安眠药，好在被发现的及时，送到医院里洗了胃，不过他还没有苏醒。”
<br/>
<br/>        他吃了一瓶安眠药？
<br/>
<br/>        夏夕绾瞳仁一缩，他是不是…疯了？

<br/>        心里不好的预感已经成真了，她还是没能离开这座海城。

<br/>
<br/>        夏夕绾走到床边，垂眸看着他，他一张俊脸苍白如纸，昏迷的样子冷硬而凛冽，这段时间他清瘦了好多，更加深沉缄默，现在他毫无生气的躺在这里，寂静到令人压抑的病房里都是心跳检测仪发出的“叮叮”声，那么的刺耳。
<br/>
<br/>        夏夕绾缓缓伸出小手，颤抖的指尖落在了他的脸上，早晨在酒店里还让她觉得讨厌的人现在就躺在这里了。
<br/>
<br/>        原来，他也有这么苍白虚弱的时候啊。
<br/>

<br/>
<br/>        泪水是这般的滚烫，灼烧着她的眼睛，让她疼的心脏都在蜷缩。

<br/>        这时病房门被推开，有人进来了，是陆老夫人。
<br/>
