        &lt;h3&gt;第三卷第二百七十六章  爱情如此美好&lt;/h3&gt;
<br/>
<br/>        凌辰和宁雅一起安安静静的逛了一次商场，他将自己的手机悄悄的静音了，这么多年他才能拉上宁雅的手。
<br/>
<br/>        其实在他的心里还暗戳戳的开心，要不是他的父亲和宁玥的这一场搅和，他都不知道什么时候能牵着宁雅的小手。
<br/>
<br/>        宁雅给王律师打了个电话，说自己下午要回家一趟，让王律师将合同签下来，没有事情就不要给她打电话，然后也将手机静音了。
<br/>
<br/>        人生不过尔尔，我们为什么不留给自己一点时间，谈一场甜甜的恋爱，赏一场风花雪月，过一段安宁而平淡的生活呢，无论结局是什么样子，只要我们都彼此真心，彼此相爱过，还有什么可后悔的呢？
<br/>
<br/>        俩人手拉着手如热恋中的情侣亦如相濡以沫的夫妻，漫步在人潮汹涌的街头，行走在琳琅满目的商场里，宁雅专心的为凌辰挑选着衣服，精心的搭配着。
<br/>
<br/>        一套两套三套......，凌辰笑着说：“我长这么大都没有一次性买这么多衣服，你看看这都多少套了？”
<br/>
<br/>        “平时都没有时间给你买衣服，今天刚好有时间陪陪你，就多买几身，你得穿着我给你挑选的衣服去上班。”
<br/>
<br/>        “我一定会穿着小雅的衣服去上班，天天都穿着。”凌辰嘚瑟得万年冰山的脸上开出了温暖的花，融化的春水如甘冽的清泉，滋润着他的心田。
<br/>
<br/>        俩个冷冰冰的人擦出了爱的火花，沸腾了整个世界，梦幻开出了七彩的花，绚丽而多姿，在他们的世界里奇异的开放着，心被温暖和柔情包裹着，甜甜的感觉通过指尖传递给彼此，世界是如此温柔如此的美好，如此的甜蜜，原来这就是爱的味道。
<br/>
<br/>        “凌辰，今天我们两家的事情肯定不会结束，你要做好思想准备，是要娶宁玥还是和我在一起，你得考虑好，无论你什么决定我都尊重你。我晚上就回北京了，以后我就在北京发展了。这边我可能就回来的少了。”
<br/>
<br/>        “我会随你一起去北京的，我们一起共同面对今后的一切，你在哪我的心就在哪，我的人也在哪。”
<br/>
<br/>        “说心我还相信，说人我就不当真了，你还有你的事业，我们不能拿爱情当饭吃，一切不落到实处的谈情说爱都是耍流氓，我们需要自强自立，而不是躺在家族的功劳簿上吃喝等死，只有自己足够强大才能掌握自己的命运。”宁雅温柔的看着凌辰。
<br/>
<br/>        “我知道，你放心，这几年我已经能够独挡一面了，即使没有凌家我也能够好好的站在这里和你说话。”凌辰这几年和董家合作已经成长起来了，虽然还没有宁家和凌家庞大，但是实力也是不容小觑的。
<br/>
<br/>        凌家是老家族，家族门第观念也是很重的，凌辰不是那种坐吃等喝的人，他每年飞来飞去都成了空中飞人，就是想做一份自己的事业，他知道，人生活在这个世界上，底气是自己给的。
<br/>
<br/>        “你和董亚子一起来杭州，不仅仅是送宁玥回来吧？”宁雅问凌辰。
<br/>
<br/>        “其实我本来不想送宁玥的，当初想着宁玥是你的妹妹，我送她回来还能和你飞父亲套套近乎，以后娶你的时候就不会有很大的阻碍，没想到搞成了这个样子。不过，小雅，我又很感激他们这样，要不然我永远不知道我有多爱你。”
<br/>
<br/>        “你这是什么意思？摇摆不定？”
<br/>
<br/>        “不是，是我不知道我到底爱你有多深，现在我知道了，我不舍得你受委屈，看不得你被人欺负，我愿意为你抛弃一切。”凌辰紧紧的拉着宁雅的手，怕一松手宁雅就会离开他。
<br/>
<br/>        “其实，我也不知道我对你是什么样的一种感情，但是现在我知道你应该就是我要找的人，我们之间有误会，有疑虑，有不确定，可是这些都不重要了，重要的是你现在还在我身边，凌辰，每个女人都想要一份安宁的生活，一份真挚的感情，想要一生一世一双人，我也不例外，我不管你以前是什么样子的，最起码，遇到我之后我希望你心里只有我一个人。”
<br/>
<br/>        “以前我心里只有你一个人，现在我的心里还是只有你一个人，今后我心里也永远是你一个人。”凌辰说的是实话，这么久他只对宁雅一个人说过那么多的情话，也只对她一个人患得患失。
<br/>
<br/>        “我心里也只有你一个人，从始至终都是。”宁雅低着头轻声说。
<br/>
<br/>        美好的时光总是很短暂，一下午就在俩个人卿卿我我中流逝，宁雅想着时间也差不多了，她应该要走了，拿起手机看了一下，王律师给她打了好几个电话：
<br/>
<br/>        “凌辰，我得走了。”时光短暂，前路漫长，我们都得朝着自己的目标前进，感谢人生路上有你的陪伴，让我不觉得孤单。
<br/>
<br/>        “小雅，我想随你一起去北京。”凌辰此时的感情如喷涌的火山，他如何舍得宁雅走？
<br/>
<br/>        “你在这里还有工作要做，你不能丢下董亚子一个人在这里，还有，你得和你父亲将事情解决了。”此时的宁雅很冷静，她并没有被爱情冲昏头脑，反而很理智的提醒凌辰，凌辰的父亲还有可能在宁家。
<br/>
<br/>        凌辰的父亲能来宁家，肯定是有原因的，像他们这种家族的人，如果两家能够站在一条线上，那肯定都是经过权衡利弊，对双方都有益处，所以，凌辰未必能反抗得了。
<br/>
<br/>        宁雅只是一个女孩子，家里给她了她就会有，如果不给她了，她也不会去争斗，但是凌辰不一样，凌辰是要继承凌家的一切，虽然说凌家最近几年发展缓慢，但，骆驼永远是骆驼。
<br/>
<br/>        “那，小雅，你先回北京吧，我这里处理完之后就去北京找你。”宁雅想到的，凌辰自然会想到，他得回去和他的父亲好好说道说道，问问他的父亲是怎么回事，他的父亲还算是明事理的人。
<br/>
<br/>        他的父亲突然来杭州和宁玥一定有关系，如何将宁玥甩掉，这是一个问题，当然，他的态度已经很明显了，如果他们没有利益瓜葛，那是很容易解决了，就怕他的父亲和宁家达成了某种利益关系。
<br/>
<br/>        有时候，在强大的利益面前，对于有些人来说，一切都是可利用的工具。
<br/>
<br/>        天边的彩虹似乎缺了一角，像是永远都到不了彼岸，彩色的云朵，悠然的在空中飘荡，似乎他们已经看透了人世间的悲欢离合。
