　　第483章
<br/>　　这一战中，帝国军死伤惨重，最后逃回黄沙城的溃兵，不过五六千人。这还是因为李乐天下令退兵及时，如此才避免了被全歼的厄运。但尽管如此，也还是有三四千人倒在了战场上。
<br/>　　如此大的伤亡，便是整支队伍的溃败，打到后来，帝国军其实已经失去了抵抗力。他们要么选择逃跑，要么挺住脑袋被人砍杀。
<br/>　　当然，在这次伤亡中，青州兵死伤最多，凉州兵的损失却很少。这是源于凉州兵已经和蛤蟆军打过不少交道，打赢对方是不太可能的，但逃跑保命，却是也是他们擅长的。
<br/>　　号称西北强军的凉州兵，在这方面不愧强兵之名。
<br/>　　程大雷并没有命令部队追击，这证明了大家最开始的猜想：他就是要将诸人困死在黄沙城。
<br/>　　战后，本来该有一次军事会议，分析此战得失。帝国军有何长处可以发挥，蛤蟆军有何弱点可以利用。但出奇一致的，没有人提出召开会议，大家垂头丧气的离开了城头。
<br/>　　原因或许是因为，通过这一战，大家没发现蛤蟆军的弱点，同样，也没有觉得帝国军有什么长处。
<br/>　　李乐天、尉迟离、崔相等等诸人，现在脑袋都嗡嗡作响，这一战，有太多东西需要他们消化了。
<br/>　　帝国军中，不乏有识之士，他们皓首穷经，冥思苦想，琢磨如何提升帝国军的战斗力，可以在和戎族的战斗中获胜。然而结果总是令人失望。
<br/>　　今天，程大雷所率领的蛤蟆军，结结实实给他们上了一课。让他们明白，通过良好的配合，士兵可以发挥出难以想象的战斗力。
<br/>　　李乐天亲眼目睹了这一战，给他造成很大的震撼。原来打仗还可以这样打，如果给自己一些时间，训练出这样一支强兵，那么面临这个诸侯林立的帝国，自己将更有底气。再给自己一些时间，自己未必不能收拾旧山河，重整李家江山。
<br/>　　然而，谁还会给自己时间呢……
<br/>　　想到这里，李乐天长长的叹口气，颇有些有心无力的感觉。
<br/>　　“殿下，殿下…“”
<br/>　　一名亲卫跑进房间，打断了李乐天的思绪。刚近来他就跪下，道：“殿下，青州兵和京州兵打起来了”
<br/>　　“打什么？”李乐天奇怪道。
<br/>　　“青州兵说凉州兵畏战怕死，凉州兵说青州兵是一群蠢猪，结果就打了起来。”
<br/>　　“我看他们都是蠢猪！”
<br/>　　李乐天怒不可遏，都到现在了还有内讧。他本想立刻赶去处理，但想了想，却又停下脚步。有些事，未必要自己亲自出面。
<br/>　　“好啦，我知道了，让尉迟将军去处理。”
<br/>　　“那个……尉迟将军已经去了，崔相也在现场，就是尉迟将军让我来请殿下的。”
<br/>　　李乐天口中骂了一声，立刻赶往现场。见双方在街道上怒目而视，为首几个人打得鼻青脸肿。
<br/>　　“青州兵太过鲁莽，不然何至于造成如此大的伤亡。”宋伯康。
<br/>　　“姓宋的，你还说风凉话，就属你跑得快！”周唤鲸。
<br/>　　看二人的意思，是要当场动手，崔相立在宋伯康身后，尉迟离站在周唤鲸身后，都没对这一幕发表意见。
<br/>　　当看见李乐天出现的时候，众人都安静了下来。
<br/>　　“殿下，你可给我们评评理，兄弟们死得这么惨，姓宋的出工不出力，还说风凉话。”
<br/>　　“还请殿下主持公道，我也是实话实说，青州兵在这一战中，表现委实太差。”
<br/>　　李乐天一张脸已经难看到极点，他目光环视一周，看到青州和凉州两军中，都有人负伤。这次内讧，参与人数达上百人，如果不是阻止及时，情形将一发不可收拾。
<br/>　　“是谁先动的手？”李乐天忽然道。
<br/>　　青州兵中无人说话，凉州兵站出一人，指着一百夫长道：“是他！”
<br/>　　“好，你给我出来。说说他怎么动的手。”李乐天又指了指那百夫长：“你也与我出来。”
<br/>　　两人都站到李乐天面前，那百夫长昂着头，道：“殿下，你可要为我做主……”
<br/>　　李乐天不等他说话，突然拔剑，一剑串了他的喉咙，那凉州兵还来不及高兴，李乐天又挥剑砍了他。
<br/>　　两具尸体倒在地上，李乐天手中的剑淅沥沥向下滴着血，场上上千人，没有一人说话。
<br/>　　“大战在即，再有私斗者，与我的剑说话！”
<br/>　　目光扫到之处，无人敢与他的目光对视。李乐天收剑回鞘，大踏步离开了现场。
<br/>　　这时崔相才慢慢抬起头来，盯着李乐天的背影，心中冒出个想法：有些李蛮子的意思了。
<br/>　　李乐天并不想杀人立威，一个帝国皇子，目前距离帝国皇位最近的人，不需要用这种手段。他只是不太高兴，两军的内讧，令他察觉到一个不安的事实：军心已经乱了。
<br/>　　兵困孤城，没有粮草支援，又打了惨败的一仗。这种情形，令所有人都感到不安，如李乐天这样的大人物都如此，何况底层士兵。
<br/>　　大人物想要的是整个天下，底层士兵想要的只是活下去。
<br/>　　到晚上，更加糟糕的事实发生，豫州牧杨林，竟然私开城门，企图连夜逃跑。
<br/>　　这消息报到李乐天面前时，杨林已经衰兵逃走。他们是从南门逃走，这样逃回豫州的距离最近。
<br/>　　尉迟离已经亲自带兵追击，然而，并没有起到太大作用。程大雷代替他行使了处置逃兵的权力。
<br/>　　三千龙字骑对豫州兵展开追杀，草原上倒下了一具又一具的尸体，豫州兵没有任何反抗的意志，遭遇了一场大败。
<br/>　　最后，虽然逃脱了一批人，但伤亡的代价是巨大的。
<br/>　　这让李乐天的心情更是糟糕到了极点，除了豫州杨林外，怕其他诸侯也不想打下去了。
<br/>　　他分析眼前的状况，思来想去，怕也只有法子。便是出城决战，以八万饿夫对程大雷的一万精兵，当然会伤亡惨重，不过却也可以挣脱这困局。
<br/>　　但是，让谁牺牲让谁活着呢……各路诸侯怕也是各有各的心思。
<br/>　　李乐天忽然能体会明帝的感受，面对这样一群人，当个昏君真是比当个明君轻松啊。
<br/>(本章完)
