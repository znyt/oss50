        “我真的不是来纠缠你的，麻烦你松开我的手。我只是在这里等人来接我而已！”童筱颖一边说着，一边扭了扭自己那被何明旭束缚住的手腕。
<br/>
<br/>        何明旭越抓越紧，压根就没有要放手的意思，依旧口无遮拦地接着童筱颖的伤疤：“说吧！你想管我要多少钱？我听我妈说，你哥哥坐牢去了。你爸妈砸锅卖铁地在凑钱，想把你哥哥从牢里赎出来。我就说你家庭是个累赘吧！有个吝啬苛刻的老妈不说，还有个像废物一样窝囊的老爸，另外还加上一个没出息的老哥。不如这样，你陪我睡一晚，我给你两千块钱，如何？”
<br/>
<br/>        “啪”地一声。
<br/>
<br/>        童筱颖扬起另一只手，一巴掌重重地掴在了何明旭的脸上，将他的眼镜给打掉在了地上。
<br/>
<br/>        虽然，哥哥诺司峻的事情，大哥傅夜沉早就替她解决了。但是，哪怕梁家跟她没血缘关系，她也不能容许别人在她的面前说他们的不是。
<br/>
<br/>        诺母再吝啬再苛刻，诺爸再废物再窝囊，那也是养了她十八年，让她叫了十八十九年的爸妈。
<br/>
<br/>        而哥哥诺司峻再没出息，那也是从小到大，就一直爱护着她的，让她叫了十八十九年的哥哥。
<br/>
<br/>        也正因为童筱颖的这一巴掌，让何明旭猛然间想到了什么，嘴角不禁扬起一抹阴森森的笑意。
<br/>
<br/>        “筱颖，你还记得马智杰吧！那个带着你去出租屋堵我和若琴，让你发现我劈腿的马智杰！”何明旭咧着嘴，微微歪着头，话中有话地问。
<br/>
<br/>        童筱颖当然记得马智杰，他是何明旭的舍友，戴着黑框眼镜，文质彬彬的样子，每次她去他们宿舍，他总是第一个主动过来跟她打招呼。
<br/>
<br/>        只是，她不明白，这个时刻，何明旭为什么会突然跟她提起他。
<br/>
<br/>        “他为了你，把我痛打了一顿。然后，我跟若琴说了后，若琴利用她老爸的关系，把马智杰从学校里开除了！”何明旭呵呵地笑了起来。
<br/>
<br/>        童筱颖难以置信地瞪大了眼睛。
<br/>
<br/>        “噢，对了！我可是有仇必报的男人！他害得我两分手，又把我痛打了一顿。这仅仅只是让他被学校领导开除，可一点都不解我气！今晚，是他留在学校里的最后一晚。所以，我特意找了道上的几个混混，把他给绑了痛打了一顿。现在他应该还在四教的天台上躺着，估计他人应该差不多奄奄一息了！我估摸着你吧！应该早就和马智杰勾搭在一块去了！还假惺惺地说喜欢着我！呵，狗男女是你们两才对！”何明旭龇牙咧嘴地吼道。
<br/>
<br/>        “你是个疯子！”童筱颖再次甩开何明旭的手。
<br/>
<br/>        这次，他在她甩开他的时候，就已经主动要放手了。
<br/>
<br/>        童筱颖二话不说，直接转身朝校门口内跑去。
<br/>
<br/>        虽然她还不是这所大学的学生，但是第四教学楼，在哪里，她还是知道的。
<br/>
<br/>        因为，第四教学楼，她曾为了何明旭，替他在那儿代过课。
<br/>
<br/>        那个时候，他们宿舍的三个男生，和她坐在一排，替她打马虎眼，马智杰更是对她格外关照。
<br/>
<br/>        童筱颖气喘吁吁地跑到了第四教学楼的大门前，第四教学楼晚上是不对学生开放的，大家上晚自习都会选择去图书馆或者是自习大楼。
<br/>
<br/>        所以，她跑到大楼下的时候，整栋大楼里黑压压的一片。
<br/>
<br/>        童筱颖已经顾不上害怕，加快了步伐，直接冲上楼去。
<br/>
<br/>        她一口气爬上了楼顶，楼道里的声控灯，因为她的脚步声顷刻间，全都亮了起来。
<br/>
<br/>        童筱颖见天台的铁门未上锁，心悸地握住铁门上的门栓，拉开铁门，跑了进去。
<br/>
<br/>        天台上，周遭黑压压的一片。
<br/>
<br/>        童筱颖连忙掏出手机，点开电筒功能，照亮了前方的路。
<br/>
<br/>        她希望何明旭是骗她的，马智杰不在这里就好！
<br/>
<br/>        可是……
<br/>
<br/>        她每走一步，就越发的胆战心惊起来。
<br/>
<br/>        “马智杰？马智杰？你在哪儿？”童筱颖一边四下扫视，一边呼唤道。
<br/>
<br/>        “咳咳咳——”突然有人一阵咳嗽，瞬间令童筱颖倒吸了口冷气。
<br/>
<br/>        童筱颖循声望去，手里举着的手机也跟着照了过去。
<br/>
<br/>        天台右边靠近女儿墙的角落里，果真躺着一个人影。
<br/>
<br/>        “马智杰，是你吗？”童筱颖立即朝那人跑了过去，手机往那人身上一照，瞬间将她给吓了一跳。
<br/>
<br/>        那种希望破灭后，带给她的是心慌意乱，甚至是害怕与不知所措。
<br/>
<br/>        “我、我马上打电话！你一定要挺住！”童筱颖立即拨打了急救电话，准确地报了地址以及伤员的情况后，“啪”地一声，跪坐在了地上。
