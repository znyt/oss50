　　“你这个人很奇怪啊，你没看到我旁边的峰哥吗？他是我老大，有问题吗？”
<br/>　　李广道。
<br/>　　“峰哥？”
<br/>　　招揽人这一次才注意到了林峰。
<br/>　　这一眼扫过去，居然不知道对方什么等级。
<br/>　　但是他心中却是咯噔了一下，甚至心脏狂跳。
<br/>　　这就说明对方比他高很多了。
<br/>　　他扫了眼身边的苏云吴三陈勾，这一眼看过去，心中更是震惊。
<br/>　　除了个吴三，其余居然都是四阶的？
<br/>　　精英！这绝对是一个精英队伍。
<br/>　　“那个，几位老大，这里请！”
<br/>　　招揽人立即在前面带队。态度十分的客气。
<br/>　　“大家让一让！”
<br/>　　他把一群挤在那里填名单的人都给推开了，然后道：“小王，你们几个给我起来，把位置让给几位大哥坐，让他们慢慢填。”
<br/>　　一个坐在椅子上负责表格处理的年轻人还没反应过来就被招揽人给推开了。
<br/>　　然后毕恭毕敬的把位置给林峰他们摆上。
<br/>　　“几位老大坐下来慢慢填，不急。”
<br/>　　“茶水呢，端茶上来。”
<br/>　　小王满脸懵逼的拿了几瓶矿泉水。现阶段那里还有什么茶水，矿泉水就不错了。
<br/>　　“那我们就填呗。”
<br/>　　林峰等人拿起笔在表格上填写自己的资料。
<br/>　　林峰在实力上，填了个五阶，逢人留三分。
<br/>　　“我去，这群人什么来路。”
<br/>　　“居然还特么有椅子和水喝。”
<br/>　　“你们没看到，这战士会的招揽人之前可牛气了，可是看到这群人就好像老鼠遇到猫一样。”
<br/>　　“好多三阶的啊，其余的我看不透他们实力，难道都是四阶？”
<br/>　　一群人在旁边眼巴巴的看着林峰他们。
<br/>　　没多久。
<br/>　　林峰等人表格就填写完了。
<br/>　　“那个，三天后是不是还要参加考核。”
<br/>　　林峰问道。
<br/>　　招揽人扫了眼众人的资料，当目光落在林峰上面时，看了眼实力一栏，填写的赫然是个五阶，他心中就是一跳。
<br/>　　“那个，你们就没必要了，我等下给诸位申请个特权，几位的实力，我们战士会肯定信得过。不需要什么考核。”
<br/>　　招揽人立即道。
<br/>　　“既然如此也好。”
<br/>　　林峰道。
<br/>　　“那个，我叫麻子，欢迎几位加入我们战士会。”
<br/>　　麻子客气道。
<br/>　　“嗯。”
<br/>　　林峰点了点头。
<br/>　　“对了，峰哥，你们是刚来曙光的吧？把临时身份证给我，我今天就去帮你们升级为正式身份证，这样你们就有永久居住权了。”
<br/>　　麻子道。
<br/>　　“那就麻烦你了。”
<br/>　　林峰把他们的身份证给了他。
<br/>　　“哪里的话，峰哥这样的高手加入我们战士会，那是我们的荣耀。”
<br/>　　麻子客气道。
<br/>　　“对了，峰哥，我们战士会有集体宿舍的，不需要交多余的晶石，算是免费提供给干事住的。干事也可以自己带家眷免费入住。”
<br/>　　“这是你们的房间钥匙，小王，别在这里忙活了，还不快带几个老大去宿舍。”
<br/>　　麻子立即道。
<br/>　　“好。”
<br/>　　一个年轻人立即是拿着钥匙像是导游一样的请林峰等人过去了。
<br/>　　“峰哥，真羡慕你们，一来就有宿舍，现在房屋资源紧张，许多人可都只能睡大街。”
<br/>　　小王羡慕道。
<br/>　　“哦？是吗？这么严重？”
<br/>　　“没错啊，我们战士会也只有干事才有这个待遇。”小王道。
<br/>　　“有这么夸张吗？难道黑铁战士都只能睡大街？”林峰纳闷道。
<br/>　　“因为前几天，军部就开始大面积的传播黑铁战士的秘密，所以几天下来，黑铁战士早就没有之前那么稀少了，峰哥，你们应该看到许多组织在招人吧，因为黑铁战士不少，所以许多势力，现在门槛都提升到了二阶的。当然了，诸如峰哥这样的高阶黑铁战士，依然还是很稀少的。”
<br/>　　林峰等人面面相觑。
<br/>　　难怪如此，是因为军部把秘密公开了。军部这样做的目的也是为了让全名进化，人人都是黑铁战士，可以说算是人类发展的一个里程碑甚至是历史性的转折点了。
<br/>　　众人跟着小王，小王家里有点小关系，所以提前几天就来到了曙光基地，对这里还是比较清楚的，不过即使如此，他一家三口人都蜗居在宿舍的一个房价里。
<br/>　　“林峰，这一次，我们算是抱你大腿了，不然得睡大街啊。”
<br/>　　陈勾道。
<br/>　　李广苏云等人也是点头。
<br/>　　小王带着众人来到了平民区。
<br/>　　一眼望过去，密密麻麻，一个个房屋挨在一起，的确像是城中村了。
<br/>　　“这里就是平民区，也算是曙光基地内人最密集的地方，许多地下势力都混迹在这里。”
<br/>　　小王道。
<br/>　　随后小王带着林峰他们来到了战士会的宿舍区。
<br/>　　这个宿舍区看起了还算不错，环境也很整洁，比其余的房屋要高大上点。
<br/>　　“这里就是你们的房间了，一人一个。”
<br/>　　小王道。
<br/>　　林峰推开一个房门看了眼，大概二十平，不大也不小，一个人足够用了，最主要的是有单独的卫生间。
<br/>　　“峰哥，宿舍区的水电费是要额外交的。看你用多少交多少。反正一度电大概三颗晶石。一箱水大概十颗晶石。”小王道。
<br/>　　“这基地内还有电？”
<br/>　　林峰意外了。
<br/>　　“有电，都是大功率备用发动机，不过都是限量供应的，这里是战士会的宿舍区，自然是特殊供电的区域。”
<br/>　　“其余地方就算供电了，大部分人都用不起，水也是，都算是很珍贵的资源了。所以现在不管身上多臭，也不能像之前那样任性的洗澡了，水箱内水就那么些，用完了就必须要去和基地官方买了。到时候他们会有运水车来给你水箱加满的。”
<br/>　　小王道。
<br/>　　众人面面相觑，还真是非常的不习惯，不过不习惯也得慢慢习惯，条件就是如此。
<br/>　　“峰哥，你们就先整理下吧，一些牙刷什么的生活用品都可以自己去买。当然，前提是你们觉得需要的话，我就先走了。”
<br/>　　小王道了句就离开了。
<br/>　　“那大家就先整理整理，老师我住你隔壁吧。”
<br/>　　林峰道。
<br/>　　苏云无所谓，“随便你。”
<br/>　　她立即把林峰之前给她找的一些内衣内裤短裤之类的拿出来。
