　　“副院长差别待遇，也难怪新生会不爽，长得一副欠扁的模样，又有美女跟着，换做是我，恐怕会忍不住，打烂你白嫩的小脸。”
<br/>　　慕清霄道：“看我不爽的人有很多，不差他们几个，你们手中有多少火能，全都交出来吧，我到时候下手会轻一点。”
<br/>　　闻言，树梢上，五名男子面面相觑，这种事情，他们真是第一次碰见，新生敲诈老生，果然够狂的！
<br/>　　“马.勒.戈.壁.的，揍死这小子！”
<br/>　　话语刚落，老生不再废话，身影快若闪电，四面八方自上空围了过来！
<br/>　　望着袭来的老生，慕清霄眼底闪过一丝戏谑，不知何时，随意伸手抓住老生踢来的小腿。
<br/>　　微微用力，这名老生就感觉一阵天旋地转，紧接着视线也变得模糊起来，随后貌似砸中了什么东西，疼得他眼泪都快要掉下来。
<br/>　　同时，耳中传来一阵惊恐的惨叫声。
<br/>　　两分钟后，慕清霄站在布满枯叶的森林中，在他的前面，正横七竖八的躺着五名老生。
<br/>　　此时，老生们呲牙咧嘴，脸色惨白，脸上都冒着冷汗，有的腿部诡异弯曲，有的手臂直接垂下......
<br/>　　甚至，有一名老生泪流满面，双手握着屁股，其上插着一根断裂的粗壮树杆，隐隐有血迹顺着树杆渗透出来，似乎伤的不轻。
<br/>　　“怎么？要我自己走过去拿吗？”
<br/>　　平淡的声音响起，却令得地上的老生们头皮发麻，队伍中的领头男子，强忍着断臂的疼痛，站起身来，一脚踢在同伴身上。
<br/>　　“别特么装死，快把火晶卡拿出来。”
<br/>　　话语刚落，躺在地上的四名老生，皆是坐起身来，其中一名拔掉嵌在屁股中央的粗壮树杆，连忙取出纳戒中的火晶卡，递给领头男子。
<br/>　　接过火晶卡，男子心中满是纠结，目光中满是复杂之色，最终叹了口气，露出一个牵强的笑容，快步走上前来，弯腰将火晶片奉上。
<br/>　　“大哥，您...您要的东西...”
<br/>　　“孺子可教。”
<br/>　　慕清霄满意的点了点头，接过一张蓝色火晶卡与四张黑色火晶卡，上面的火能，加上他们刚才抢夺的，总共有二百五十点。
<br/>　　“火晶卡怎么用？”
<br/>　　“大哥，两张晶卡贴拢，搓一下就行了。”
<br/>　　话语刚落，慕清霄将两张晶卡贴拢，轻轻一搓，光芒闪烁间，晶卡上的数字已经有了变化。
<br/>　　瞥了眼他自己的黑色晶卡，只见上面的数字，已经从五变成了六十七，而男子的蓝色火晶卡，数字则变成了零。
<br/>　　望着半年的努力化为乌有，男子肝疼的厉害，干脆闭上双眼，眼不看为净。
<br/>　　当晶卡上的数字全部汇聚在黑色晶卡上，慕清霄将晶片还给了老生，似笑非笑，道：“学长，谢谢你们专程来送火能，你们真是好人。”
<br/>　　“尼玛，好气啊！”
<br/>　　深吸了口气，男子脸上肌肉狠狠抽搐了几下，满脸赔笑，道：“学长照顾学弟，应该的，应该的...”
<br/>　　“仙儿，我们走吧，顺路看看有没有别的好人。”
<br/>　　说着，慕清霄就不再理会老生，与他们擦肩而过，向着内院方向走去。
<br/>　　望着老生点头哈腰的欢送，小医仙捂嘴轻笑，随后收回目光，快步跟了上去。
<br/>　　见两人的身影消失在森林中，男子脸色阴沉无比，道：“关系户？这特么是逗老子玩的吧？”
<br/>　　其中一名断臂的老生，抹去脸上的冷汗，道：“队长，要找机会报复吗？”
<br/>　　闻言，男子微微摇头，道：“不算重伤，用点药物就能恢复，但这种人我们惹不起，走，去找先前给我们指路的五名新生，老子非得弄死他们不可。”
<br/>　　“对，我一定要粗树杆cha爆他们，居然给我们挖坑，我的菊花啊！”其中一名老生，脸上挂着泪痕，生无可恋的说道。
<br/>　　就在老生准备离开的时候，身躯陡然僵在原地，只见刚才离去的慕清霄与小医仙，不知为何又返回此处。
<br/>　　老生们一动不敢动，仿佛老鼠见了猫，双眼瞪的滚圆，生怕招来一顿虐待。
<br/>　　慕清霄有些哭笑不得，道：“有森林的大致路线图吗？”
<br/>　　“有的，有的。”
<br/>　　男子立刻苦笑着点头，他是彻底被慕清霄的实力给震慑住了，哪里敢说没有：“路线图不算精确，是年初在内院买的。”
<br/>　　说着，乖乖的冲纳戒中掏出一张粗糙的纸片，不舍的递给了慕清霄。
<br/>　　“多谢。”
<br/>　　“不谢，不谢，您老开心就好。”男子脸上挂着比哭还难看的笑容。
<br/>　　“对了，能跟我说说火能的作用吗？最好全面一点。”慕清霄似笑非笑的说道。
<br/>　　男子小鸡啄米一般的点了点头，憋屈无比，强忍着快哭的冲动，解释道：“火能是内院中最重要的东西，这也是新生进入内院修习后，实力突飞猛进的原因。”
<br/>　　“在内院中，有着一座深埋地底的‘天焚炼气塔’，在塔里面修炼，能够取得事半功倍的奇效，而且越往塔底下面，修炼速度就越快，当然，费用也是昂贵的离谱...”
<br/>　　“而想要进入‘天焚炼气塔’，就必须要火晶卡中的‘火能’充当费用，火能是以天数显示，新生手中的火晶卡，便足够他们在塔中修炼五天的费用，而五天过后，火能来源就只能自己去赚，而在每个月第一天，内院也会给每人发七天量的火能，却不足以维持生活。”
<br/>　　一口气将火能的作用说完，男子只求慕清霄快点离开，别在来迫害，剥削他们这群可怜的老生了。

<br/>　　想到这里，五人心中都升起一股浓浓的快意，不能只有他们几个倒霉，要倒霉也是一起。
