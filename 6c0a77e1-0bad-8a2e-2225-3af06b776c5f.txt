　　轰隆。
<br/>　　楚风眠出手，毫无保留，将远古战龙血脉的力量，催动到了极致，凝聚在了他的掌心之中。

<br/>　　一爪之威，撕裂天地，相传始祖天龙出手，一爪便是就足以撕裂天地。
<br/>　　楚风眠虽然远没有远古战龙的力量，但是他出手的威势，已经是有着几分始祖天龙出手的影子。
<br/>　　“始祖天龙血脉？怪不得有着如此玄妙的龙族武技，看来圣龙秘境内，的确是有着曾经始祖天龙遗留下来的东西。”
<br/>　　那老者看到楚风眠出手，刹那之间也看出楚风眠这一爪的玄妙，他也不敢托大，一步上千，手中刀锋闪烁，向着楚风眠一刀斩杀而来。

<br/>　　这老者对五行之力的运用，已经是达到了登峰造极的地步，一刀斩出，他身上的气息，足足是强横了五倍。
<br/>　　耀眼的遁光，斩杀而来。
<br/>　　楚风眠见到这刀光的到来，也毫不退缩，蕴含始祖第一式的一爪，轰击而去。
<br/>　　一刀一爪，凌空碰撞。
<br/>　　轰隆！
<br/>　　交锋的刹那之间，整个独桥都在晃动，在独桥之内的无数武者，都感觉到了这惊天动地的交手。
<br/>　　楚风眠跟如今这老者的交手，威力几乎是跟真正九劫古帝级别强者的交手，没有丝毫的区别。
<br/>　　一招一式，惊天动地。
<br/>　　碰！
<br/>　　在这交锋的瞬间，楚风眠也感觉到了一股无比巨大的压力，向着他的身躯冲击而来。
<br/>　　始祖第一式，虽然足以抵挡住这五行神刀大部分的力量，可还有着一部分的轰击在了楚风眠的身躯之上。

<br/>　　“九劫古帝的力量，果然是跟八劫古帝，截然不同！”
<br/>　　楚风眠目光闪烁，这也算是他第一次真正跟九劫古帝交手。
<br/>　　刚刚那始祖第一式的一爪，足以轻易轰杀一名八劫古帝，就算是曾经的小叶帝，得到过力量传承的天才，楚风眠都有着自信，一掌轰杀。
<br/>　　可现在面对九劫古帝的时候，这始祖第一式，居然是没有占据任何的优势，甚至反而是被反震了一部分的力量。
<br/>　　九劫古帝的力量，比起八劫古帝，要足足强大数十倍。
<br/>　　楚风眠心中惊讶，这老者的心中更加的惊讶。
<br/>　　他刚刚的那一刀，可没有丝毫的保留，算的上是全力出手。
<br/>　　在他的想法之中，刚刚楚风眠纵然是不死，也要受到重伤，伤筋动骨才对，可现在楚风眠居然是只是退了两三步。
<br/>　　虽然他是占据了上风，但是在这个级别的强者交手之中，这样的上风，几乎是没有任何的意义。
<br/>　　这也就代表着，楚风眠有着跟他平起平坐的力量。
<br/>　　八劫古帝，跟九劫古帝的差距，比起真帝跟古帝的差距，还要巨大，像是在圣龙秘境，八劫古帝，也就是一堂之主，四宗的太上长老。
<br/>　　而九劫古帝，却是真正的高层，地位天壤之别，足以证明这实力的差距。
<br/>　　以八劫古帝，拥有九劫古帝实力的，都是位列历史记载的天才，可以说无一例外。
<br/>　　“此人的资质，不亚于小天主，在这样发展下去，定然是第二个云尊，幸好此人还没有踏入九劫古帝，不然的话那个时候，还真的没有人可以杀了他了！”
<br/>　　这老者的目光之中，凝聚了几分杀意，盯着楚风眠。
<br/>　　他们五行天，现在的打算是要灭了五行天，自然不会放任，楚风眠成为第二个云尊。

<br/>　　刹那之间，这老者便是再度凝聚灵力，打入到了他手中的刀锋之内，一步踏出向前，不给楚风眠喘息之机，再度向着楚风眠斩杀而去。
<br/>　　这一刀一出，正如其名字所说，生机灭绝！
<br/>　　这周围的空间，本是因为那建木树枝的存在，生命力无比充沛，可是在他出刀的一刻，所有的生命力都被刀锋所湮灭。
<br/>　　一刀之下，不留活口，毁灭一切，杀招之中的杀招。
<br/>　　“来的好，九劫古帝，果然都有些能耐！”
<br/>　　楚风眠目光冰冷无比，面对这一刀的到来，楚风眠却是毫不退缩，在他背后的天涯剑匣之中，剑光闪烁。

<br/>　　轰轰轰！

<br/>　　纵然是刚刚进入了这铜门的武者，都足以清晰的感觉的到，来自这独桥深处的灵力碰撞。

<br/>　　独桥之上。

<br/>　　只是这力量的交锋，显然是不止一人，还有另一人的存在。
