　　清晨，朝阳初升。
<br/>　　苏莫换了一身青色的长衫，走出了自己的房间。
<br/>　　今日，便是会武的日子。
<br/>　　走出房间，冯光已经在院落中等他了，对方在昨日夜间，从九重浮屠塔返回。
<br/>　　“苏莫，快走吧，所有人在府邸门前集合！”冯光招呼道。
<br/>　　“走吧！”
<br/>　　苏莫点了点头，随即便和对方一起离开了院落。
<br/>　　“苏莫啊！今日你去观战，能大开眼界，见识一番七大势力的妖孽天才，对于你日后的修炼，也会有很大的裨益！”一边行走，冯光一边说道。
<br/>　　“嗯！”苏莫应付性的点了点头，心中却在想青璇的事情。
<br/>　　“七大势力的顶尖天才，战力均是强大无比，堪比一般的武尊境一重武者，根本不是你能想象的！”冯光继续说道。
<br/>　　“嗯！”苏莫不住的点头。
<br/>　　两人的速度很快，片刻之间便来到了府邸的门前。
<br/>　　此时，府邸的门前，已经聚集了不少的道宗弟子和长老，余下的弟子也是纷纷从住处走出，向府邸门前汇聚。
<br/>　　一刻钟之后，两百多名道宗弟子，和十几名长老，便全部集合在了一起。
<br/>　　“好了，我们走！”
<br/>　　真枯半圣环视众人，见人已经到齐，便招呼一声，带领众人向会武之地走去。
<br/>　　会武之地，显然不会在太阴城之中，因为武帝境以上的武者，破坏力太大，会严重破坏太阴城。
<br/>　　而若是用阵法之力护住，场地有限，则会有太多的束缚。
<br/>　　众人穿过一条条街道，最会来到了一座恢弘的宫殿之前，这宫殿高大数十丈，占地方圆千丈，极为庞大。
<br/>　　此时，有大量的太阴族之人，正不断的涌进宫殿之中。
<br/>　　“阵法空间！”
<br/>　　苏莫望着宫殿的大门，心中恍然，殿门之前阵法之力弥漫，想必其内是由阵法构筑的虚幻空间。
<br/>　　众人一路不停，鱼贯而入，纷纷的走进了宫殿之中。
<br/>　　穿过宫殿的大门，眼前场景变换，众人来到了一片虚空中。
<br/>　　苏莫抬眼望去，眼前是一个巨大的武斗场，庞大无比，一眼居然看不到尽头。
<br/>　　这武斗场，应该是阵法之力构建而成，略显虚幻。
<br/>　　武斗场的中央，是一个直径接近两万里战台，通体闪烁着淡淡的白光。
<br/>　　而在战台的四周，分别坐落着梯形的看台，一层又一层，延绵数十里。
<br/>　　几乎每一方的看台上，都能轻易容纳千万之众，非常的震撼。
<br/>　　而此刻，东西两方的看台上，已经人满为患，有千万武者落座，喧嚣震天，热闹非凡。
<br/>　　这些都是太阴城中的太阴族外族之人，早早的便来此等待会武。
<br/>　　而在北方看台上，则是空无一人，这里是太阴族内族之人观战的看台。
<br/>　　南方看台上，人数极少，庞大的看台上只有区区几百人，正是参加会武的各大势力之人。
<br/>　　黑炎族之人、闪灵族之人、啸天狼族之人，均已到此，其它几大势力之人，则还未出现。
<br/>　　真亏半圣带着众人，径直飞到了南方的看台上，此看台便是参加会武的七大势力的坐席。
<br/>　　随即，众道宗弟子，便纷纷在南方看台上落座。
<br/>　　“真枯半圣，别来无恙！”
<br/>　　黑炎族人群中，一名高瘦的中年黑炎族之人，目光望着真枯半圣，微笑着说道，只不过其眸中充斥着一丝戏虐的笑意。
<br/>　　黑炎族和道宗，时常有小的冲突，不仅是门下弟子不对眼，就连高层人物也是关系不好。
<br/>　　“真枯半圣！”
<br/>　　“真枯兄似乎修为又有精进！”
<br/>　　闪灵族和啸天狼族阵营，也有两位中年人向真枯半圣找打招呼你这两人中一人矮小瘦弱，如同猴精，一人高大威猛，如同人形猛兽，形成鲜明的对比。
<br/>　　“炎樘半圣、灵阳半圣、森狼半圣！”
<br/>　　真枯半圣微微抱拳，向三人客气了一句，这三人分别是黑炎族、闪灵族和啸天狼族的带队之人，均是半圣境的武者。
<br/>　　这种级别的会武，各大势力的武圣境强者，一般都不会来，均是半圣带队。
<br/>　　苏莫抬眼打量着这三大势力之人，每一个势力的人都不少，仅弟子而言，除了闪灵族只有两百人出头以外，其它两族人数均是超过了三百。
<br/>　　苏莫看到了炎火，也看到了那位墨狼，但是两人估计不是各自势力中最强的天才，仅凭气息而言，有不下五人的气息更在两人之上。
<br/>　　当然，真实的战斗力，并不是仅凭气息就能看出来的，毕竟有的人气息外放，有的人气息内敛。
<br/>　　至于闪灵族之人，则是完全看不出深浅，这个势力的强大，再与其族有特殊的本领，瞬移。
<br/>　　闪灵族之人，都有瞬息的能力，非常的诡异强大。
<br/>　　此刻，因为道宗之人的到来，整个武斗场更加的喧嚣，那众多的太阴族之人，议论之声不绝于耳。
<br/>　　“道宗的人来了，现在还有天虚宗、御剑谷和雷云禽族没到！”
<br/>　　“道宗的弟子之中，好像没有什么太厉害的妖孽啊！”
<br/>　　“道宗也就只有吕罡、林皇和唐古三人，有一丝希望进入前十，其他人都不值一提！”
<br/>　　“道宗弟子的实力比之其它几大势力的弟子，明显处于弱势啊！”
<br/>　　“是啊，黑炎族的炎澜天、啸天狼族的幽狼、闪灵族的灵枫，都是妖孽天才，铁定能入前十名，道宗貌似没有能和他们抗衡的天才！”
<br/>　　无数的议论声，沸沸扬扬，汇聚成浪，震荡四野。
<br/>　　一众道宗的弟子，耳中听闻这不绝于耳的议论，脸色都是很不好看。
<br/>　　这种被无数人，当面轻视的感觉，让他们非常的不是滋味。
<br/>　　“可恶！”
<br/>　　吕罡双拳紧握，捏的咯咯作响，他在道宗核心弟子之中，是一等一的天才，在太阴星上也有不小的名气。
<br/>　　但是来到这里，却完全不被人看好，他哪能受的了这种轻视？
<br/>　　一众道宗的长老，倒是没有太过在意，因为前几届会武，也是这种情景，他们都已经习惯了。
<br/>　　“你们都听到了吧！”
<br/>　　真枯半圣淡淡的开口了，声音响在一众道宗弟子的耳畔：“你们如此的不被看好，那你们更要争口气，夺回属于你们的荣誉，也夺回道宗的荣誉！”
<br/>　　众道宗弟子闻言，均是沉默不语，但每个人心中都憋了一口气。
