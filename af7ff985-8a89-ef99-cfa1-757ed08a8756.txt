　　第200章
<br/>　　“阁下可是要去风雨亭？”
<br/>　　大队停下，程大雷和老道在路旁相对而坐，他也明白，这老道突然冒出来，定然不是为了在自己面前扮高人。
<br/>　　“这件事你也知道了。”程大雷笑笑：“看来消息传得很快么。”
<br/>　　“去风雨亭做什么？”老道。
<br/>　　“哈哈，杨龙停准备送我一座城，以我为城主，幽州百姓也会到，大概是表彰大会什么的吧。”
<br/>　　“大当家，这种机密的事情，还是不要与他说。”徐神机道。
<br/>　　老道轻轻叹口气，显得忧国忧民起来：“程寨主英雄人物，其中关窍当真不明白么。”
<br/>　　“什么意思？”
<br/>　　“风雨亭不仅没什么表彰大会，反而囤满官兵，杨龙停要招待程寨主的不是山珍海味，而是刀枪，程寨主做何感想？”
<br/>　　程大雷一惊，其中事情他想不明白，但知道老道如此说必然有几分道理。
<br/>　　“还要向道长请教？”
<br/>　　老道摸着青肿的脸颊，心道程大雷刚才下手可是真重呐。
<br/>　　“程寨主打赢这一仗，不仅伤了戎族元气，也解了帝国危机。程寨主与天下人都是有功的，靠着这一战，程寨主便是封王也不过分。”
<br/>　　“难道是杨龙停贪了我的功劳？”
<br/>　　“倒也不是贪。”
<br/>　　“那便好。”程大雷和其他人松了口气。
<br/>　　“因为那功劳本就是杨龙停的。”
<br/>　　“什么？”
<br/>　　“想那幽州王杨龙停，何等英雄人物。力战戎族不退，歼敌十万。怪不得，如今天下人都说他乃是帝国战神。无数文人才子为其慷慨高歌，青楼歌姬口中，茶楼说书先生嘴里，都是他的英雄事迹。我听闻帝国朝廷，已经准备在青牛山为杨龙停刻碑立传，让后来人世世代代铭记幽州王做过的事情。”
<br/>　　“混蛋！”
<br/>　　“那事情是我们做的！”
<br/>　　“杨龙停那样的懦夫，见到戎族只知道逃，何曾打过一仗。”
<br/>　　程大雷还未说话，手下人已经是勃然大怒，众人当场就要去和杨龙停拼命。
<br/>　　程大雷示意众人稍安勿躁，开口道：“那杨龙停竟如此卑鄙，把我们的功劳据为己有，既然朝廷不明白，就要想办法告诉他们实情，让他们知道杨龙停所作所为，莫让懦夫被捧成了英雄。”
<br/>　　大家打这一仗，其实并未有任何建功立业，保家卫国的心思，无非是为了活着二字而已。可这也并不表示，大家可以看着自己流血牺牲做的事情，被按在别人头上，让狗熊成为被天下人敬仰的英雄。
<br/>　　“程寨主准备向朝廷拆穿……没用。”老道。
<br/>　　“呃……为何没用？”
<br/>　　“程寨主以为只要你打赢一仗，朝廷的封赏就会下来么，没那么简单。同一件事，放在不同人身上，可能结果就会完全不同。比如杨龙停，他乃是京城崔家的人，崔家在朝廷上下有多少关系，靠着这一战又有多少人可以获的利益，所以败仗可以是胜仗，懦夫可以成为战神。而像那西北大将林问天，死战戎族不退，最后粮草接济不力，才无奈兵败。可却被视作帝国耻辱，林将没有死在戎族之手，反而死在朝廷之手，死后坟墓还要被人唾弃。”
<br/>　　程大雷沉默了，在庙堂政治方面，自己的确太过幼稚。
<br/>　　杨龙停所作所为，真无人知道么，不，是很多人不想知道。只要明帝身边的太监说上一句：那幽州王打得不错呐。就可以有无数人从这件事中捞到利益，所以杨龙停会被捧为帝国战神。
<br/>　　而程大雷什么身份，一个四等不入的山贼而已。他打了胜仗，又有谁可以从中捞到政治资源？
<br/>　　你说仗是你打的，呸，你一个山贼也想和帝国战神争功，你也不撒泡尿看看你的样子。
<br/>　　“如今杨龙停已成了帝国军人的偶像，加官进爵，自然不在话下。可是幽州人很多人都知道，杨龙停其实和这件事没多大关系，是一个叫程大雷的山贼做的。如果想要花好月圆，万事大吉，杨龙停就要控制这个山贼，而控制又两种手段，一种是将他变成自己人，可以这个山贼平日的为人，很显然不是那么听话的主儿，那就只好用另一种控制的手段了……”郭烦人看着程大雷。
<br/>　　“杀了。”程大雷口吐两个字。
<br/>　　郭烦人轻轻点头。
<br/>　　程大雷倒抽一口凉气，冷汗将小衣湿透，昨日所有的欢喜，瞬间烟消云散。他只是个山贼而已，对这些庙堂上的手段，完全一窍不通。今日若非是郭烦人指点，他当真是犹在梦中，死到临头还不自知。
<br/>　　“多谢道长指点，大恩不言谢，还请道长再指点一下，接下来的路，我们该怎么走？”程大雷躬身深施一礼。
<br/>　　“幽州一线打得很漂亮，幽州王以及其他各城城主都有功劳，现在大家是你好我好谁都好。这样的花好月圆，断然不会让一个山贼毁了。如今要杀你们的，绝非一个幽州王。整个幽州，都将是围捕诸位的猎场。趁现在大兵未至，诸位还是早早散去吧。”
<br/>　　程大雷起身又施一礼：“多谢道长为在下指点迷津，道破玄机。”
<br/>　　郭烦人起身还礼：“程寨主力抗戎族，庙堂之上的贵人不知，天下人心里是知道的。”
<br/>　　程大雷转身面对山寨众人：“诸位兄弟，天下无有不散之筵席，今日大家就此别过，来日有缘，咱们再聚。”
<br/>　　“大当家，咱们不走，和他们杀上一场，戎族咱们都打跑了，还怕他们不曾。”
<br/>　　程大雷摇摇头，如今整个幽州的官兵都要杀他们，这些人打戎族没胆量，打自己人，胆量可是大得紧。
<br/>　　“诸位可以从青牛山突围，那里还未被大兵封锁，如今走，却还是有机会的。”
<br/>　　“大当家，咱们一起走。”
<br/>　　“一起走目标太大，戎族境地更加危险，大家只能化整为零，能打的弟兄带上几个身子骨弱的，能不抛下一个，就不抛下一个。”
<br/>　　程大雷目光扫过众人，经历过这场大战，多少人都被锻造成铁打的汉子，此刻却止不住眼含热泪。
<br/>　　程大雷忽然咧嘴一笑，高举拳头朝向空中。
<br/>　　“让我们在香波地岛再聚吧。”
<br/>　　
<br/>(本章完)
