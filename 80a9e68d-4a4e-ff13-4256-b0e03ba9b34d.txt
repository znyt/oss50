
“咣当”王惊蛰下车后，走过来第一句话就问道：“佛龛没有请回来了么？”<br/>　　<br/>　　“请了，从宝光寺请了一尊佛陀回来，我和那里的方丈是朋友，每年也给他们不少的香火钱，这次求的佛龛据说也是受了香火的，请回来以后就供在了小鱼儿的卧室里……”<br/>　　<br/>　　一般请的佛或者菩萨什么的分为两种，第一种是大部分人都知道的那种开了光的，就是被大师给加持过，能够起到镇宅，驱邪的作用，这种是比较常见而且也比较容易弄到的。<br/>　　<br/>　　还有一种就挺难得了，相对来说就是价格高点，这种就是受了香火的菩萨或者佛，这首先得是被供在寺庙的大殿中的，而且这寺庙的香火一定要旺还得灵，多数得是名山大川中的古寺才行，现代修建起来的都未必管用，因为历史太短，寺庙本身的念力就不多，没办法加持在供奉的物品上。<br/>　　<br/>　　供在这种寺庙大殿中的佛陀或者菩萨，常年来一直都受了香客的香火，也承受了寺中的气息，久而久之就沾染上了念力，是属于真正的灵物，一般请回到家中或者放在公司里，效果比被开光了的还要管用，只是这一类的供奉之物有些不太好弄，因为名山大川中的古寺就那么多，每年受的香火也有限，还得供的年头多一点才行，在加上国内现在信奉的香客也不多了，想搞一件真正受过香火熏染的东西也挺不容易的。<br/>　　<br/>　　不过说来，这种东西在国外比较常见，就比如泰国的寺庙中卖的那些佛牌就是，因为这个国家几乎是全民都信佛的，差不多人人都会去寺庙祭拜，虽然人口基数不多，可去参拜的都是诚心诚意的，哪像咱们国内啊，不夸张的说，有绝大多数的人都是有事了去求一下，没事就是游玩参观，然后就算求成了还不懂得还愿，那效果自然差了很多。<br/>　　<br/>　　沈君听了王惊蛰的话后，当天就去宝光寺求了一尊佛陀回来，放在了沈沉鱼的卧室中，插上香摆上贡品也诚心诚意的祈祷了一番。<br/>　　<br/>　　等到晚上的时候，沈沉鱼又在十二点的时候中邪了，这一次则更是邪门了，沈沉鱼从卧室里出来后哪里也没去，直接就从家里出来了，沈君跟在后面原以为她就在庄园里逛逛，但哪里想到沈沉鱼出去之后就没影了。<br/>　　<br/>　　没找到妹妹的人，沈君就急了，连忙报警然后也拖人去找，一天之后找出了不少的线索，最后才查到失踪了的沈沉鱼似乎去了大凉山一带的一处山村里。<br/>　　<br/>　　沈君连忙追了过来，等他赶到这里的时候，让他掉下巴的一幕出现了，沈沉鱼赖在了一户人家不走了。<br/>　　<br/>　　沈君惆怅的说道：“幸亏这户人家为人还不错，家里突然多了一个女人就赶紧报警了，并且也没对她做什么，小鱼儿去了这户人家以后就非说那是她的家，说什么都不走，还说家里的老人是她的丈夫，老头的儿子也是他的儿子，最关键的是……”<br/>　　<br/>　　沈君说到这里神情就很古怪了，还有种担忧的感觉，王惊蛰接着说道：“她还能说出那一家老小的名字，并且这家的哪哪都很熟悉，这还不算，就连街坊四邻的人也都能叫出来，就好像是真的回到了自己家里一样”<br/>　　<br/>　　沈君当即一愣，有些激动的点头说道：“对，就是这么回事，而且这户人家还是大凉山下的彝族，我妹妹从来都没有说过彝族话，但来到他们家后，张嘴就说了一口流利的彝族话出来”<br/>　　<br/>　　两人边走边聊就来到了那户人家的院外，就看见沈沉鱼穿着身彝族的服饰正在院子里收拾着猪草，手里的镰刀使的是相当的熟练了，旁边的草已经堆了好几捆了。<br/>　　<br/>　　沈沉鱼身前，站着四个人，一个七十多岁的老头，外加一对父母还领着个七八岁的孩子，这几个人一脸无语和惊呆的看着正干活的沈沉鱼，脸上的表情就四个字，他们被鸠占鹊巢了。<br/>　　<br/>　　“我想把她带回去，但小鱼儿不愿意，后来争执起来了，她把镰刀架在了自己的脖子上……”沈君搓了搓憋闷的脸，说道：“我是真犯愁了，不知道她这到底是中了什么邪”<br/>　　<br/>　　“不是中邪，也不是鬼上身，她是被人接上了灵媒……”<br/>　　<br/>　　菜刀文当即一愣，讶异的说道：“什么玩意儿？被接上了灵媒，谁下手这么狠啊，这不是得给人折腾死了嘛？”<br/>　　<br/>　　一般来讲，中邪和鬼上身比较好理解也挺常见的，人就是人被脏东西给附上了，这种状况解决起来也不麻烦，但被接上了灵媒就挺棘手的了，属于一种歪门邪术，并且非常阴毒。<br/>　　<br/>　　接灵媒，就是把一个刚死了还没过头七的人，要过鬼门进入阴曹地府准备投胎的魂魄给截留下来，然后转接到活人身上，这也算是鬼上身的一种，不过技术水平却要高了不少，通俗易懂的解释就是，被嫁接了灵媒的人已经不是自己了，而变成了另外一个人，也就是那个死者。<br/>　　<br/>　　就像沈沉鱼现在这样，笼统的说她已经不是之前的沈沉鱼了，连她亲哥哥沈君都不认识了，取而代之的则就是这户大凉山脚下人家里的老婆婆了。<br/>　　<br/>　　外人看的一脸懵逼，但她自己却是就这么认为的，这里就是她的家，这里的人就是她的亲人。<br/>　　<br/>　　但是呢，由于死者是阳寿已尽，必须得去阴曹地府报道了，但被这么给截留了下来，于天理所不容，久而久之下来，被嫁接了灵媒的人就会把生气给耗尽了，然后也随之死去。<br/>　　<br/>　　但这还不算，沈沉鱼属于强行截留进入阴曹地府的灵魂，阴间的礼法是不允许的，所以沈沉鱼死了之后得去阴间受训，判官会判其有罪的。<br/>　　<br/>　　“你妹妹或者是你，得罪人了吧，不然轻易没人会用这种手段对付她的……”<br/>　　<br/>　　沈君的脸色“唰”的一下就阴了：“祸不及家人，真要是因为我的话，这么干就越界了……”
