范恬流产了？
<br/>
<br/>        夏夕绾站在门边，看着厉嫣然，“没想到你们下手这么狠，竟然连流产的事情都做出来了，那接下来的事情应该是你们在我这里搜到了流产药，那不要浪费时间了，快拿出来给我定罪吧。”
<br/>
<br/>        厉嫣然还准备好好的给夏夕绾一个惊吓的，能看到夏夕绾惊慌的样子，她会很有满足感，但是哪里知道夏夕绾竟然已经准确猜测到了接下来要发生的事情，甚至还在催促，厉嫣然唇角抽了抽，
<br/>
<br/>        这时工作人员就从抽屉里翻出了一包流产药，“找到了！流产药找到了！夏夕绾果然是罪魁祸首！”
<br/>
<br/>        “刷”一下，所有人的目光都落在了夏夕绾的身上，指指点点的。
<br/>
<br/>        夜滢拿走了那包流产药，“夏夕绾，现在证据确凿，不容你抵赖，你先是因为私仇而陷害范恬，连累夜老，现在范恬怀孕，你见自己的奸计败露，又想出了这个恶毒的办法，用流产药将范恬给害流产了！”
<br/>
<br/>        “为了揭露你的恶性，还无辜者清白，所以明天我们会召开媒体记者发布会，趁现在还有时间，你好好想一下该怎么求饶吧！”
<br/>
<br/>        说完，夜滢带着人气势汹汹的离开了。
<br/>
<br/>        厉嫣然翘起了得意的小尾巴，“夏夕绾，这就是跟我们斗的下场，你就等着哭吧！”
<br/>
<br/>        厉嫣然也走了。
<br/>
<br/>        夏夕绾没什么表情，她将办公室里被翻动的一切都放了回去，然后走出门。
<br/>
<br/>        外面回廊里乱糟糟的，很多人聚在那里，范恬是真的流产了，现在脸色煞白，一身的血。
<br/>
<br/>        范恬看到了夏夕绾，她迅速伸出了沾血的手指指向夏夕绾，无比的愤恨，“都是你，夏夕绾你好歹毒的心啊，你竟然给我下流产药，你还我的孩子，这可是一条无辜的小生命啊，你这个杀人凶手！”
<br/>
<br/>        这时夜滢带着人将范恬抬上了担架，夜滢同仇敌忾的握住了范恬的手，“范恬，你现在不要说话了，我送你去医院，明天的媒体记者发布会就会揭露夏夕绾的一切罪行，到时她会受到应有的惩罚的！”
<br/>
<br/>        范恬无比感激的看向夜滢，“台长，谢谢你。”
<br/>
<br/>        夜滢适时的结束了这一场苦情戏，手一挥，“走，我们去医院。”
<br/>
<br/>        夜滢亲自带着范恬去医院了。
<br/>
<br/>        夏夕绾站在原地看着这两个人的身影，她们不去演戏还真是可惜了，刚才这双簧唱的可以拿奥斯卡双影后了。
<br/>
<br/>        夜滢走了…
<br/>
<br/>        夏夕绾澄亮的翦瞳落在了夜滢的台长办公室里，通往禁地的那把钥匙被夜滢放在了哪里，会在办公室里吗？
<br/>
<br/>        现在是一个绝好的机会，她要进去找钥匙！
<br/>
<br/>        还有很多人围观在这里，大家义愤填膺，纷纷指责夏夕绾，
<br/>
<br/>        人家范恬都流产了，她竟然站在这里面不改色的，她的心肠究竟有多么狠毒啊。
<br/>
<br/>        夏夕绾一来我们科学院，就闹出了这么大的动静，她竟然还敢设计我们得高望重的夜老。
<br/>
<br/>        不但如此，她还将我们陆院士给迷得团团转的。
<br/>
<br/>        夏夕绾，你没有心！
