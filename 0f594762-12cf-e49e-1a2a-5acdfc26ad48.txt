霍璇拿着房卡打开了顾夜瑾的房间门，走了进去。
<br/>
<br/>        偌大的房间低调而奢华，作为这艘游轮的主人，顾夜瑾的房间里还有着一张超大size的床，柜台上放着各种珍藏版的红酒。
<br/>
<br/>        顾夜瑾不在房间里，他在沐浴间里洗澡，里面传来了“哗啦啦”的水流声。
<br/>
<br/>        霍璇一颗芳心砰砰乱动，现在她来到了顾夜瑾的房间里，只要想到接下来会发生的事情她漂亮的小脸就红了。
<br/>
<br/>        她伸手，想抚一抚这柔软的大床。
<br/>
<br/>        但是这时里面传来了动静，顾夜瑾已经洗好澡准备出来了。
<br/>
<br/>        霍璇觉得自己的心跳快要跳出嗓子眼了，她快速的关了房间里的灯，然后躲在了窗帘后面。
<br/>
<br/>        如果不出所料的话，这张房卡是顾夜瑾送给叶翎的，这么多年顾夜瑾近乎病态的将叶翎护在自己的羽翼里，从不允许任何男人的接近，整个海城都以为他这是在宠妹。
<br/>
<br/>        事实却是，他只是想将叶翎占为己有。
<br/>
<br/>        如果他发现进来的是自己，而不是叶翎，那他会怎么样？
<br/>
<br/>        霍璇想起叶翎说的那句顾夜瑾喜欢放荡的女人，在她的记忆里出身簪缨世家的顾家少主顾夜瑾从小就接受着最正统的继承人教育，精良的衬衫西裤将他衬的清寒而严谨，他整个人都是禁欲的。
<br/>
<br/>        霍璇从没敢想他在床上会是什么样子，不过这些年他将叶翎护在身边不过就是满足自己的私欲，这足以证明他的内心是多么的禁忌。
<br/>
<br/>        男人嘛~
<br/>
<br/>        霍璇顿时有了自信，她躲在窗帘后面伸手将身上的短裙往下拉了拉，露出自己姣好的曲线。
<br/>
<br/>        这时“嗒”一声，沐浴间的门打开了，顾夜瑾走了出来。
<br/>
<br/>        顾夜瑾刚冲过冷水澡，身上一件深蓝色的丝绸睡衣，短发湿漉漉的还滴着水，整个人俊美的就像是漫画里走出来的男主角。
<br/>
<br/>        房间里的灯被关了，他清寒的黑眸准确的落在了窗帘后面那道躲着的身影上。
<br/>
<br/>        将手里的毛巾扔了，顾夜瑾大步走了过去，薄唇勾出一道弧线，他低声开腔道，“玩什么呢，恩？”
<br/>
<br/>        霍璇听出了他嗓音里带着的笑意，有几分邪气，她浑身一麻。
<br/>
<br/>        “出来，别惹我。”顾夜瑾伸出大手去扣她的皓腕。
<br/>
<br/>        霍璇一颗芳心乱跳，她往后一缩，想避开。
<br/>
<br/>        顾夜瑾在黑暗里的速度犹如闪电，他一把扣住了霍璇的皓腕伸手一扯，连人带着窗帘直接扯进了自己的怀里。
<br/>
<br/>        他将女人用力的锁在怀里，薄唇隔着窗帘落在她的发上，声线低哑而凶狠，“我抓到你了，这么不老实，是不是要我收拾你一顿恩？小妖精！”
<br/>
<br/>        这声“小妖精”让霍璇双腿一软，差点软在地毯上，她从来不知道斯文如玉的顾夜瑾关上房门会是这样发狠的模样。
<br/>
<br/>        顾夜瑾敛着猩红的俊眸，但是很快他就打开了眼，因为他意识到了怀里的不对劲，这根本就不是叶翎！
<br/>
<br/>        “你是谁？”
<br/>
<br/>        顾夜瑾俊美的五官里当即覆上了一层阴戾的雾霾，“啪”一声打开了房间里的壁灯，他伸手一甩，霍璇的身体顿时如断了线的风筝被挥撞在了墙壁上。
