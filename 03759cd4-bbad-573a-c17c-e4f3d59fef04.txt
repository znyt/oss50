　　灵神界第七重天，大罗天。
<br/>　　能够进入大罗天的修士，修为一般都为大罗金仙，武道修为相当于天地法相境。大罗金仙在这个圣人不出手的年代下，已经算是一方高手了，也是一方势力的顶级人才，因为在他上面，就只有准圣与圣人两个境界的高手，圣人一般不出手，准圣强者也很少。
<br/>　　能够成为大罗金仙高手，在一族之中，一般都是长老级别的人物。而此刻洪荒中的这些大罗高手，全部汇聚在大罗天中，他们足足有数百万之多，简直是不可思议。这还只是得到灵神令的大罗金仙，洪荒之中，没有灵神令的大罗金仙，那就更多了。
<br/>　　由此可以预见，洪荒之中的顶级高手情况。大罗金仙是如此，大罗以下者，恐怕数量更是恐怖。
<br/>　　此刻，在大罗天的大罗金仙们，他们中除了少部分人直接出去历练寻宝之外，其他的大多数却是汇聚在一起在商量着要怎么在灵神界中建立一方势力。
<br/>　　这些人大多数都是一个种族的精英，他们自然是知道，一方全新世界的资源到底有多重要，多得不说，就是灵神界之中的各种神奇之处，就不是他们能够拒绝的。如今初次来到这样的一个世界，最重要的不是去探险什么的，而是在这样的世界之中建立根基。
<br/>　　世界初开，大家的情况都是一样的，他们一同进入这个世界中，对这个世界的了解情况也是一样的，大家都是白手起家，想要占据更多的资源，想要得到更多的好处，地盘问题是在所难免的。
<br/>　　只有建立一个稳定的根据地，在灵神界中才能更好的发展。若是没有一方稳定的场所，人就会如同无根浮萍，随波逐流，那样就算得到资源，也是极为有限的。
<br/>　　于是，刚刚出现灵神界中，同一个种族的族人们便默契的汇聚在一起，他们共同探索灵神界的基本情况，并确立一方山头，建立势力，打下根基。
<br/>　　在大罗天的东方，一样有一座昆仑山，那是元始按照洪荒昆仑的模样设立的，如今在此昆仑山中，亦有一个玉虚宫，里面全是元始的徒弟们。
<br/>　　元始的十七个徒弟，除了燃灯这个听过一次道祖讲道的已经成了准圣高手外，其他的，十二金仙，云中子，五色孔雀孔宣、金翅大鹏迦楼罗，道金开灵的黄金道人，如今都已经是大罗金仙了。
<br/>　　虽然他们之中修为有高有低，最高的广成子已经大罗圆满，距离准圣也是只差一丝。广成子学了元始的五行印、四象印、阴阳印与无极印，也是元始徒弟之中，真正得到元始传承最多的，他更是根据元始的这四印，开创了自己的仙法《长生诀》，其中蕴含着阴阳四象五行之道，变幻无穷，威力无边。他还有一方半截不周山返本还源的极品先天灵宝翻天印，一印之下，苍天可翻，威力巨大，可战准圣高手。
<br/>　　第二高手不是十二金仙之一，而是五色孔雀孔宣，他也是大罗圆满高手，他学习了元始的五行印与诛神印，配合他的先天五行本源，先天之下，无物不刷，威力巨大，也是可战准圣。
<br/>　　孔宣之后，才是玉鼎真人、太乙真人、金翅大鹏、文殊广法天尊、黄金道人等，他们都是大罗后期高手，他们之外，其余的几个都是大罗中期。
<br/>　　他们之中，或多或少都有传承到元始的几道根本印法，但不知为何，却始终没有人能够学到元始的道一印与终极印法元始印。当然，这可能与天资机缘有关，最重要的是，元始印需要其他印法全部精通，才可能施展出，要知道，九印聚，元始现，可不只是说说而已。
<br/>　　此刻，广成子身为阐教大弟子，正在布置关于灵神界的任务。
<br/>　　元始虽然身为他们的老师，但一样没有给他们任何有关灵神界的消息，而是让他们自行探索，而且还给他们布置了许多任务，需要他们在百年之中做出突破，不求境界上突破，至少战力上要有重大突破。
<br/>　　元始收徒弟走的是精英路线。什么事精英，精英便是能够以一当十，当百，当千的人物，而不是只能一对一胜利的一般人。想要成为精英，自然也就需要更多的磨砺，对此，为了特殊照顾他们，元始还特地的在昆仑山附近，安排了许多战力逆天的神兽，凶兽等，那些可都是一群可以战准圣的怪物。
<br/>　　……
<br/>　　灵神界的东海之上，有一座岛屿，此处是截教的地址，名为金鳌岛，其上有一座宫殿，名为碧游宫。
<br/>　　此次截教并没有与阐教在一起，而是选择分开，不知道这是元始的意思，还是通天的意思，亦或是两者的意思。
<br/>　　通天的传教方式是有教无类，除了自己的几个真传弟子之外，其他的大多数都选择放养模式。截教的成员是在是太多了，号称万仙来朝，什么样的人都会有些，傲慢的，懒惰的，桀骜不驯的，有魔头，有狠厉之辈，有大凶之辈，当然，也有善良的修士。他们这些人中，又分很多派系，各自之间，谁也不服谁，表面上不怎么样，但暗地里，却是各自都有争斗。
<br/>　　不得不说，通天教主还是很有远见的，他并没有强行让他们改正，而是选择不管，体现出道家的无为而不治的思想。这样虽然狠厉的更加狠厉了，傲慢的更加傲慢了，但也正是这样，他的教派之中，形成了一种默契的竞争，各自都在进步。
<br/>　　对内他们虽然谁也不服谁，但是一旦对外，他们大多数都会联合起来，一致对外。最重要的是，通天一般对弟子们极好，所以大家都很尊敬通天。而不会像是阐教那样，元始的弟子们，那是对元始出了尊敬之外，更多的却是敬畏。
<br/>　　对此，元始一直都知道，但他并没有想要改变什么，他的性格就是那样，说是淡然冷漠，或者不屑，其实就是对一切都毫不在意，他只追求实力，其他的，大多都不放在心上，能用武力解决的问题，他一般都不会去浪费脑力。
<br/>　　这样，久而久之，他的弟子们自然而然会对他产生敬畏，忌惮之情。
<br/>　　而通天教主对弟子却是极好，极为护短，当然，其中也有他的弟子们的原因，种种原因，导致通天并不像元始那般淡漠。
<br/>　　元始与通天之间，没别的，只不过是性格不同而已。
<br/>　　通天的弟子们，也比较尊重多宝、无当、金灵、龟灵四位真传弟子，此刻灵神界中，便是通天的四位真传弟子带领着几位截教弟子，当然，也只是大罗金仙境的弟子，他们也在灵神界之中开发。
<br/>　　人教之中，便只有玄都一人，他倒是随便，也没有什么固定场所，在这灵神界之中走走停停，随遇而安。
<br/>　　除了三教弟子之外，人族是进入灵神界中人数最多的种族，他们也建立的势力。除此之外，其他的，如阿修罗族、冥族、天虎族、龙族、凤族、冰灵族、青丘狐族、虫族、魔族、天蝎族、兽族、精灵族、木灵族、巨人族等，都在灵神界中建立了势力。
<br/>　　灵神界中，散人模式、小队模式、诸侯争霸模式等，也即将开启。
<br/>　　……
