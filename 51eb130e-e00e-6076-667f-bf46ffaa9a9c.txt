　　谢雨晴将一张A4纸递给他，上面打印着一张身份证的资料，叶少阳先看照片，是陈悦，然后看资料：
<br/>　　姓名：陈悦，性别：女，民族：汉族，生日：1994年11月29日，文化程度：初中。职业：自由职业。
<br/>　　家庭住址：贵州省毕节市田寨区3组6户。
<br/>　　这就是陈悦的资料？
<br/>　　她还真的叫陈悦啊，即便到了虚幻世界，她也没有改名字。叶少阳觉得这有点怪异，但也不是什么大问题。再仔细看了几遍资料，目光落在出生日期上面，怔怔发呆。
<br/>　　“怎么了，有什么问题？”谢雨晴忍不住问道。
<br/>　　叶少阳看了陈悦一眼，吃惊道：“你竟然跟我一般大。”
<br/>　　谢雨晴道：“这有什么，你看上去比她老多了。”
<br/>　　“走开，我是说，同年同月同日生！”（注，第一章故事发生时间略微修正，避免后面很多地方有关少阳的年龄出现矛盾，此为修正后的官方年龄）
<br/>　　三人都怔住。
<br/>　　半晌，谢雨晴道：“这……意味着什么？”
<br/>　　“我不知道。”叶少阳喃喃道，他的确不知道这是为什么，同年同月同日生，这也太巧了吧？叶少阳觉得，这大概不是什么巧合，但一时间也想不到有什么背后的联系，只好暂时放下这个，去看她的其他信息。
<br/>　　她是贵州的。而且看后面什么组第几户的，应该是农村。
<br/>　　叶少阳问陈悦对这地址有没有印象，陈悦表示没什么印象。
<br/>　　“这个简单，你陪她去一下就是了。”谢雨晴道，“虽然身份证上的地址，不一定是实际居住地，也可能搬家了，但总能找到线索的，你跟她一块过去看看，肯定能找到她家。”
<br/>　　叶少阳一想也是，而且陈悦失忆了，看上去呆呆的，一个人真的怕她走丢了，自己陪她一起去，倒也没什么。
<br/>　　“不过我要先去找下道风，然后再过去。”
<br/>　　陈悦当然没意见。
<br/>　　谢雨晴道：“可以的，正好我要准备两三天。”
<br/>　　“你准备什么？”
<br/>　　“我也去。”
<br/>　　“你？”叶少阳瞪眼看着她。
<br/>　　“我申请了休假，正好跟你们一块过去，顺便当是旅行了，我还没去贵州玩过。”谢雨晴冲他挑了挑眉毛。
<br/>　　“好吧，如果可以的话，我陪你玩一下。”
<br/>　　谢雨晴点头，然后先带陈悦去补办了一张临时身份证，然后送他们出去。
<br/>　　“对了，小茹跟你联系了没有？”分别之前，叶少阳问谢雨晴。
<br/>　　“没有。”
<br/>　　这时候滴滴到了，叶少阳三人上车。上车之后，叶少阳才想起来，谢雨晴的回答有问题，假如她真的没跟周静茹联系过，她一定会好奇自己为啥这么问，而不是只回答一句“没有”。
<br/>　　不过车开了，叶少阳想着下次见面再问她。
<br/>　　中午，张小蕊和叶小萌来到叶少阳家中，带着腾永清——之前在承德他们见到腾永清，都觉得这和尚挺帅的，脸皮又薄，很好调戏，于是回到石城之后，立刻把他安排可住宿，带着他一起玩。
<br/>　　然后张小蕊当指挥，叶少阳大厨，叶小萌打下手，一起做了顿饭，有荤有素，腾永清不是四宝那样的花和尚，坚持吃素的。不过还是被张小蕊灌了点啤酒。
<br/>　　有这两个姑娘在，气氛还是很活跃的，闹了一个中午，吃完饭，两个涩涩的妹子拉着腾永清这个小奶狗去逛街了。
<br/>　　叶少阳收拾完碗筷，到自己房间里，迫不及待地用小九留下的玉石，打开了虚空裂缝，带着瓜瓜一起去了青冥界——包子这个坑货最近也迷上了电视剧，跟碧清成立了“追剧二人组”，一起看剧，对别的一切都没兴趣。
<br/>　　昨晚半夜还因为意见不合打了一架，包子不是对手，被拔掉了一撮根须，但为了看剧，还是忍辱负重地跟她和解了……
<br/>　　界河边上，防御森严，看到叶少阳和瓜瓜出现，一大队的士兵立刻聚拢上来，将他们团团围住，喝问他们的来历。
<br/>　　叶少阳扫了一眼，全是穿着盔甲的士兵，没有青丘山的人，也没有别的门派，刚要跟那个气势汹汹的长官解释，瓜瓜跳上前去，双手叉腰，恶狠狠地说道：“没长眼啊你们，认不得我是谁，找你们大元帅来！”
<br/>　　那长官一愣，道：“什么人也敢提到大元帅。”
<br/>　　“混蛋，我是你们家瓜瓜大爷，你职位太低，没见过我总该听过名字吧！”
<br/>　　那长官张口结舌，还在发愣，边上一个士兵对他耳语了几句，那长官浑身一颤，栗声道：“可是捉鬼联盟的大少爷瓜瓜仙长？”
<br/>　　“这下知道我是谁了！”瓜瓜得意洋洋。
<br/>　　这长官慌得单膝下拜，“竟然是您老人家，小人有眼不识泰山，罪该万死，罪该万死……”
<br/>　　边上那些士兵也跟着参拜。
<br/>　　“嘿嘿。”瓜瓜装逼成功，心情大好，主动上前把这长官扶起来，说道：“跟你闹着玩的，别见怪啊。”
<br/>　　“小人岂敢！”这长官受宠若惊，有点激动。
<br/>　　“对了你们大帅已经回来了吧？”
<br/>　　“对，昨天就回山了。”那长官朝叶少阳看去，“敢问这位是……”
<br/>　　“他就是我老大啊，叶少阳。”
<br/>　　这军官一听，双眼瞪大，立刻又要跪，被瓜瓜托住，道：“行了行了，我们又不算你直系长官，你们不必多礼。”
<br/>　　那长官一本正经道：“这如何使得！我们大帅不止一次跟我们说过，见捉鬼联盟任何人，都得像见到他一样尊重，再说连大帅都要叫叶天师一声大哥，自然是我们的主子啊。”
<br/>　　因为跪不下去，只好对叶少阳拱了拱手，激动地说道：“叶天师，小的早就听说过您老人家的名字，今日一睹天颜，小人……小人激动地不知道说什么好了。”
<br/>　　身边他那些士兵也都怀着绝对的敬畏打量叶少阳。
<br/>　　这给自己夸的……跟皇帝似的，叶少阳尴尬地不行，跟他说了几句话，表示自己要去青丘山，那军官不敢多问，立刻表示要派人护送，叶少阳谢绝了，在这一群士兵的恭送下赶紧走远。
<br/>(本章完)
